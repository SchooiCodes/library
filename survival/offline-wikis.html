What Are ZIM Archives and Kiwix?

ZIM is an open file format designed for storing web content (especially Wikipedia) for offline use. It compresses HTML pages, images, and metadata into a single file that can be read by Kiwix and other ZIM-compatible readers. Think of it as a portable hard drive of the internet.

  • ZIM — The container format. High compression, random access, metadata-rich.
  • Kiwix — The flagship ZIM reader. Available for Windows, macOS, Linux, Android, iOS, and as a server.
  • ZIM files — Available for Wikipedia (all languages), Wiktionary, WikiHow, Stack Exchange, Project Gutenberg, TED Talks, Khan Academy videos, and more.
  • zimfarm — The automated system that produces and updates ZIM archives regularly.
Tip: ZIM files use solid compression — the full English Wikipedia in ZIM format is ~100GB, while the raw HTML would be over 20TB. ZIM also supports full-text search with indexing.

Installing Kiwix

Kiwix comes in several flavors depending on your use case:

# Desktop (Windows/macOS/Linux)
# Download from https://www.kiwix.org/en/downloads/
# Windows: Kiwix Desktop MSI installer
# macOS: Kiwix DMG
# Linux: AppImage (works on any distro)
chmod +x kiwix-desktop*.AppImage
./kiwix-desktop*.AppImage

# Kiwix Serve (headless server)
sudo apt install kiwix-tools  # Debian/Ubuntu
# Or download static binary:
wget https://download.kiwix.org/release/kiwix-tools/kiwix-serve-3.3.0-linux-x86_64.tar.gz
tar xzf kiwix-serve-*.tar.gz
./kiwix-serve --port=8080 /path/to/zimfile.zim

# Android: Install from F-Droid or Play Store
# iOS: Install from App Store

Kiwix Desktop provides a full-featured reader with tabbed browsing, search, and bookmarking. Kiwix Serve turns any machine into a local server for ZIM content — accessible from any device on your network.

Where to Download ZIM Files

The official source for ZIM archives is the Kiwix download server. Here are the primary locations:

  • download.kiwix.org/zim/ — The main repository. Browse by category and language.
  • Kiwix Content List — Searchable table of all available ZIM files by language.
  • Kiwix Library — In-app download catalog (available in Kiwix Desktop and mobile).
  • FAU mirror (Germany) — Fast mirror of all ZIM archives.
  • MIT mirror (US) — MIT-hosted mirror for faster US downloads.

Most ZIM archives are updated monthly or quarterly. Wikipedia ZIMs are rebuilt every month, while Stack Exchange archives are updated less frequently.

Full Wikipedia Offline

The crown jewel of offline knowledge: the entire English Wikipedia, searchable, browsable, and fully linked — with zero internet required.

  • Full English Wikipedia — ~100 GB with images, ~45 GB text-only. Updated monthly.
  • Simple English Wikipedia — ~5 GB with images. Uses simpler vocabulary, great for ESL learners and kids.
  • Topical subsets — Wikipedia by topic: Medicine (~500 MB), History (~1 GB), Science (~2 GB), Geography (~800 MB), etc.
  • Other languages — German (~30 GB), French (~25 GB), Spanish (~20 GB), Japanese (~15 GB), and 300+ other languages.
# Download English Wikipedia (text only) via torrent
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_all_nopic.zim.torrent
# Or download the full version with images:
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_all_maxi.zim.torrent

# Check file integrity
md5sum wikipedia_en_all_nopic.zim
# Compare with the .md5 file from the download server

Torrent downloads are strongly recommended for large ZIM files — Kiwix provides .torrent files for every archive. This reduces server load and can resume interrupted downloads.

Wiktionary, WikiHow, and Reference ZIMs

Beyond Wikipedia, several other Wikimedia projects and reference works are available as ZIM archives:

Wiktionary

Full dictionary and thesaurus for hundreds of languages. English Wiktionary includes definitions, etymologies, pronunciations (audio), translations, and usage examples. ~30 GB with media, ~19 GB text-only.

WikiHow

How-to guides covering thousands of topics: first aid, electronics repair, cooking, programming, survival skills, and more. ~7 GB with images. An indispensable offline resource for emergency situations.

Other Reference ZIMs

  • Wikibooks — Open-content textbooks. Covers programming, science, and humanities.
  • Wikiquote — Quotations from notable people and works.
  • Wikisource — Free-content textual sources (laws, historical documents, literature).
  • Wikiversity — Learning resources, projects, and research.
  • Wikivoyage — Travel guides. Download this before a trip where you'll have no signal.
  • IbisReader — ZIM-compatible reader focused on travel guides and Wikivoyage content.

Stack Exchange Archives

The Stack Exchange network (Stack Overflow, Super User, Server Fault, Ask Ubuntu, etc.) publishes quarterly data dumps that are converted to ZIM format. This is an invaluable offline resource for programmers and system administrators.

  • Stack Overflow — The main Q&A for programming. ~80 GB text only. All questions, answers, comments, and votes.
  • Super User — Computing and hardware Q&A. ~5 GB.
  • Server Fault — Server administration Q&A. ~3 GB.
  • Ask Ubuntu — Ubuntu-specific Q&A. ~2 GB.
  • Unix & Linux — Unix/Linux general Q&A. ~2 GB.
  • All sites bundle — The complete Stack Exchange network in one download. ~95 GB.
Note: Stack Exchange ZIMs are text-only (no images). They're updated quarterly, approximately 3-4 months behind the live sites. Still, the vast majority of content is evergreen and remains relevant for years.

Project Gutenberg and Books

Project Gutenberg offers over 70,000 free public domain ebooks in multiple formats. The ZIM archive bundles them all into a searchable, browsable offline library.

  • Project Gutenberg ZIM — ~10 GB. Contains all books with metadata, sorted by author, title, and subject.
  • Books in multiple languages — English, French, German, Italian, Spanish, Chinese, Dutch, Finnish, and 50+ more.
  • Standard Ebooks — High-quality, modern, carefully formatted public domain ebooks. Smaller curated collection.
# Download Project Gutenberg ZIM
wget https://download.kiwix.org/zim/gutenberg/gutenberg_en_all.zim.torrent

# Search within the ZIM using kiwix-serve
# Open http://localhost:8080 in your browser
# Full-text search across 70,000+ books

Serving ZIMs on a Local Network

With Kiwix Serve, you can host your entire ZIM library on a local server and access it from any device on your network — phones, tablets, laptops, even smart TVs via the browser.

# Simple local server (single ZIM)
kiwix-serve --port=8080 wikipedia_en_all_maxi.zim

# Multi-ZIM server (serve your whole library)
kiwix-serve --port=8080 --library=my_library.xml

# Create a library file:
# kiwix-manage my_library.xml add /path/to/zim1.zim
# kiwix-manage my_library.xml add /path/to/zim2.zim

# Run as a system service (systemd example)
# /etc/systemd/system/kiwix-serve.service:
[Unit]
Description=Kiwix ZIM server
After=network.target

[Service]
ExecStart=/usr/local/bin/kiwix-serve --port=8080 \
  --library=/opt/zim/library.xml
Restart=always

[Install]
WantedBy=multi-user.target

Combine this with a Raspberry Pi (low power, ~5W) and a portable battery pack for a completely offline, portable knowledge server that fits in a backpack.

Building Your Offline Knowledge Library

Here's a recommended priority list for building your offline ZIM library:

Priority ZIM Archive Size Use Case
1Simple English Wikipedia~5 GBBest size-to-value ratio
2WikiHow~7 GBEmergency how-to guides
3Project Gutenberg~10 GBBooks and literature
4Wikivoyage~2 GBTravel and navigation
5Stack Overflow~80 GBProgramming reference
6Full English Wikipedia~100 GBComplete knowledge base

Start small — download the Simple English Wikipedia or a topic-specific subset first. Add more archives as your storage allows. A 256GB USB drive or SD card can hold the entire essential library with room to spare.

Pro tip: Use `zimcheck` (part of kiwix-tools) to verify ZIM file integrity after download: zimcheck wikipedia_en_all_maxi.zim. This checks the file structure and reports any issues.
