What Are ZIM Archives and Kiwix?

ZIM is an open file format designed for storing web content (especially Wikipedia) for offline use. It compresses HTML pages, images, and metadata into a single file that can be read by Kiwix and other ZIM-compatible readers. Think of it as a portable hard drive of the internet.

  • ZIM — The container format. High compression, random access, metadata-rich.
  • Kiwix — The flagship ZIM reader. Available for Windows, macOS, Linux, Android, iOS, and as a server.
  • ZIM files — Available for Wikipedia (all languages), Wiktionary, WikiHow, Stack Exchange, Project Gutenberg, TED Talks, Khan Academy videos, and more.
  • zimfarm — The automated system that produces and updates ZIM archives regularly.
Tip: ZIM files use solid compression — the full English Wikipedia in ZIM format is ~100GB, while the raw HTML would be over 20TB. ZIM also supports full-text search with indexing.

Installing Kiwix

Kiwix comes in several flavors depending on your use case:

# Desktop (Windows/macOS/Linux)
# Download from https://www.kiwix.org/en/downloads/
# Windows: Kiwix Desktop MSI installer
# macOS: Kiwix DMG
# Linux: AppImage (works on any distro)
chmod +x kiwix-desktop*.AppImage
./kiwix-desktop*.AppImage

# Kiwix Serve (headless server)
sudo apt install kiwix-tools # Debian/Ubuntu
# Or download static binary:
wget https://download.kiwix.org/release/kiwix-tools/kiwix-serve-3.3.0-linux-x86_64.tar.gz
tar xzf kiwix-serve-*.tar.gz
./kiwix-serve --port=8080 /path/to/zimfile.zim

# Android: Install from F-Droid or Play Store
# iOS: Install from App Store

Kiwix Desktop provides a full-featured reader with tabbed browsing, search, and bookmarking. Kiwix Serve turns any machine into a local server for ZIM content — accessible from any device on your network.

Where to Download ZIM Files

The official source for ZIM archives is the Kiwix download server. Here are the primary locations:

  • download.kiwix.org/zim/ — The main repository. Browse by category and language.
  • Kiwix Content List — Searchable table of all available ZIM files by language.
  • Kiwix Library — In-app download catalog (available in Kiwix Desktop and mobile).
  • FAU mirror (Germany) — Fast mirror of all ZIM archives.
  • MIT mirror (US) — MIT-hosted mirror for faster US downloads.

Most ZIM archives are updated monthly or quarterly. Wikipedia ZIMs are rebuilt every month, while Stack Exchange archives are updated less frequently.

Full Wikipedia Offline

The crown jewel of offline knowledge: the entire English Wikipedia, searchable, browsable, and fully linked — with zero internet required.

  • Full English Wikipedia — ~100 GB with images, ~45 GB text-only. Updated monthly.
  • Simple English Wikipedia — ~5 GB with images. Uses simpler vocabulary, great for ESL learners and kids.
  • Topical subsets — Wikipedia by topic: Medicine (~500 MB), History (~1 GB), Science (~2 GB), Geography (~800 MB), etc.
  • Other languages — German (~30 GB), French (~25 GB), Spanish (~20 GB), Japanese (~15 GB), and 300+ other languages.
# Download English Wikipedia (text only) via torrent
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_all_nopic.zim.torrent
# Or download the full version with images:
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_all_maxi.zim.torrent

# Check file integrity
md5sum wikipedia_en_all_nopic.zim
# Compare with the .md5 file from the download server

Torrent downloads are strongly recommended for large ZIM files — Kiwix provides .torrent files for every archive. This reduces server load and can resume interrupted downloads.

Wiktionary, WikiHow, and Reference ZIMs

Beyond Wikipedia, several other Wikimedia projects and reference works are available as ZIM archives:

Wiktionary

Full dictionary and thesaurus for hundreds of languages. English Wiktionary includes definitions, etymologies, pronunciations (audio), translations, and usage examples. ~30 GB with media, ~19 GB text-only.

WikiHow

How-to guides covering thousands of topics: first aid, electronics repair, cooking, programming, survival skills, and more. ~7 GB with images. An indispensable offline resource for emergency situations.

Other Reference ZIMs

  • Wikibooks — Open-content textbooks. Covers programming, science, and humanities.
  • Wikiquote — Quotations from notable people and works.
  • Wikisource — Free-content textual sources (laws, historical documents, literature).
  • Wikiversity — Learning resources, projects, and research.
  • Wikivoyage — Travel guides. Download this before a trip where you'll have no signal.
  • IbisReader — ZIM-compatible reader focused on travel guides and Wikivoyage content.

Stack Exchange Archives

The Stack Exchange network (Stack Overflow, Super User, Server Fault, Ask Ubuntu, etc.) publishes quarterly data dumps that are converted to ZIM format. This is an invaluable offline resource for programmers and system administrators.

  • Stack Overflow — The main Q&A for programming. ~80 GB text only. All questions, answers, comments, and votes.
  • Super User — Computing and hardware Q&A. ~5 GB.
  • Server Fault — Server administration Q&A. ~3 GB.
  • Ask Ubuntu — Ubuntu-specific Q&A. ~2 GB.
  • Unix & Linux — Unix/Linux general Q&A. ~2 GB.
  • All sites bundle — The complete Stack Exchange network in one download. ~95 GB.
Note: Stack Exchange ZIMs are text-only (no images). They're updated quarterly, approximately 3-4 months behind the live sites. Still, the vast majority of content is evergreen and remains relevant for years.

Project Gutenberg and Books

Project Gutenberg offers over 70,000 free public domain ebooks in multiple formats. The ZIM archive bundles them all into a searchable, browsable offline library.

  • Project Gutenberg ZIM — ~10 GB. Contains all books with metadata, sorted by author, title, and subject.
  • Books in multiple languages — English, French, German, Italian, Spanish, Chinese, Dutch, Finnish, and 50+ more.
  • Standard Ebooks — High-quality, modern, carefully formatted public domain ebooks. Smaller curated collection.
# Download Project Gutenberg ZIM
wget https://download.kiwix.org/zim/gutenberg/gutenberg_en_all.zim.torrent

# Search within the ZIM using kiwix-serve
# Open http://localhost:8080 in your browser
# Full-text search across 70,000+ books

Serving ZIMs on a Local Network

With Kiwix Serve, you can host your entire ZIM library on a local server and access it from any device on your network — phones, tablets, laptops, even smart TVs via the browser.

# Simple local server (single ZIM)
kiwix-serve --port=8080 wikipedia_en_all_maxi.zim

# Multi-ZIM server (serve your whole library)
kiwix-serve --port=8080 --library=my_library.xml

# Create a library file:
# kiwix-manage my_library.xml add /path/to/zim1.zim
# kiwix-manage my_library.xml add /path/to/zim2.zim

# Run as a system service (systemd example)
# /etc/systemd/system/kiwix-serve.service:
[Unit]
Description=Kiwix ZIM server
After=network.target

[Service]
ExecStart=/usr/local/bin/kiwix-serve --port=8080 \
 --library=/opt/zim/library.xml
Restart=always

[Install]
WantedBy=multi-user.target

Combine this with a Raspberry Pi (low power, ~5W) and a portable battery pack for a completely offline, portable knowledge server that fits in a backpack.

Full English Wikipedia Download & Management

Downloading the full English Wikipedia (~45 GB text-only, ~100 GB with images) requires planning and reliable internet. Here is the recommended approach.

Download via Torrent

Torrent downloads are strongly recommended for large ZIM files. They support resume, integrity checking, and can be faster than direct HTTP downloads. Kiwix provides .torrent files for every archive.

# Download the torrent file:
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_all_maxi.zim.torrent

# Use any BitTorrent client:
# qBittorrent (GUI, all platforms)
# Transmission (lightweight, Linux)
# aria2c (command-line):
aria2c --seed-time=0 wikipedia_en_all_maxi.zim.torrent

# Text-only version (much smaller):
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_all_nopic.zim.torrent

Storage Management

A full Wikipedia ZIM requires significant storage. Consider these strategies:

  • External SSD — 128 GB or 256 GB USB-C SSD for portable Wikipedia access.
  • SD card — 256 GB microSD for use in laptops or tablets.
  • NAS storage — Store Wikipedia on your NAS and serve via Kiwix Serve.
  • Partial downloads — Consider topical subsets (Medicine, History, Science) if storage is limited.

Smaller Wikipedia Variants

Not everyone needs the full English Wikipedia. These smaller variants provide excellent knowledge density in fraction of the space.

  • Simple English Wikipedia (~5 GB with images, ~1.5 GB text-only) — Uses simpler vocabulary. Perfect for ESL learners, kids, and quick reference. Covers the same topics as the full Wikipedia but with simplified language.
  • Wikipedia for Schools (~2 GB) — Curated subset of Wikipedia articles suitable for K-12 education. Screened for appropriate content.
  • Thematic sub-sets — Wikipedia by topic: Medicine (~500 MB), History (~1 GB), Science (~2 GB), Geography (~800 MB), Technology (~1.5 GB). These are available from the Kiwix download server under the "wikipedia" category.
Strategy: Start with Simple English Wikipedia (~5 GB). It covers 95% of topics you need for general reference at 5% of the storage cost of the full Wikipedia.

Kiwix-serve for Local Network Hosting

Kiwix-serve allows you to host ZIM archives on your local network, making them accessible from any device with a browser — phones, tablets, laptops, smart TVs, and even game consoles.

# Basic single-ZIM server:
kiwix-serve --port=8080 wikipedia_en_all_maxi.zim

# Multi-ZIM server with library:
kiwix-serve --port=8080 --library=/opt/zim/library.xml

# Create the library file:
kiwix-manage /opt/zim/library.xml create
kiwix-manage /opt/zim/library.xml add /opt/zim/wikipedia_en.zim
kiwix-manage /opt/zim/library.xml add /opt/zim/wikihow_en.zim
kiwix-manage /opt/zim/library.xml add /opt/zim/gutenberg_en.zim

# Access from any device:
# http://server-ip:8080 — browse the ZIM library
# http://server-ip:8080/search?query=searchterm — full-text search

Kiwix-hotspot for Wi-Fi Access Point Distribution

Kiwix-hotspot is a specialized tool that turns a Raspberry Pi (or any Linux computer) into a Wi-Fi access point that serves ZIM content. Users connect to the hotspot's Wi-Fi network and automatically see the ZIM library in their browser (captive portal style). No internet connection needed.

# Kiwix-hotspot on Raspberry Pi 4/5:
# 1. Flash Raspberry Pi OS Lite to SD card
# 2. Install kiwix-hotspot from the Kiwix download page
# 3. Place ZIM files in /data/zim/
# 4. Run: sudo ./kiwix-hotspot

# The Pi creates:
# - Wi-Fi network name: "Kiwix Hotspot"
# - Captive portal at: http://192.168.2.1
# - Full ZIM library browsing and search

# Power requirements: ~5W (Raspberry Pi 4)
# A 20,000 mAh power bank runs this for 10+ hours

Mobile Kiwix on Android/iOS

Kiwix mobile apps bring the full power of ZIM archives to your phone or tablet. No internet connection required after downloading the ZIM files.

  • Kiwix Android — Available on F-Droid and Google Play. Supports multi-window, split-screen, and external ZIM files on SD cards. Download ZIM files directly in the app via the library browser.
  • Kiwix iOS — Available on the App Store. Full-text search across ZIM files, bookmarking, and reader mode. iOS file management is more restricted — use iTunes File Sharing to transfer ZIM files.
# Kiwix Android — external ZIM storage:
# Place ZIM files in: /storage/XXXX-XXXX/Kiwix/
# (XXXX-XXXX is your SD card identifier)
# Open Kiwix → it automatically detects ZIM files

# Kiwix iOS — transferring ZIM files:
# 1. Connect iPhone to computer
# 2. Open Finder (macOS) or iTunes (Windows)
# 3. Select iPhone → Files → Kiwix
# 4. Drag and drop ZIM files
# 5. Open Kiwix — ZIMs appear in the library

Wiktionary as Offline Dictionary

Wiktionary is Wikipedia's sister project — a multilingual dictionary with definitions, etymologies, pronunciations, translations, and usage examples. The English Wiktionary ZIM is ~30 GB with audio, ~19 GB text-only.

For practical everyday use, Wiktionary replaces several reference works: dictionary, thesaurus, translation dictionary, and etymological dictionary. In Kiwix, full-text search works across all word entries, making it faster than a physical dictionary.

Smaller alternative: The English Wiktionary "nopic" version (text-only, ~19 GB) is sufficient for most dictionary needs. Audio pronunciations and images add significant size without much practical benefit for reading.

WikiHow for Practical Guides

WikiHow is an indispensable offline resource for emergency scenarios. It contains step-by-step guides for thousands of practical topics: first aid, shelter building, food preparation, electronics repair, and more. The ZIM archive (~7 GB with images) is one of the most useful single downloads for an offline library.

Recommended for: Emergency preparedness kits, camping trips, field research, and any situation where you might need practical guidance without internet access. Pair it with MedLine ZIM for comprehensive medical reference.

Searching Across ZIM Files

When you have multiple ZIM files, searching across all of them efficiently requires the right approach:

  • Kiwix Desktop — Supports searching across all open ZIM files. Results are ranked and grouped by source ZIM.
  • Kiwix-serve with library — When serving multiple ZIMs via kiwi-serve with a library.xml, the search feature covers all ZIMs in the library.
  • ZIM indexing — Kiwix creates full-text indexes for ZIM files on first open. This takes time for large files (10-30 minutes for full Wikipedia). The index is cached for future searches.
# Pre-index ZIM files (speeds up first search):
kiwix-index wikipedia_en_all_maxi.zim
# Creates a .idx file alongside the ZIM

Sharing Via USB/SD Card

Physical distribution of ZIM archives is often the fastest and most reliable method, especially in areas with limited internet. A USB drive or SD card can carry terabytes of knowledge.

  • USB 3.0 drive — A 256 GB USB 3.0 drive costs ~$30 and can hold the full English Wikipedia plus several other ZIM archives. Read speed: ~100-200 MB/s.
  • microSD card — 256 GB microSD costs ~$25. Use in phones, tablets, laptops, and Raspberry Pi. Read speed: ~30-90 MB/s.
  • Portable SSD — 512 GB USB-C SSD costs ~$60. Fastest option (500+ MB/s). Durable and pocket-sized.

Distribution strategy: Load one master drive with a curated ZIM library. Clone it to additional drives. Distribute to community members or emergency contacts. This is the "sneakernet" approach to knowledge distribution.

Creating Local ZIM Archives From Web Content

You can create ZIM archives from any web content using Zimwriterfs or the warc2zim converter. This is useful for preserving websites, online documentation, or personal wikis.

# Option 1: warc2zim (recommended for websites)
# First, download the website as WARC files:
wget --warc-file=mysite --recursive --level=3 \
  https://docs.example.com/

# Convert WARC to ZIM:
pip install warc2zim
warc2zim ./mysite.warc.gz --output=mysite.zim

# Option 2: Zimwriterfs (for local HTML files)
# Use when you already have the HTML content locally:
sudo apt install zimwriterfs
zimwriterfs --welcome=index.html \
  --language=eng \
  --title="My Site" \
  ./html-dir/ \
  mysite.zim
