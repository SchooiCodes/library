Digital Emergency Prep Checklist

A comprehensive checklist for digital emergency preparedness - immediate actions, weekly prep, monthly checks, essential downloads, and communication planning.

Immediate Actions

# RIGHT NOW (30 min):
# 1. Install Bitwarden/KeePassXC - start using a password manager
# 2. Enable 2FA on email - email is the key to everything
# 3. Back up critical documents - Desktop, Documents, Photos
# 4. Download Firefox Portable on USB - pre-load uBlock + Bitwarden
# 5. Write down master password - seal in envelope, store in safe

Weekly Prep

# EVERY WEEK (15 min):
# 1. Run antivirus scan (Windows Defender quick scan)
# 2. Check for OS updates - install security patches
# 3. Verify automated backup completed
# 4. Update ZIM archives (if due)
# 5. Check emergency USB still works

Monthly Checks

# EVERY MONTH (1 hour):
# 1. Test restore 3 random files from backup
# 2. Review password manager for weak/reused passwords
# 3. Check drive health (SMART)
# 4. Update emergency USB - portable apps, ISOs
# 5. Test offline tools - Ventoy, Meshtastic

Essential Downloads

Security: Malwarebytes Portable, KVRT, TDSSKiller, AdwCleaner, HitmanPro.

Recovery: Recuva Portable, TestDisk/PhotoRec, DDRescue.

ISOs: Hiren Boot CD PE, SystemRescue, GParted Live, Ubuntu Desktop Live.

Portable Apps: Firefox, KeePassXC, VLC, 7-Zip, Notepad++, VeraCrypt.

Document Backup

Tier 1: Passwords, 2FA codes, crypto seed phrases, SSH/GPG keys.

Tier 2: Scanned ID, insurance, financial records, medical records, property docs.

Tier 3: Photos, personal projects, work documents, email archives.

Locations: Local encrypted HDD + cloud backup + offsite physical + encrypted USB.

Communication Plan

Primary: Family Signal/WhatsApp group chat.

Fallback 1: Phone SMS - memorize 1 emergency number.

Fallback 2: Meshtastic/Briar - 2+ nodes tested monthly.

Fallback 3: Ham/CB/FRS radios with pre-programmed frequencies.

Emergency contact: Person outside the area. Everyone knows to call them.

Communication Checklist

  • Mesh network contacts — Meshtastic or Briar node contacts synced. Test with your group monthly. Establish common channels and encryption keys.
  • HAM frequencies — Program 5-10 common emergency frequencies into your handheld. Include local repeaters, National Weather Service, and simplex channels.
  • Emergency meeting points — Designate 3 meeting points: immediate vicinity (e.g., coffee shop 2 blocks away), neighborhood (1 mile), city-wide (10 miles).
  • Out-of-area contact — Designate one person outside your region. Everyone calls/texts them to check in. Long-distance calls often work when local circuits are jammed.
  • Backup power for comms — Keep a power bank dedicated to communication devices. Rechargeable batteries for handheld radios. Solar charger for extended outages.

Gear Bag Recommendations

Your digital emergency kit should be grab-and-go ready. Here is what goes in the bag:

  • Primary USB drive (128 GB+) — In USB-A and USB-C form factor (use adapter)
  • Backup USB drive — Smaller (64 GB), stored in a different compartment
  • SSD external drive — 1-2 TB portable SSD for full system backups (Samsung T7, SanDisk Extreme)
  • USB-C hub — With HDMI, USB-A, SD card, Ethernet for connecting to any machine
  • Network gear — Portable router (GL.iNet), Ethernet cable, Wi-Fi adapter
  • Power bank — 20,000 mAh minimum with USB-C PD
  • Cables — USB-C to C, USB-C to A, Lightning, micro-USB, charging brick
  • Notebook + pen — For writing down recovery codes, network configs, contact info

Cloud Backup Verification

Cloud backups can silently fail. Verify them systematically:

# Monthly cloud backup verification procedure:
# 1. Log into cloud backup provider dashboard
# 2. Check last successful backup timestamp
# 3. Verify backup size roughly matches expected
# 4. Download 3 random files, verify their checksums
# 5. Check that deleted local files are still in backup (version history)
# 6. Verify encryption: download a file, decrypt with your key
# 7. Confirm billing is current (expired account = no backups)

# Automated monitoring with healthchecks.io:
# - Set backup script to ping healthchecks.io on completion
# - Get alert if ping is missed (email/SMS)
Critical: Cloud backup is NOT the same as cloud sync. Services like Dropbox/OneDrive/Google Drive sync deletions — ransomware can propagate through sync. Use proper backup software with versioning (Backblaze, Arq, Duplicati).

Digital Inheritance Preparation

Digital inheritance ensures your accounts and assets can be accessed by trusted people after you are gone:

  1. Create a digital will document — List all online accounts, approximate value/importance, and instructions. Store encrypted in password manager.
  2. Bitwarden Emergency Access — Designate 1-2 trusted contacts. Set a 7-day waiting period. They get full vault access if you do not respond.
  3. Apple Legacy Contact — In iCloud settings, designate someone who can request access to your Apple data after death.
  4. Google Inactive Account Manager — Set up to notify trusted contacts after 3-6 months of inactivity.
  5. Crypto/seed phrase inheritance — Split seed phrase into 2-of-3 or 3-of-5 multisig shares. Store in different locations. Include instructions in your will.
  6. Physical envelope — Contains master password, 2FA recovery codes, phone unlock code, and instructions. Seal, sign across seal, store in lawyer's office or safe deposit box.

Review digital inheritance setup yearly as part of your annual drill.
