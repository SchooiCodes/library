Offline Communication

Communicate without internet or cellular - Briar mesh, Meshtastic devices, LoRa radio, goTenna, and ham radio basics.

Briar (Mesh Messaging)

Briar is an open-source messaging app using Bluetooth, Wi-Fi Direct, and Tor - no internet needed for local mesh.

Use cases: Protests, disaster zones, remote areas, internet shutdowns.

# Install from briarproject.org (F-Droid)
# Create account (no phone/email)
# Add contacts by scanning QR code
# Messages sync automatically when in range

Limitation: Android only.

Meshtastic Devices

Meshtastic is open-source long-range mesh on LoRa radios. Range: 1-3 km urban, 10-30 km line-of-sight.

Features: 868/915 MHz ISM band (license-free), AES-256 encrypted, mesh relay, standalone or via Bluetooth to phone.

Hardware: LilyGO T-Beam, Heltec Wireless Tracker, RAK WisBlock. $15-60 per node.

goTenna

goTenna creates a private mobile mesh network. More plug-and-play than Meshtastic but proprietary.

Models: goTenna Mesh (~$200/pair, 1-6 km), goTenna Pro (ruggedized).

vs Meshtastic: goTenna is easier but closed-source and more expensive. Meshtastic is cheaper and open-source.

LoRa Radio

LoRa (Long Range) is the radio modulation behind Meshtastic. EU: 868 MHz, US: 915 MHz, license-free.

Spreading Factor: SF7 = fastest/shortest (~1 km urban), SF12 = slowest/longest (~10+ km urban).

Antenna is critical. Good antenna doubles range.

Ham Radio Basics

Ham radio is the most resilient communication - no towers or internet needed.

# Licensing (USA): Technician (VHF/UHF, 2-4 weeks)
# Emergency frequencies:
# 146.520 MHz - VHF simplex calling
# 446.000 MHz - UHF simplex calling
# Starter: Baofeng UV-5R (~$25)

Meshtastic Setup

# 1. Connect ESP32 board via USB
# 2. Flash firmware: https://flasher.meshtastic.org
# 3. Configure via web UI: set region, node name, GPS
# 4. Power: 18650 battery or USB power bank
# 5. Test with a second node 1 km away
