PortableApps.com Platform

PortableApps.com is the gold standard for portable Windows software. It provides a menu launcher and a curated catalog of 400+ apps that run entirely from a USB drive without installation.

  • Platform features: App store, automatic updates, backup/restore of settings, search, and a lightweight menu system.
  • How it works: Download the PortableApps.com Platform installer, point it to your USB drive, and use the built-in app store to install portable apps. Each app stores its settings in its own folder — no registry entries, no system files.
  • Multi-device sync: Use the backup feature to save app settings. Restore to any USB drive. This makes it easy to clone your emergency USB.
# Install PortableApps.com Platform
# 1. Download from https://portableapps.com/download
# 2. Run the installer, select your USB drive (e.g., E:\)
# 3. The platform menu will appear in the system tray
# 4. Click the platform icon → "Apps" → "Browse for more apps"
# 5. Browse categories, click "Install" on any app

# PortableApps.com directory structure:
# X:\PortableApps\     - Apps installed here
# X:\PortableApps\Platform\ - Platform files
# X:\Documents\       - Default document folder
# X:\Backup\        - Backup files

# To move to a new USB drive:
# Copy entire PortableApps folder → run PortableApps.com Platform installer
# Point to new drive → platform detects existing apps

Alternatives: LiberKey (similar platform, European-focused), Lupo PenSuite (Italian, large collection), or manual portable app collection (download .exe/.AppImage files directly).

Tip: Even if you don't use the PortableApps.com launcher, the individual portable apps can be downloaded separately and run standalone. The launcher just adds convenience.

Media Players and Editors

Watch videos, listen to audio, and do basic media editing without installing anything:

  • VLC Media Player Portable (~40 MB) — Plays anything: MP4, MKV, AVI, MP3, FLAC, DVD ISOs, network streams. No codec packs needed.
  • MPC-HC Portable (~20 MB) — Media Player Classic. Lighter than VLC, excellent video playback. Good for old/low-end hardware.
  • Audacity Portable (~30 MB) — Audio recording and editing. Record from microphone, edit audio files, convert formats.
  • foobar2000 Portable (~15 MB) — Lightweight music player and tag editor. Handles huge libraries, supports many audio formats.
  • HandBrake Portable (~80 MB) — Video transcoding. Convert video files between formats. Useful for making videos playable on other devices.
  • OBS Studio Portable (~150 MB) — Screen recording and streaming. Record tutorials, capture evidence, or stream when connectivity allows.
# VLC Portable command-line tricks
# Take a screenshot from a video:
vlc video.mp4 --video-filter=scene --vout=dummy \
 --scene-ratio=60 --scene-path=./screenshots/ vlc://quit

# Convert media files:
vlc input.mp4 --sout=#transcode{vcodec=h264,vb=800,acodec=mp3,ab=128}:standard{mux=mp4,dst=output.mp4} vlc://quit

# Stream a video over the network:
vlc video.mp4 --sout="#http{mux=ts,dst=:8080/}" --no-sout-all

For emergency situations, VLC is the single most important media tool. It can even repair partially damaged video files and play incomplete downloads.

Security and Password Tools

These portable security tools protect your data, manage credentials, and clean infected systems:

  • KeePassXC Portable (~30 MB) — Offline password manager. Encrypted database with 2FA/TOTP support, password generator, browser integration. Your digital keys.
  • VeraCrypt Portable (~15 MB) — On-the-fly disk encryption. Create encrypted containers, encrypt USB drives, hide volumes. Essential for securing portable data.
  • Malwarebytes Portable (~100 MB) — On-demand malware scanner. Run from USB to clean an infected system without booting into it.
  • ClamWin Portable (~150 MB) — Open-source antivirus. Signature-based detection, scheduled scanning, virus database updates via USB.
  • Eraser Portable (~5 MB) — Secure file deletion. Permanently erase sensitive files beyond recovery. Gutmann, US DoD 5220.22-M, and other standards.
# VeraCrypt portable — create an encrypted container
# 1. Run VeraCrypt Portable (requires admin for driver install)
# 2. "Create Volume" → "Standard VeraCrypt Volume"
# 3. Choose file location (e.g., E:\encrypted.tc)
# 4. Select encryption: AES-256 + SHA-512
# 5. Set size (e.g., 2 GB for tools + docs)
# 6. Set password (use a strong, memorable passphrase)
# 7. Format → Done

# Mount on any Windows machine without installation:
# E:\VeraCrypt\VeraCrypt.exe /q /v E:\encrypted.tc /l Z /p "yourpassword"
# Or use keyfiles stored on the same USB
Critical: Keep a backup of your KeePassXC database (separate from your main USB) and store the VeraCrypt container keyfile in a safe place. If your emergency USB is lost or damaged, you need a way to recover your passwords.

Development and Coding

Write, edit, and compile code from any machine with these portable development tools:

  • VS Code Portable (~250 MB) — Full IDE. Extensions, terminal, Git integration, debugging. Portable version stores settings in the app folder.
  • Sublime Text Portable (~20 MB) — Lightning-fast editor. Great for quick edits, regex search/replace, multiple cursors.
  • Notepad++ Portable (~10 MB) — Lightweight code editor with syntax highlighting for 80+ languages. The ultimate quick-edit tool.
  • Git Portable (~50 MB) — Git version control with bash, SSH client, and all standard Git commands. Clone repositories locally or over SSH.
  • Python Portable (~100 MB) — Full Python interpreter and standard library. Run scripts, start a web server, parse data — all from USB.
  • Node.js Portable (~50 MB) — JavaScript runtime. npm package manager. Run JavaScript tools, servers, and scripts offline.
  • ConEmu / Cmder (~5 MB) — Portable terminal emulator. Tabbed console, split panes, custom color schemes. Much better than cmd.exe.
# VS Code Portable — extensions pre-loaded
# Place extensions in: PortableApps\VSCode\data\extensions\
# Settings in: PortableApps\VSCode\data\user-data\User\settings.json

# Pre-load these extensions for offline use:
# - ms-python.python
# - dbaeumer.vscode-eslint
# - esbenp.prettier-vscode
# - GitHub.copilot (requires internet eventually)
# - ms-vscode.live-server (local HTTP server)

# Start Python HTTP server from USB:
cd E:\Projects
python -m http.server 8000
# Browse at http://localhost:8000

# Git portable — clone repos to USB:
git clone --depth 1 https://github.com/user/repo.git
# --depth 1 = latest commit only (faster, smaller)

For maximum portability, pre-download any npm packages or Python libraries you might need. Store them on the USB in a portable package cache.

System Utilities and Diagnostics

These portable utilities diagnose hardware issues, recover data, and manage system settings:

  • CrystalDiskInfo Portable (~5 MB) — Hard drive health monitoring. Check S.M.A.R.T. status, temperatures, and error counts for SSDs and HDDs.
  • CrystalDiskMark Portable (~5 MB) — Disk speed benchmark. Test read/write speeds. Useful for verifying USB drive performance.
  • HWiNFO Portable (~50 MB) — Comprehensive system information. CPU, GPU, motherboard, sensors, BIOS version. Monitor temperatures and voltages.
  • CPU-Z Portable (~5 MB) — CPU, motherboard, and memory details. Lightweight, fast, no installation.
  • GPU-Z Portable (~5 MB) — Graphics card information. Monitor GPU temperature, clock speed, memory usage, and driver version.
  • Process Explorer Portable (~3 MB) — Advanced task manager with VirusTotal integration. Identify suspicious processes and open file handles.
  • Autoruns Portable (~3 MB) — See everything that runs at startup. Disable malware persistence mechanisms.
  • Recuva Portable (~5 MB) — File recovery for deleted files. Works on USB drives, memory cards, and hard drives.
# Quick system health check with portable tools
# Run in order:

# 1. HWiNFO → Sensors → Check CPU/GPU/SSD temps
# 2. CrystalDiskInfo → Check S.M.A.R.T. health status
# 3. Process Explorer → Look for suspicious processes
# 4. Autoruns → Check startup entries
# 5. Recuva → Recover recently deleted files if needed

# Process Explorer — check for malware:
# Run → VirusTotal.com → Check → Submit unknown files
# Look for: unsigned processes, suspicious names, no description

Keep all these utilities on a single USB. When a friend says "my computer is slow" or "I think I have a virus," this toolkit can diagnose and fix without installing anything on their machine.

File Management and Archiving

Essential tools for managing, compressing, and securely transferring files:

  • 7-Zip Portable (~2 MB) — Archive manager. Opens ZIP, RAR, 7z, TAR, GZ, BZ2, XZ, ISO, and 30+ other formats. High compression ratio (7z format).
  • Double Commander Portable (~15 MB) — Two-pane file manager (like Total Commander/NC). Tabbed browsing, built-in viewer, file comparison, FTP client.
  • FreeFileSync Portable (~20 MB) — Folder comparison and synchronization. Mirror, update, or two-way sync. Essential for backing up data to external drives.
  • Everything Portable (~3 MB) — Instant file search. Indexes NTFS volumes in seconds. Find any file on any drive by name instantly.
  • HashCheck (or HashTab) (~1 MB) — File integrity checking. Compute MD5, SHA-1, SHA-256 hashes. Verify downloaded files are not corrupted.
# 7-Zip portable — command-line archive operations
# Create a self-extracting archive:
7z a -sfx archive.exe "E:\PortableApps\Tools\*"

# Split large archive into volumes:
7z a -v100m archive.7z "E:\Documents\*"
# Creates: archive.7z.001, archive.7z.002, ...

# Encrypt with AES-256:
7z a -p -mhe=on secret.7z "E:\Sensitive\*"
# -p = prompt for password, -mhe=on = encrypt file names

# Everything portable — remote indexing
# Create an .efu file list of a drive:
everything.exe -create-filelist C:\filelist.efu
# Search file lists without the drive connected

Everything Portable is arguably the most underrated tool in this list. When you're on a borrowed computer and need to find a specific file across multiple drives, it's indispensable.

Graphics and Design Tools

Portable image editing, screen capture, and design tools for when you need to work with visuals:

  • GIMP Portable (~200 MB) — Full-featured image editor. Photoshop alternative with layers, masks, filters, color correction. Extensible with plugins.
  • IrfanView Portable (~5 MB) — Ultra-lightweight image viewer and basic editor. Batch convert/resize/rename. Opens 80+ image formats. Indispensable for quick image tasks.
  • Greenshot Portable (~10 MB) — Screen capture with annotation tools. Capture full screen, window, or region. Add arrows, text, blur, highlights.
  • Paint.NET Portable (~20 MB) — Windows image editor (requires .NET). Easier than GIMP, more powerful than Paint. Layers, effects, plugins.
  • ShareX Portable (~15 MB) — Screen capture, recording, file upload. OCR, color picker, QR code generator, image editor built in.
  • Inkscape Portable (~200 MB) — Vector graphics editor. SVG native format. Create diagrams, logos, illustrations. Alternative to Adobe Illustrator.
# IrfanView — batch convert images
# Command line:
i_view32.exe E:\Photos\*.jpg /convert=E:\Converted\*.png

# Greenshot — quick screenshot to file
# Run Greenshot Portable → Print Screen
# Capture region → annotate with arrows/text → save as PNG
# Output to: clipboard, file, printer, or email

For most tasks, IrfanView + Greenshot covers 90% of image needs. Keep GIMP for serious editing work.

Office and Productivity

Read, edit, and create documents without Microsoft Office:

  • LibreOffice Portable (~400 MB) — Full office suite. Writer (Word), Calc (Excel), Impress (PowerPoint), Draw, Base, Math. Opens all Microsoft Office formats.
  • OnlyOffice Portable (~300 MB) — Lightweight office suite with better Microsoft Office compatibility. Clean interface, collaborative editing.
  • SumatraPDF Portable (~10 MB) — PDF, ePub, MOBI, CBZ/CBR (comic books), DjVu reader. Minimalist, fast, keyboard-friendly.
  • Foxit Reader Portable (~50 MB) — Full-featured PDF reader. Annotations, form filling, signatures. Lighter than Adobe Acrobat.
  • Calibre Portable (~200 MB) — Ebook library manager. Convert between formats (EPUB, MOBI, PDF, AZW3), sync to e-readers, metadata editing.
  • PDFCreator Portable (~50 MB) — Virtual PDF printer. "Print" any document to PDF from any application. Essential for document sharing.
# SumatraPDF — the ultimate lightweight reader
# Opens PDF, ePub, MOBI, CBZ, CBR, DjVu, CHM, XPS
# Portable, no installation, ~10 MB
# Settings stored in sumatra-settings.txt (in app folder)
# Command line:
SumatraPDF.exe document.pdf -page 42    # Open to page 42
SumatraPDF.exe -search "keyword" doc.pdf  # Search on open

# LibreOffice — batch convert documents
soffice.exe --headless --convert-to pdf *.docx
soffice.exe --headless --convert-to csv *.xlsx
soffice.exe --headless --convert-to pptx *.ppt

LibreOffice Portable is large but comprehensive. If space is tight, SumatraPDF + a text editor covers reading needs, and the online/cloud versions of Google Docs or Office.com can fill gaps when internet is available.

Pro tip: Use a high-quality USB 3.2 drive (e.g., Samsung BAR Plus, SanDisk Extreme Pro) for best performance. A 128GB drive costs ~$15-25 and can hold the entire Tier 1-5 stack with room for personal files.

# Emergency USB setup checklist
# ___ Format drive as NTFS or exFAT (supports files >4GB)
# ___ Install PortableApps.com Platform
# ___ Install core portable apps (Tier 1-3)
# ___ Create VeraCrypt encrypted container for sensitive data
# ___ Set up KeePassXC database with all passwords
# ___ Download ZIM archives (Tier 5)
# ___ Copy DevDocs Desktop + Zeal + docsets
# ___ Copy Python + VS Code + Git portable
# ___ Add personal documents (scanned ID, insurance, medical records)

# ___ Test that all portable apps run without admin rights
# ___ Label the drive with a physical tag
# ___ Make a second backup drive and store off-site
You are now prepared. With this USB drive, you can browse, code, edit, diagnose, recover, and create on any Windows computer — without leaving a trace. Store it in your go-bag, update it quarterly, and you'll never be caught without your digital toolkit.

On-the-Go Networking Tools

Portable networking tools for diagnosing network issues, scanning ports, and analyzing traffic:

  • Wireshark Portable (~50 MB) — Network protocol analyzer. Capture and inspect packets, filter by protocol, follow TCP streams, identify suspicious traffic. Run from USB to analyze a network without installing.
  • Nmap Portable (~20 MB) — Network discovery and security scanning. Scan open ports, identify OS versions, detect running services. Zenmap provides a GUI. Essential for mapping a network.
  • Advanced IP Scanner (~5 MB) — Fast LAN scanner. Shows all devices on your network, their IP addresses, MAC addresses, and shared folders. One-click access to RDP and network shares.
  • Angry IP Scanner (~2 MB) — Cross-platform IP/port scanner. Java-based, works on Windows, Linux, macOS. Portable, configurable, fast.
  • Netcat Portable (~1 MB) — The Swiss Army knife of networking. Scan ports, transfer files, create backdoors (legitimate for testing). nc -lvnp 4444 for listening.
  • PuTTY Portable (~5 MB) — SSH, Telnet, serial console. Pre-configure sessions for your servers and store them on the USB.
  • KiTTY Portable (~5 MB) — Fork of PuTTY with additional features: auto-login, session filtering, port forwarding manager.

Password Cracking Tools

Portable password recovery and cracking tools for testing password strength and recovering lost credentials:

  • John the Ripper Portable (~20 MB) — Fast password cracker. Supports cracking Windows LM/NT hashes, Linux shadow files, macOS passwords, ZIP/RAR archives. john --format=nt hashes.txt
  • hashcat Portable (~30 MB) — The fastest password recovery tool. Uses GPU acceleration (OpenCL/CUDA). Cracks MD5, SHA1/256/512, bcrypt, scrypt, and 300+ hash types. hashcat -m 1000 -a 3 hashes.txt ?a?a?a?a?a?a?a?a
  • Ophcrack (~10 MB) — Windows password cracker using rainbow tables. Finds simple passwords in seconds. Boot from the Ophcrack Live CD for physical access.
  • Hash Suite (~15 MB) — Windows-focused password auditor. GUI-based, supports LM/NT hashes, checks against common passwords and dictionary attacks.
# hashcat basic usage:
hashcat -m 1000 -a 3 hashes.txt ?d?d?d?d?d?d?d?d
# -m 1000 = NTLM hash mode
# -a 3 = brute force attack
# ?d?d?d?d?d?d?d?d = 8 digits (0-9)
# Output: cracked passwords are shown in the console

# John the Ripper on Linux shadow:
unshadow /etc/passwd /etc/shadow > combined.txt
john combined.txt
john --show combined.txt

Virtual Machine Tools

Run entire operating systems from your USB without affecting the host:

  • VirtualBox Portable (~200 MB) — Full virtualization platform. Run VMs from USB. Pre-configure VMs for different tasks (Windows XP for legacy apps, Kali for testing, Ubuntu for development). Store VM disk files on external SSD for performance.
  • QEMU Portable (~50 MB) — Lightweight emulator/virtualizer. Faster than VirtualBox for Linux VMs. Supports KVM acceleration when available. qemu-system-x86_64 -hda ubuntu.img -m 2048
  • VMware Player Portable (~300 MB) — Requires extraction from installer. Somewhat portable but may leave traces. Good for running existing .vmx virtual machines.

Forensic Tools

Digital forensics tools for evidence acquisition and analysis:

  • Autopsy Portable (~200 MB) — Digital forensics platform. Analyze disk images, recover deleted files, extract metadata, generate timeline analysis. GUI-based, extensible with modules.
  • FTK Imager Portable (~30 MB) — Forensic imaging tool. Create forensic disk images (E01, AFF, DD), preview drive contents, export files, capture RAM dumps. Essential for evidence preservation.
  • Guymager (~5 MB) — Linux forensic imager. Create dd, E01, AFF images with hardware write-blocker support. Included in many Linux forensic distros.
  • Bulk Extractor (~10 MB) — Extracts features from digital media without parsing filesystem. Finds email addresses, credit cards, URLs, phone numbers in raw disk images.

Additional Communication Tools

Staying connected when traditional networks are down or monitored:

  • WireGuard Portable (~10 MB) — Modern VPN protocol. Pre-configure profiles for your home server or commercial VPN provider. One-click connect to encrypt all traffic. Much faster than OpenVPN.
  • Signal Portable (~100 MB) — Encrypted messaging. Requires phone for initial registration. Desktop app syncs with phone for continued use. End-to-end encrypted voice and video calls.
  • Element (Matrix) Portable (~150 MB) — Decentralized encrypted communication. Works without phone number. Create accounts on any Matrix server. Supports rooms, direct messages, file sharing.
  • Briar Portable (~20 MB) — Peer-to-peer encrypted messaging over Bluetooth or Wi-Fi Direct. No internet required. Ideal for local mesh networks in emergencies.
  • Tor Browser Portable (~200 MB) — Anonymous browsing. Essential for accessing .onion sites and bypassing censorship. Pre-configure with bridges for restricted networks.

Disk Wiping Tools

Secure disk erasure tools for permanently destroying data before disposing of drives:

  • DBAN (Darik's Boot and Nuke) (~20 MB) — Bootable disk wiper. Performs DoD 5220.22-M compliant wiping. Supports HDDs and SSDs. Boot from USB, select drive, start wipe. Simple and effective.
  • nwipe Portable (~5 MB) — Fork of DBAN that runs from Linux. More control over wipe methods: Gutmann (35 passes), DoD 5220.22-M (3 passes), PRNG stream (1 pass for SSDs).
  • Eraser Portable (~5 MB) — Windows-based secure deletion. Erase individual files, folders, or entire drives. Supports Gutmann, DoD, and custom wipe patterns.
  • HDD Low Level Format Tool (~5 MB) — Writes zeros to every sector. Often sufficient for HDDs (data cannot be recovered after a full zero write). Fast but not military-grade.
# nwipe — wipe a drive from Linux live USB:
sudo nwipe /dev/sda
# Interactive menu → choose drive → choose method
# Methods:
# - DoD Short: 3 passes (random, zero, verify)
# - Gutmann: 35 passes (very thorough, slow)
# - PRNG: 1 pass using random data (fast, sufficient for SSDs)
#
# WARNING: This permanently destroys all data on the drive!
# Triple-check the device name before confirming.

Hardware Monitoring Expansion

Additional hardware diagnostic tools beyond the basics:

  • MemTest86+ Portable (~5 MB) — Bootable memory tester. Download the ISO, add to Ventoy USB. Boot and run overnight to detect faulty RAM. Essential for diagnosing blue screens and crashes.
  • Prime95 Portable (~5 MB) — Stress testing tool. Run CPU and memory at maximum load to test stability and cooling. Use after hardware changes.

  • HWMonitor Portable (~5 MB) — Reads system sensors. Simplified version of HWiNFO. Shows min/max values for all sensors.

Additional Security Scanners

Beyond the standard AV tools, add these specialized scanners to your toolkit:

  • ESET Online Scanner (~10 MB) — On-demand portable scanner. Download the latest version each time (daily definition updates). Detects malware other scanners miss.
  • HitmanPro Portable (~15 MB) — Behavior-based detection. Scans without signatures — looks at file behavior. Catches zero-day malware. Free trial scans once.
  • KVRT (Kaspersky Virus Removal Tool) (~200 MB) — Portable trojan remover. Updates definitions on launch. Strong against trojans, worms, and rootkits.
  • Emsisoft Emergency Kit (~200 MB) — Portable scanner with dual-engine (Bitdefender + Emsisoft). Command-line and GUI modes. Free for emergency use.
  • Sophos Scan & Clean (~100 MB) — Portable scanner. No installation. Strong against network worms and ransomware.

Eduroam / Captive Portal Access Tools

When you need to connect to restricted networks (universities, hotels, public Wi-Fi) from a borrowed computer:

  • eduroam CAT (Configuration Assistant Tool) — Download the eduroam connection wizard for your home institution. Run it on any machine to auto-configure eduroam network settings. Stores the configuration for reuse.
  • Captive Portal Bypass Kit — Keep a text file with instructions for common captive portal workarounds: http://captive.apple.com, http://nmcheck.gnome.org, http://google.com/generate_204. These URLs trigger captive portal login pages on most networks.
  • MAC address spoofer — Some captive portals limit free access by MAC. Use Technitium MAC Address Changer (portable) to change your MAC address and get a fresh session.
  • Wi-Fi profile export — On Windows: netsh wlan show profiles → netsh wlan export profile. Import on another machine: netsh wlan add profile. Store Wi-Fi profiles for networks you use regularly.
# Export Wi-Fi profiles from Windows:
netsh wlan show profiles
netsh wlan export profile name="MyNetwork" folder=E:\WiFiProfiles\ key=clear
# Import on another machine:
netsh wlan add profile filename="E:\WiFiProfiles\MyNetwork.xml"

# Test captive portal connectivity:
curl -I http://captive.apple.com
curl -I http://nmcheck.gnome.org
curl -I http://httpbin.org/get
