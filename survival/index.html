Digital Survival Kit

When Things Go Wrong

Emergency resources, offline tools, system rescue, and security breach response — your digital go-bag when everything hits the fan.

Prepare before disaster strikes. Bookmark this page now. Download portable apps and save them to a USB drive. Keep offline copies of critical tools. When the internet goes down or your system fails, you'll thank yourself.

Emergency / Survival Resources

When things break — hard drive failure, ransomware, corrupted files, lost passwords, network outage — these are the first tools you reach for. This page is organized by scenario. Scan each section now and prep what you can.

Immediate Response

System crashing? Suspect malware? Disconnect from the internet immediately, boot into safe mode, and run offline scans. Do not plug in unknown USB drives.

Backup First

Before attempting any recovery operation, image the failing drive with Clonezilla or dd. Work on the copy, not the original. One wrong command can destroy data permanently.

Password Emergencies

Use chntpw to reset Windows passwords from a Linux live USB. Use Bitwarden Emergency Access for vault inheritance. Keep a physical copy of your 2FA recovery codes.

BIOS / Firmware Backup

Back up your motherboard/chromebook BIOS before flashing. Use tools like flashrom or a hardware programmer to dump the firmware. Store multiple copies of the backup.

Communication Last Resort

If the internet is cut but local mesh is active, use Briar or Meshtastic devices. Signal works over any data connection. Tor can bypass censorship when standard routes are blocked.

Emergency Communication

When normal communication channels are down, blocked, or compromised, these tools keep you connected.

Signal

End-to-end encrypted messaging and calls. Open-source, audited protocol. Requires phone number for registration but offers disappearing messages, screen security, and sealed sender. Gold standard for private communication.

Download

Session

Encrypted messaging with no phone number required. Uses onion routing and does not collect metadata. Register with a random ID. Ideal for anonymity-sensitive situations.

Download

Element / Matrix

Decentralized, open-source communication protocol. You control your data. Self-host your own server or use the Matrix.org homeserver. Bridges to Slack, Discord, Telegram. End-to-end encryption available.

Download

Tor Browser

Anonymous browsing through the onion routing network. Circumvents censorship, hides your IP, and provides access to .onion sites. Critical for accessing information in restricted environments.

Download

WireGuard VPN

Secure VPN tunnel. Minimal codebase (security), blazing fast, works over unreliable networks. Set up your own server on a VPS or use a provider. Essential for encrypted communication when on untrusted networks.

Data Recovery & Backup

Accidental deletion, drive corruption, and ransomware are not a matter of if but when. These tools give you the best chance of getting your data back.

Recuva

File recovery for Windows. Recovers deleted files from hard drives, USB drives, memory cards, and even MP3 players. Deep scan mode finds files that have been deleted for weeks. Free version is powerful enough for most cases.

Download

TestDisk & PhotoRec

TestDisk recovers lost partitions and fixes boot sectors. PhotoRec recovers 480+ file types from virtually any storage media. Both are free, open-source, and work on Windows, macOS, Linux. Run from a live USB for best results.

Download

rsync / FreeFileSync

rsync is the gold standard for Unix file synchronization and backup. Incremental, over SSH, preserves permissions. FreeFileSync provides a GUI and is cross-platform. Both are free and essential for any backup strategy.

rsync | FreeFileSync

Restic / Borg

Encrypted, deduplicated backups. Restic supports multiple backends (local, S3, SFTP, REST server). Borg is Python-based, highly efficient with deduplication. Both encrypt your backup data client-side. Speed + security.

Restic | Borg

Clonezilla

Disk imaging and cloning. Create a bit-for-bit image of any drive. Boot from Clonezilla live media, save images to network shares or external drives. Restore entire systems in minutes. Essential before any recovery attempt.

Download

Windows File History

Built-in Windows backup. Set it up once and forget it. Backs up files to an external drive or network location. Restore previous versions of files from File Explorer. Not as powerful as dedicated tools, but already installed.

Security Breach Emergency

You clicked something you shouldn't have. An account is compromised. Ransomware is encrypting your files. Here's your incident response checklist.

Response Checklist

  1. Disconnect — Pull the ethernet cable and turn off Wi-Fi immediately. If on a phone, enable airplane mode.
  2. Boot to safe mode — Restart holding Shift (Windows) or boot from a rescue USB. Do not let the malware load.
  3. Scan offline — Run Malwarebytes and AdwCleaner from safe mode or a rescue environment.
  4. Check processes — Use Process Explorer and Autoruns to find suspicious entries.
  5. Check connections — Use TCPView to see if malware is phoning home. Block IPs at the firewall.
  6. Change credentials — Use a clean device to change passwords. Enable 2FA everywhere.
  7. Restore from backup — Only after the malware is confirmed removed. Scan backup files before restoring.
  8. Report — File a report with local cybercrime authorities and notify your bank if financial data is involved.

Malwarebytes

Industry-standard malware removal. Free version scans and removes malware on demand. Premium provides real-time protection. Portable version runs from USB without installation. Run in safe mode for deep cleaning.

Download

AdwCleaner

Targeted adware, PUP, and browser hijacker removal. Fast scan, picks up what Malwarebytes might miss. Free, no installation required. Reset browser settings, remove unwanted toolbars, and kill persistent adware.

Download

Process Explorer

Advanced task manager from Sysinternals. Shows every running process, open handles, and DLLs loaded. Identify suspicious processes with VirusTotal integration. Replace Task Manager with this tool during investigations.

Download

Autoruns

Shows everything that runs at system startup. Processes, services, scheduled tasks, drivers, browser extensions, and more. Disable or delete suspicious entries. Essential for finding persistence mechanisms.

Download

TCPView

Real-time TCP/UDP endpoint monitoring. See every connection your system is making. Identify unknown IPs and processes phoning home. Close suspicious connections immediately. Free from Microsoft Sysinternals.

Download

Bitwarden Emergency Access

Designate emergency contacts who can request access to your Bitwarden vault. Set a waiting period (1-7 days). If you don't deny the request, they get full access. Critical for ensuring family can reach accounts.

Emergency Offline Communication

  • Briar — Android messaging app that works over Bluetooth, Wi-Fi direct, or Tor. No internet required for local mesh.
  • Meshtastic — Long-range, low-power mesh networking over LoRa radio. Encrypted text messaging without cell service or internet.
  • goTenna / beartooth — Consumer mesh devices that create off-grid networks for phones.
  • Amateur radio (HAM) — Obtain a license and set up a handheld radio. Can reach across the world without any infrastructure.

Mesh Networking Basics

A mesh network lets devices communicate directly without a central router or internet connection. Each device acts as a node, forwarding data to others. Useful when infrastructure is damaged or controlled.

  • Protocols: BATMAN-adv, OLSR, 802.11s (Wi-Fi mesh), LoRa (long range, low bandwidth)
  • Software: LibreMesh (router firmware), Yggdrasil (IPv6 mesh), CJDNS (encrypted mesh)
  • Hardware: Routers with open firmware (TP-Link, Ubiquiti), single-board computers (Raspberry Pi), LoRa modules (Heltec, TTGO)
  • Community networks: NYC Mesh, Guifi.net, Freifunk — learn from existing deployments
# Quick mesh test with two Linux machines (ad-hoc Wi-Fi)
sudo ip link set wlan0 down
sudo iwconfig wlan0 mode ad-hoc
sudo iwconfig wlan0 essid "emergency-mesh"
sudo ip link set wlan0 up
sudo ip addr add 10.0.0.1/24 dev wlan0
# On second machine: sudo ip addr add 10.0.0.2/24 dev wlan0
# Now ping 10.0.0.2 — you have a mesh!

Travel Tech Survival

When traveling, you're at your most vulnerable — unfamiliar networks, device theft risk, and limited access to backups. Prepare before you leave.

VPN Before Departure

Install and test your VPN before you travel. Some countries block VPN websites, making it impossible to download after arrival. Set up WireGuard at home and test the connection. Keep a backup OpenVPN config.

Device Preparation

Factory reset your phone before crossing borders if you carry sensitive data. Use a "travel phone" with minimal apps. Enable device encryption, set strong PINs, disable biometric unlock at border crossings.

Digital Finance Prep

Carry multiple payment methods (credit card, debit, cash). Pre-load a Revolut or Wise card with small amounts. Store 2FA backup codes offline. Know your bank's international helpline number.

Offline Data Kit

Download offline maps (Organic Maps), translate packs (Google Translate offline), your password database (KeePassXC), important documents (passport scan, insurance), and emergency contacts — all stored on encrypted USB.

Border Crossing Protocol

Power off devices before border checkpoints. Use encrypted devices only. Consider a "burner phone" for travel. Upload sensitive data to cloud before crossing, then download on the other side. Use Veracrypt hidden volumes for deniability.

Remote Wipe Plan

Enable Find My Device (Android) or Find My (iPhone). Set up remote wipe capability. Keep a cloud backup that you can access from any device. If your device is stolen, wipe it immediately from a friend's phone.

Password Change Emergency Checklist

If you suspect any account has been compromised, follow this order strictly. Use a clean, trusted device — do not change passwords on a potentially infected machine.

  1. Email accounts first — If an attacker controls your email, they can reset every other password. Change email passwords immediately.
  2. Password manager — Change your master password and check for any new, unknown entries.
  3. Financial accounts — Banking, PayPal, credit cards, crypto exchanges, investment accounts.
  4. Social media — Facebook, Twitter/X, Instagram, LinkedIn, Discord, Telegram, Signal. Check active sessions.
  5. Cloud storage — Google Drive, iCloud, OneDrive, Dropbox. Check for file changes you didn't make.
  6. Domain & DNS — If you own domains, check that registrar and DNS settings haven't been changed.
  7. Device logouts — Use "sign out of all devices" on every service. Revoke all session tokens.
  8. Enable 2FA — Add two-factor authentication to every service that supports it. Use an authenticator app, not SMS if possible.
  9. Check forwarding rules — Attackers often set up email forwarding to silently read your mail. Check all forwarding rules.
  10. Monitor — Watch your accounts for suspicious activity for the next 30 days. Set up alerts for new logins.
Save this checklist offline. Print a copy and keep it in your emergency kit. When you're panicking during a breach, you won't want to search for instructions — you'll want a piece of paper telling you exactly what to do.

Quick Reference: Emergency Boot Flow

When your computer won't start, follow this decision tree:

1. Power on → no POST beep?
  → Remove all peripherals, reseat RAM, clear CMOS

2. POST but no boot?
  → Enter BIOS → check boot order → verify drive is detected

3. "Operating System not found"?
  → Boot from rescue USB → check partition table (TestDisk)
  → Check if Boot Configuration Data is corrupt (bcdedit /rebuild)

4. Windows loads but crashes?
  → Safe Mode (F8 or Shift+Restart) → recent update uninstall
  → System Restore → chkdsk /f C: → sfc /scannow

5. Bluescreen / Kernel panic?
  → Note the error code → boot rescue media → check drive health
  → Restore from backup or reinstall

6. Ransomware screen?
  → IMMEDIATELY disconnect power (hold power button 10s)
  → Boot from rescue USB → do not touch the infected drive
  → Use decryption tools (nomoreransom.org)
   → Restore from backup
