Password Reset & Recovery

Reset or recover passwords on Windows, Linux, and macOS - plus Bitwarden emergency access and 2FA recovery codes.

Table of Contents

  • Windows Password Reset (chntpw)
  • Linux Password Reset
  • macOS Password Reset

Windows Password Reset (chntpw)

chntpw is the most reliable tool for resetting local Windows passwords.

# Boot from SystemRescue or Ubuntu Live
sudo mount /dev/sda2 /mnt/windows
cd /mnt/windows/Windows/System32/config
sudo chntpw SAM
# Select user, choose clear or change password
# Reboot and log in

Microsoft account: Reset at account.live.com. BitLocker requires recovery key first.

Linux Password Reset

# Single-user mode via GRUB:
# Press e to edit kernel, add init=/bin/bash
mount -o remount,rw /
passwd username
reboot -f

# From Live CD:
# Mount root, chroot, passwd, exit

LUKS: Need original passphrase for encrypted home.

macOS Password Reset

# Apple ID: wrong password 3x - reset via Apple ID
# Recovery: Hold power/Cmd+R - Terminal - resetpassword
# Single-user: Cmd+S - mount -uw / - passwd username

FileVault: Need 64-char recovery key first.

Bitwarden Emergency Access

Bitwarden Emergency Access allows trusted contacts to request vault access.

# Settings - Emergency Access - Add Contact
# Set waiting period (7 days recommended)
# Contact requests access, waiting period starts
# No response after period = access granted

Offline: Write master password on paper, seal in envelope, store in safe.

2FA Recovery

  • Recovery codes - Find in password manager or safe
  • Backup methods - SMS backup, YubiKey, backup authenticator
  • Contact support - Proof of identity, 3-14 days

Storage: Password manager secure note + printed in safe + encrypted USB.
