Minecraft Server Guides

The complete guide to hosting and joining crossplay, any-version, and cracked Minecraft servers — all for free using Aternos.

Hosting

Hosting a Server

Requirements

The only requirements are a Minecraft launcher and an Aternos account.

For cracked Minecraft: SKLauncher

For purchased Minecraft: Prism Launcher, Minecraft Launcher, or any client (Lunar, Badlion, etc.)

If you already have an Aternos account, log in at aternos.org/go/. Otherwise, follow the steps below.

Creating an Aternos Account

  1. Go to aternos.org/signup
  2. Choose a username, tick the checkboxes to accept the terms, and click "Next"
  3. Choose a password (optionally add an email for recovery)
  4. Alternatively, log in with Google

Creating the Server

  1. Go to aternos.org/servers
  2. Press "Create a server"
  3. Set a name and MOTD (optional)
  4. Do not change the platform. Keep it on "Java Edition"
  5. Press "Create"
  6. If prompted about backups, follow the steps or press "Skip"
  7. You should now be on aternos.org/server/
Versions

Making It Playable on Any Version

These plugins let players join your server from any Minecraft version, old or new.

  1. On the left side panel, press "Software" (aternos.org/software/)
  2. Choose "Paper / Bukkit" (aternos.org/software/papermc/)
  3. Select the latest version (or the version your world is on) and press "Install"
  4. Once installed, install ViaVersion — Install link. Press "Install" on the latest version and wait for it to finish.
  5. Install ViaRewind — Install link
  6. Install ViaBackwards — Install link

After installing all three, your server will accept connections from any Minecraft version.

Crossplay

Making It Crossplay

Crossplay lets Bedrock Edition players (mobile, console, Windows 10) join your Java server.

  1. Install GeyserMC — Install link
  2. The next time you start the server, it will automatically support Bedrock players alongside Java players
Cracked

Making It Cracked

This setting allows players without a premium Minecraft account to join.

  1. On the left side panel, press "Options" (aternos.org/options/)
  2. Scroll down until you find the "Cracked" setting
  3. Enable it
  4. The server now accepts cracked (non-premium) clients
You may change any other settings you want, as long as you don't tamper with the plugins and tweaks we've applied above.
Join

Joining the Server

To join the server, it must be running. Go to aternos.org/server/ and press "Start".

Get Server Info

Before joining, you'll need the server's address and port:

  1. Press "Server" on the left side panel
  2. Press "Connect" under the server IP
  3. Note the Server IP and Server Port somewhere

Java Edition

In Java Edition, the address is a single string combining IP and port with a colon:

Format: [Server IP]:[Server Port]
Example: example.aternos.me:12345
  1. Open Minecraft and go to Multiplayer
  2. Press "Add Server" in the bottom right
  3. Enter any name you like ("Server Name" textbox)
  4. In the address box ("Server Address"), paste the full address (e.g. example.aternos.me:12345)
  5. Press "Done"
  6. Double-click the server or press the play button to join

Bedrock Edition

In Bedrock Edition, the IP and port are entered in separate fields:

Server Address: example.aternos.me
Server Port: 12345
(Java full address for comparison: example.aternos.me:12345)
  1. In the main menu, press "Play" → "Servers" tab
  2. Press "Add Server"
  3. Enter any name ("Server Name")
  4. Set "Server Address" to your IP and "Server Port" to your port
  5. Press "Save" and join

Don't see the "Add Server" button? See Workarounds below.

Workarounds

No "Add Server" Button? (Realms Workaround)

If your Bedrock Edition doesn't have the "Add Server" button, use this workaround through Realms:

  1. Go to the "Realms" tab
  2. Press "Add/join Realm"
  3. Select "Join an existing Realm"
  4. Enter the following invite link or code:
    Code: gMAHyBCCj9W-fNg
    Link: https://realms.gg/gMAHyBCCj9W-fNg
  5. Press "Join" twice, then "Play"
  6. Find the "Join any server" NPC inside the Realm
  7. Open its menu and add your server
  8. Join through the NPC
More

Other Workarounds

If you don't see the "Add/join Realm" button either, try these methods.

The Friend Method

  1. Go to the "Friends" tab
  2. Press "Add Friend"
  3. Search for "TheServerHub"
  4. Add him as a friend
  5. In the Friends tab, join "TheServerHub" Realm
  6. Follow the NPC steps from the Realms workaround above

Old UI Method

  1. Go to the "Servers" tab
  2. Join "The Hive"
  3. Open chat and type /region oc
  4. Press "Yes, take me to Oceania!"
  5. Wait — it will say "Invalid IP!" and kick you
  6. Exit the error menu — you'll now have the old "Play" menu
  7. Go to the "Realms" tab and add the Realm normally
Make sure you ADD the Realm so that it shows up in the Realms menu permanently. Otherwise you'll have to repeat this process every time.
Also check out OniionCraft and their project TheServerHub: oniioncraft.com/serverhub — they provide public Realms for this exact purpose.
