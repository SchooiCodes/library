Hosting

Hosting a Server

Requirements

  • A Minecraft launcher
  • An Aternos account

For cracked Minecraft: SKLauncher

For purchased Minecraft: Prism Launcher, Minecraft Launcher, or any client (Lunar, Badlion, etc.)

Creating an Aternos Account

  1. Go to aternos.org/signup
  2. Choose a username, accept the terms, and click "Next"
  3. Choose a password (optionally add email for recovery)
  4. Alternatively, log in with Google

Creating the Server

  1. Go to aternos.org/servers
  2. Press "Create a server"
  3. Set name and MOTD (optional)
  4. Keep platform on "Java Edition"
  5. Press "Create"
Versions

Making It Playable on Any Version

  1. In the Aternos side panel, press "Software"
  2. Choose "Paper / Bukkit" with the latest version
  3. Press "Install"
  4. Install ViaVersion: Install link
  5. Install ViaRewind: Install link
  6. Install ViaBackwards: Install link
Crossplay

Making It Crossplay

  1. Install GeyserMC: Install link
  2. Next time you start the server, it will automatically support Bedrock players
Cracked

Making It Cracked

  1. In the side panel, press "Options"
  2. Scroll to "Cracked" setting and enable it
  3. The server now accepts cracked (non-premium) clients
Join

Joining the Server

Get Server Info

  1. Press "Start" on your Aternos server page
  2. Press "Server" in the side panel
  3. Press "Connect" under the server IP
  4. Note the Server IP and Server Port

Java Edition

Full address format: [Server IP]:[Server Port]

Example: example.aternos.me:12345

  1. In Minecraft, go to Multiplayer → "Add Server"
  2. Enter any name and paste the full address
  3. Press "Done" and double-click to join
Bedrock

Bedrock Edition

  1. In the main menu, press "Play" → "Servers" tab
  2. Press "Add Server"
  3. Enter: Server Address = your IP, Server Port = your port
  4. Press "Save" and join

No "Add Server" Button?

Use the Realms workaround:

  1. Go to the "Realms" tab → "Add/join Realm"
  2. Enter Realm code: gMAHyBCCj9W-fNg
  3. Join the Realm, then find the "Join any server" NPC
  4. Add your server through the NPC
Workarounds

Other Workarounds

The Friend Method

  1. Go to the "Friends" tab → "Add Friend"
  2. Search for "TheServerHub"
  3. Join "TheServerHub" Realm
  4. Follow the NPC steps

Old UI Method

  1. Go to "Servers" tab → Join "The Hive"
  2. Type /region oc in chat
  3. Press "Yes, take me to Oceania!"
  4. Wait for "Invalid IP!" kick
  5. Exit error — you'll have the old Play menu
  6. Add the Realm normally
