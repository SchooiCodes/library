⌨️ Windows Keyboard Shortcuts

Essential Windows keyboard shortcuts for power users — desktop, file explorer, task manager, virtual desktops, and hidden shortcuts.

Desktop Shortcuts (Win key)

ShortcutAction
Win + DShow/hide desktop
Win + EOpen File Explorer
Win + IOpen Settings
Win + LLock your PC
Win + ROpen Run dialog
Win + SOpen Search
Win + VOpen clipboard history
Win + . or Win + ;Open emoji picker
Win + TabTask View (open windows + virtual desktops)
Win + XQuick Link menu (power user menu)
Win + Pause/BreakSystem properties

File Explorer

Ctrl + N     Open new window
Ctrl + W     Close current window
Ctrl + Shift + N Create new folder
Alt + D     Focus address bar
Alt + P     Show/hide preview pane
F2        Rename selected item
F5        Refresh
F11       Full screen

Task Manager

Ctrl + Shift + Esc opens Task Manager directly (no intermediate menu). More shortcuts once inside:

ShortcutAction
Alt + EEnd task
Alt + DSwitch to Details tab
Ctrl + TabSwitch between tabs
Tip: Press Ctrl + Shift + Esc then Alt + D then delete key — fastest way to kill a hung process.

Virtual Desktops

ShortcutAction
Win + Ctrl + DCreate new virtual desktop
Win + Ctrl + ←/→Switch between desktops
Win + Ctrl + F4Close current virtual desktop

Use virtual desktops to separate workspaces — one for coding, one for browsing, one for communication.

Screenshot Shortcuts

PrtScn      Copy full screen to clipboard
Alt + PrtScn   Copy active window to clipboard
Win + PrtScn   Save full screen to Screenshots folder
Win + Shift + S Snipping Tool (select area)
Win + G     Game bar (record screen, take screenshot)

Hidden & Niche Shortcuts

ShortcutAction
Win + HomeMinimize all windows except the active one
Win + TCycle through taskbar apps (then Enter to open)
Win + Shift + ←/→Move window to another monitor
Ctrl + Shift + Click taskbar iconOpen app as administrator
Alt + F4Close current window (or shutdown dialog on desktop)
Shift + Right-clickExtended context menu (copy as path, etc.)

Custom Shortcuts

Create your own shortcuts to launch apps:

1. Right-click app shortcut -> Properties
2. Set "Shortcut key" (e.g., Ctrl + Alt + N for Notepad)
3. Apply

PowerToys Keyboard Manager lets you remap any key:
- Download from Microsoft Store or GitHub
- Remap Caps Lock to Ctrl, or create custom chords
