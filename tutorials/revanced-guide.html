ReVanced Guide

Patch Android APKs with ReVanced to remove ads, add features, and customize apps like YouTube, Reddit, and more.

Overview

ReVanced is a continuation of the original Vanced project (which was shut down due to a Google cease-and-desist). It is a set of tools that applies pre-built patches to Android APK files, modifying the app's behavior at the binary level. Unlike Vanced, which distributed pre-modded APKs, ReVanced requires you to patch apps yourself — which keeps the project legally safer and more flexible.

Safety Warning. Only use the official ReVanced tools from https://github.com/ReVanced. Websites like revanced.net and revanced.io are NOT affiliated with the official project and may distribute malware. Always patch APKs yourself or download from trusted community mirrors only.

What is ReVanced?

The ReVanced project consists of several components:

  • ReVanced Manager — Android app that automates patching on your phone.
  • ReVanced CLI — Command-line tool for patching on PC.
  • ReVanced Patches — The actual modifications (ad removal, sponsorblock, etc.).
  • ReVanced Integrations — Support library needed by some patches.
  • microG — A reimplementation of Google Play Services required for YouTube ReVanced to sign in.

ReVanced Manager

The easiest way to patch apps on Android:

  1. Download the latest ReVanced Manager APK from the official GitHub releases: https://github.com/ReVanced/revanced-manager/releases
  2. Install the APK on your device (enable "Install from unknown sources").
  3. Open ReVanced Manager. It will automatically check for updates to itself and the patches.
  4. Tap the Patcher button at the bottom.
  5. Select an app to patch. If you have the app installed, the Manager can extract its APK. Otherwise, select an APK file from storage.
  6. Select the patches you want to apply (see Recommended Patches).
  7. Tap Patch. The process takes 1-5 minutes depending on the app size.
  8. After patching, tap Install to install the modded APK.
Manager Limitations. ReVanced Manager works best on Android 8+. Some patches require root access (listed as "root" patches). The Manager will warn you if a patch needs root.

Patching APKs

Patching modifies the original APK. The process involves:

  1. Decompilation — The APK is unpacked and its DEX bytecode is converted to smali.
  2. Patch Application — Patches modify smali code, resources, and AndroidManifest.xml.
  3. Recompilation — The modified files are repackaged into a new APK.
  4. Signing — The APK is signed with a debug or custom keystore so Android will install it.

You must use the correct version of the original app. Each set of patches is tested against specific app versions. The Manager will suggest the recommended version.

Other Apps

ReVanced patches also exist for TikTok, Twitter/X, Twitch, and many more. Check the patches repository for the full list: https://github.com/ReVanced/revanced-patches.

Obtaining APKs

To patch an app, you need the original APK. Sources:

  • APKMirror (https://www.apkmirror.com) — Most trusted. All APKs are verified against the original signature. Recommended for all apps.
  • APKPure (https://apkpure.net) — Large library, but some APKs may be re-signed. Use with caution.
  • APKCombo (https://apkcombo.com) — Good for older versions.
  • Extract from your phone — Use SAI (Split APK Installer) or APK Extractor to extract APKs from installed apps.

Important: You need the exact version supported by the current patches. ReVanced Manager will tell you the recommended version. For YouTube, the recommended version is usually the latest stable release from Google Play.

microG Setup

microG is required to sign into YouTube ReVanced with your Google account. Without it, you can only watch videos without logging in.

  1. Download the latest Vanced microG APK from: https://github.com/ReVanced/GmsCore/releases
  2. Install the microG APK on your device.
  3. Open microG settings from the app drawer and ensure the "Cloud Messaging" and "Google device registration" options are enabled.
  4. Open YouTube ReVanced. Tap "Sign in" and follow the usual Google sign-in flow.
  5. microG will handle the authentication without Google Play Services.
microG Privacy. microG is open-source and does not send data to Google. However, signing into your Google account naturally gives Google access to your YouTube activity. Create an alternate account if privacy is a concern.

CLI Patching (Advanced)

For users who prefer patching on a PC:

# Download the required JAR files from GitHub releases:
# - revanced-cli.jar
# - revanced-patches.jar
# - revanced-integrations.apk

# Download the original APK (e.g., YouTube).

# Run the patcher:
java -jar revanced-cli.jar patch \
  --patches revanced-patches.jar \
  --integrations revanced-integrations.apk \
  --include microg-support \
  --include sponsorblock \
  --include return-youtube-dislike \
  --out output.apk \
  youtube.apk

# Install the patched APK:
adb install output.apk

CLI patching is faster and gives you more control over which patches are applied. It also allows you to patch apps that the Manager does not support.

Troubleshooting

App Crashes on Launch

  • You used the wrong base APK version. Check the recommended version and re-patch.
  • You applied conflicting patches. Try with only microg-support and sponsorblock first.
  • Clear the app data and cache after installing the patched APK.

"App not installed" Error

  • You already have the official app installed. Uninstall it first, then install the patched version.
  • Your Android version is too old (needs 8.0+).
  • The APK was not signed properly. Use the Manager or CLI with default signing.

Can't Sign In (YouTube)

  • microG is not installed or is outdated.
  • You did not include the microg-support patch.
  • Clear microG data and try again.
  • Make sure Google Play Services are not disabled on your device.

SponsorBlock Not Working

  • Ensure the sponsorblock patch was included.
  • Check SponsorBlock settings in the YouTube settings menu.
  • Some segments may not be in the database. Submit missing segments via the SponsorBlock extension on desktop.

Patches Outdated

ReVanced releases new patches every 1-2 weeks. If YouTube updates, patches may break. Update your patches and re-patch.

Tips & Best Practices

  • Use the recommended app version. Patching the wrong version will almost certainly fail or crash.
  • Keep patches updated. Follow the ReVanced GitHub releases or use the Manager's built-in updater.
  • Export your patched APK. After patching, save the APK to your computer or cloud storage so you do not have to re-patch if you reset your phone.
  • Create a separate Google account. Use an alt account for modded YouTube to avoid any (however unlikely) account actions.
  • Do not use revanced.net. That site is unaffiliated and has been known to bundle spyware.
  • Patch on Wi-Fi. Downloading APKs and patches uses data. Do it on Wi-Fi.
  • Root is optional. ReVanced works perfectly without root. Root-only patches are clearly labeled.
  • Join the community. The ReVanced subreddit (/r/revancedapp) and Discord server are the best sources for support and patch recommendations.
