Windows Terminal Overview

Windows Terminal is a modern, open-source terminal application for Windows 10 and 11. It supports multiple tabs, panes, GPU-accelerated rendering, and tabs for PowerShell, CMD, WSL, and SSH connections. It replaced the legacy console host as the default terminal in Windows 11.

Install from Microsoft Store, winget, or GitHub:

# winget
winget install Microsoft.WindowsTerminal

# Chocolatey
choco install microsoft-windows-terminal

# Scoop
scoop bucket add extras
scoop install windows-terminal

Profiles & Settings

Windows Terminal settings are stored in settings.json (JSONC format). Access via Settings UI or directly:

// ~/AppData/Local/Packages/Microsoft.WindowsTerminal_8wekyb3d8bbwe/LocalState/settings.json
{
    "$schema": "https://aka.ms/terminal-profiles-schema",
    "profiles": {
        "defaults": {
            "fontFace": "Cascadia Code NF",
            "fontSize": 12,
            "opacity": 90,
            "useAcrylic": true,
            "cursorShape": "filledBox"
        },
        "list": [
            {
                "name": "PowerShell",
                "commandline": "pwsh.exe -NoLogo",
                "icon": "ms-appx:///ProfileIcons/{61c54bbd-c2c6-5271-96e7-009a87ff44bf}.png"
            },
            {
                "name": "Ubuntu WSL",
                "source": "Windows.Terminal.Wsl",
                "distribution": "Ubuntu"
            }
        ]
    }
}

Oh My Posh Prompt Customization

Oh My Posh provides beautiful prompt themes for PowerShell, CMD, and WSL:

# Install Oh My Posh (winget)
winget install JanDeDobbeleer.OhMyPosh

# Install a Nerd Font (Cascadia Code)
oh-my-posh font install

# Edit PowerShell profile
notepad $PROFILE
# Add:
oh-my-posh init pwsh --config ~/jandedobbeleer.omp.json | Invoke-Expression

# Install themes
oh-my-posh init pwsh --config "$(oh-my-posh config path)/themes/cloud-native.omp.json"

# Browse themes
Get-PoshThemes

Keybindings & Productivity

Boost productivity with custom keybindings and actions:

// In settings.json "actions" array:
{
    "command": { "action": "splitPane", "split": "vertical" },
    "keys": "alt+shift+d"
},
{
    "command": { "action": "splitPane", "split": "horizontal" },
    "keys": "alt+shift+- "
},
{
    "command": "closePane",
    "keys": "ctrl+shift+w"
},
{
    "command": { "action": "newTab", "index": 0 },
    "keys": "ctrl+1"
},
{
    "command": { "action": "sendInput", "input": "clear\u000d" },
    "keys": "ctrl+l"
}

Use quake mode (win+`) for a drop-down terminal that appears on any keypress.

Color Schemes & Theming

Windows Terminal supports custom color schemes. Popular ones include Catppuccin, Dracula, and Nord:

// In settings.json
"schemes": [
    {
        "name": "Catppuccin Mocha",
        "background": "#1e1e2e",
        "foreground": "#cdd6f4",
        "black": "#45475a",
        "red": "#f38ba8",
        "green": "#a6e3a1",
        "yellow": "#f9e2af",
        "blue": "#89b4fa",
        "purple": "#cba6f7",
        "cyan": "#94e2d5",
        "white": "#bac2de"
    }
],

// Apply to profile
"profiles": {
    "defaults": { "colorScheme": "Catppuccin Mocha" }
}
