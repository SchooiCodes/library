🖥️ System Utilities

These tools will make your Windows experience smoother, faster, and more private. Every power user should have these installed.

Everything

Instant file search for Windows. Indexes your entire drive and finds files as you type — faster than Windows Search by a mile. Searches by name, date, size, and more in milliseconds.

Download

7-Zip

Free archive manager supporting zip, rar, 7z, tar, gzip, and more. Better compression ratios than most paid alternatives. Integrates into the right-click context menu.

Download

O&O ShutUp10++

Privacy tool for Windows 10/11. Disable telemetry, Cortana, OneDrive, and dozens of other data-collection features with one click. Portable — no installation needed.

Download

Autoruns

See every program that starts automatically with Windows. Disable unwanted startup entries, services, scheduled tasks, drivers, and codecs from one interface.

Download

Process Explorer

Advanced task manager from Sysinternals. See which file handles are open, what DLLs are loaded, and dig deep into process details. Replace Task Manager with this.

Download

HWiNFO

Comprehensive hardware monitoring. Track CPU temperature, fan speeds, voltages, and more. Essential for diagnosing system issues and monitoring sensor data in real-time.

Download

Geek Uninstaller

Portable app uninstaller that thoroughly removes programs and cleans leftover files, folders, and registry entries. Forces removal of stubborn apps that won't uninstall normally.

Download

TinyTask

Extremely lightweight macro recorder. Records mouse clicks and keyboard actions, then plays them back on loop. The entire program is only ~36KB — no bloat, no scripts.

Download

GS Auto Clicker

Feature-rich auto clicker with configurable click intervals, hotkeys, and multiple click types (single, double, right-click). Great for gaming and repetitive tasks.

Download

OP AutoClicker

Simple, open-source auto clicker with a clean interface. Set hotkeys to start/stop, choose between single/double/right clicks, and adjust speed by the millisecond.

Pro Tip — Install Everything with Scoop or Winget:

Almost every app on this page can be installed from the command line, keeping them updated automatically and avoiding manual downloads. Here's how:

Scoop:

scoop bucket add extras
scoop install vlc firefox brave qbittorrent git nodejs python ffmpeg yt-dlp
scoop install notepadplusplus irfanview obs-studio gimp krita audacity
scoop install bitwarden veracrypt wireshark autohotkey everything

Winget (built into Windows 10/11):

winget install VideoLAN.VLC Mozilla.Firefox Git.Git OpenJS.NodeJS
winget install 7zip.7zip Notepad++.Notepad++ OBSProject.OBSStudio
winget install Irfanview.Irfanview GIMP.GIMP Krita.Krita
winget install Bitwarden.Bitwarden VeraCrypt.VeraCrypt Malwarebytes.Malwarebytes
winget install Microsoft.PowerToys Microsoft.WindowsTerminal

Mix and match — use whichever works best for each app. Winget uses the Microsoft repository while Scoop installs to your user directory without admin rights. Both handle updates automatically.
