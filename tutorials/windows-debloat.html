🧹 Windows Debloating & Optimization

A complete guide to removing bloatware, disabling telemetry, and optimizing Windows 10/11 for performance and privacy.

🤔 Why Debloat Windows?

Out of the box, Windows 10 and 11 come loaded with pre-installed apps, background services, and telemetry features that most people don't need. This bloatware consumes system resources, slows down your PC, and sends data to Microsoft and third parties.

Debloating removes unnecessary apps, disables background processes, and cuts telemetry to the minimum. The results:

  • Better performance — fewer background processes means more CPU and RAM for your actual work
  • Improved privacy — stop telemetry, Cortana listening, and data collection
  • More disk space — bloatware apps can take up several GB on your system drive
  • Faster boot times — fewer startup programs means quicker login
  • Longer battery life — less background activity on laptops

This guide covers multiple methods from beginner-friendly GUI tools to advanced PowerShell commands. Pick your level.

🛡️ Using O&O ShutUp10++

O&O ShutUp10++ is the safest and easiest way to disable Windows privacy violations. It's a portable app — no installation needed.

Getting Started

  1. Download from oo-software.com
  2. Run the executable (no install required)
  3. Click "Apply all recommended settings" in the toolbar
  4. Review the changes — each setting has a clear description
  5. Click "Apply selected actions"
  6. Restart your PC when prompted

Key Tweaks You Should Enable

  • Telemetry — Set to 0 (Security). Disables data collection and diagnostic data sending.
  • Cortana — Completely disable Cortana and web search in Windows Search.
  • OneDrive — Disable OneDrive integration. Stop it from syncing and running at startup.
  • Windows Defender notifications — Reduce or disable non-critical security notifications.
  • App notifications — Disable tips, suggestions, and "Get even more out of Windows" prompts.
  • Location tracking — Disable location services and Wi-Fi sensing.
  • Advertising ID — Reset and disable the advertising ID used for personalized ads.
  • Microsoft Office telemetry — Disable if you have Office installed.
Note: ShutUp10++ creates a restore point before applying changes. If something breaks, run the tool again and click "Restore all default settings".

⚡ Chris Titus Windows Utility

The Chris Titus Windows Utility is a powerful open-source script that debloats, tweaks, and hardens Windows in one go. It's frequently updated and community-tested.

Installation

Open PowerShell as Administrator and run:

irm https://christitus.com/win | iex

The "Tweaks" Tab

This is where the real work happens. The Tweaks tab is organized into several groups:

  • Essential Tweaks — Enable dark mode, show file extensions, disable mouse acceleration, enable NumLock at startup, disable Sticky Keys prompts, and more.
  • Privacy Tweaks — Disable telemetry, Cortana, Bing Search, activity history, location tracking, feedback frequency, advertising ID, and automatic sample submission.
  • Remove Bloatware — A curated list of Microsoft and third-party bloatware that can be removed with a single click. This includes apps like:
    • Xbox (all Xbox-related apps and services)
    • Skype (pre-installed version)
    • OneDrive (if you don't use it)
    • Microsoft Teams (consumer version)
    • Cortana
    • News, Weather, and other MSN apps
    • Mail and Calendar
    • Microsoft People, Xbox TCUI, Xbox Identity Provider
    • Bing Weather, Bing News, Bing Sports, Bing Finance
    • 3D Viewer, Paint 3D, Mixed Reality Portal
    • Skype, Solitaire Collection, Office Hub

Just check what you want to remove and click "Run Tweaks". The utility will also offer to apply recommended registry tweaks for better performance.

🔧 Manual Bloatware Removal

If you prefer to do things manually, Windows provides several ways to remove bloatware yourself.

Using Settings

  1. Open Settings (Win + I)
  2. Go to Apps → Installed apps
  3. Browse the list and click the three dots next to any app you want to remove
  4. Click Uninstall

Using PowerShell (Advanced)

Open PowerShell as Administrator and use the Get-AppxPackage and Remove-AppxPackage cmdlets.

List all installed Windows Store apps:

Get-AppxPackage | Select Name, PackageFullName

Remove a specific app (replace xbox with any package name):

Get-AppxPackage *xbox* | Remove-AppxPackage

⚙️ Service Tweaks

Disabling unnecessary Windows services frees up RAM and reduces background CPU usage. Open services.msc (Win + R, type services.msc) to manage services manually.

Services You Can Safely Disable

Service Name Why Disable
SysMain (Superfetch)Preloads apps into RAM. On SSDs it provides no benefit and uses memory.
Windows SearchIndexes files for search. Disable if you use Everything or don't search often.
Windows Error ReportingSends crash reports to Microsoft. Disable for privacy.
Diagnostic Service HostCollects diagnostic data. Part of telemetry — safe to disable.
Diagnostic System HostSame as above.
Connected User Experiences and TelemetryMain telemetry service. Disable via ShutUp10++ for best results.
Xbox Live Auth ManagerNot needed if you don't use Xbox features.
Xbox Live Game SaveSame as above.
Xbox Live NetworkingSame as above.
Delivery OptimizationP2P update sharing. Disable unless you have limited bandwidth.
Print SpoolerDisable if you don't use a printer.
Bluetooth Support ServiceDisable if you don't use Bluetooth.
Touch Keyboard and HandwritingDisable if you don't use a touchscreen.

How to Disable a Service

  1. Press Win + R, type services.msc, press Enter
  2. Find the service in the list
  3. Right-click → Properties
  4. Set "Startup type" to Disabled
  5. Click "Stop" to stop it immediately
  6. Click OK

🚀 Startup Management

Many programs add themselves to startup, slowing down boot time and consuming resources in the background.

Using Task Manager

  1. Press Ctrl + Shift + Esc to open Task Manager
  2. Click the Startup tab (on Windows 11, click the hamburger menu → Startup apps)
  3. Review each app and its "Startup impact"
  4. Right-click and Disable anything you don't need at boot

Common Apps to Disable at Startup

  • OneDrive — Sync runs at boot; disable if you use it on-demand
  • Spotify — Doesn't need to be running until you open it
  • Discord — Same, launch it when you need it
  • Steam — Game client autostart; disable unless you game daily
  • Epic Games Launcher — No reason to run at boot
  • Adobe Creative Cloud — Background updater; disable and update manually
  • Microsoft Teams — Auto-starts and sits in system tray
  • Skype — Another unnecessary autostart
  • Any hardware software (printer, scanner, etc.) — rarely needed at boot

For more advanced startup management, use Autoruns from Sysinternals. It shows every autostart entry including services, scheduled tasks, and drivers.

🎨 Visual Tweaks

Windows eye candy consumes GPU resources and RAM. Disabling animations and transparency can make older systems feel snappier.

Adjust for Best Performance

  1. Right-click This PC → Properties
  2. Click Advanced system settings
  3. Under Performance, click Settings
  4. Select "Adjust for best performance" — this disables all animations, shadows, and visual effects
  5. Alternatively, uncheck specific effects you want to disable:
    • ✓ Animate controls and elements inside windows
    • ✓ Animate windows when minimizing and maximizing
    • ✓ Fade or slide menus into view
    • ✓ Show shadows under windows
    • ✓ Show shadows under mouse pointer
    • ✓ Smooth-scroll list boxes
    • ✓ Taskbar animations

Disable Transparency

  1. Open Settings → Personalization → Colors
  2. Turn Transparency effects → Off

💾 Disk Cleanup & Storage Sense

Regular disk cleanup keeps your system drive from filling up with temporary files, update caches, and old Windows installations.

Storage Sense

  1. Open Settings → System → Storage
  2. Toggle Storage Sense → On
  3. Click "Configure Storage Sense or run it now" to customize:
    • Run Storage Sense: Every week
    • Delete temporary files not in use: Check
    • Delete files in Recycle Bin after: 30 days
    • Delete files in Downloads folder after: 30 or 60 days (or never if you use it)
  4. Click "Run Storage Sense now" for an immediate cleanup

Classic Disk Cleanup

Press Win + R, type cleanmgr, press Enter. Select your system drive (usually C:) and check these categories:

  • ✓ Windows Update Cleanup — can free 2-10+ GB
  • ✓ Delivery Optimization Files
  • ✓ Recycle Bin
  • ✓ Temporary Internet Files
  • ✓ Temporary files
  • ✓ Thumbnails

For deeper cleanup, click "Clean up system files" (requires admin). This also shows Previous Windows installations which can free 10-20 GB if you don't need to roll back.

System File Checker

Open PowerShell or CMD as Administrator and run these commands to repair system files:

dism /online /cleanup-image /restorehealth
sfc /scannow

Run DISM first to repair the system image, then SFC to scan and repair Windows files. This is good practice after major debloating.

🔋 Power Settings

Windows power plans control how your CPU, GPU, and system behave. The High Performance plan prevents the system from throttling down for power savings.

High Performance Plan

  1. Open Control Panel → Power Options
  2. Select High Performance
  3. If you don't see it, click "Show additional plans"

Ultimate Performance Plan

Available on Windows 10/11 Pro and Workstation editions. This plan removes micro-latency tuning and is designed for demanding workloads.

Enable it via command line (admin):

powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

After running, go to Power Options and select Ultimate Performance.

Additional Power Tweaks

  • Click "Change plan settings" → "Change advanced power settings"
  • Set Hard disk → Turn off hard disk after → 0 (never, for desktops)
  • Set Processor power management → Minimum processor state → 100% (for performance)
  • Set Processor power management → Maximum processor state → 100%
  • Disable USB selective suspend if you have USB device issues
Note: High Performance power plans consume more electricity. On laptops, use "Balanced" when on battery and "High Performance" when plugged in.
