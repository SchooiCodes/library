GIMP Image Editing

Professional photo retouching, compositing, and graphic design with the free open-source GIMP.

GIMP Interface & Basics

GNU Image Manipulation Program (GIMP) is a free, open-source alternative to Photoshop. It supports layers, masks, channels, paths, and a full suite of editing tools.

Core panels: toolbox (left), tool options (under toolbox), layers (right), brushes/patterns/gradients (right). Dockable dialogs can be rearranged freely.

Basic workflow: open image → duplicate layer → edit non-destructively → export. Always work on duplicate layers to preserve the original.

Layers & Masks

Layers stack like sheets of glass. Each layer can be edited independently. Layer masks hide/reveal portions of a layer without deleting pixels.

Add a layer mask: right-click layer → Add Layer Mask. Paint with black to hide, white to reveal, gray for partial transparency. Layer masks are essential for non-destructive compositing.

Blend modes control how layers interact: multiply (darken), screen (lighten), overlay (contrast), soft light (gentle contrast), and color (tint with hue/saturation).

Group layers with folder icons. Apply layer masks to groups for complex compositions.

Color Correction & Retouching

Levels adjust shadows, midtones, and highlights. Curves offer precise tonal control. Hue/Saturation adjusts color intensity. Color Balance shifts shadows/midtones/highlights.

Clone tool removes blemishes by sampling from a source area. Heal tool blends the sampled area with the target texture. Perspective clone matches perspective distortion.

Unsharp Mask sharpens images. Selective Gaussian Blur smooths without destroying edges. Use layer masks to apply effects only where needed.

Paths & Selections

The Paths tool creates bezier curves for precise selections. Convert paths to selections with Path → Selection. Selection tools: rectangle, ellipse, free select (lasso), fuzzy select (magic wand), and select by color.

Quick Mask mode (Shift+Q) paints selections directly on the image. Save selections to channels for later reuse. Feather selections for smooth transitions.

Foreground Select tool isolates foreground subjects using a rough scribble. This is GIMP's answer to Photoshop's Select Subject.

Plugins & Scripting

GIMP supports Python and Script-Fu (Scheme) scripting. Extend GIMP with plugins like G'MIC for artistic filters, Resynthesizer for content-aware fill, and BIMP for batch processing.

#!/usr/bin/env python
from gimpfu import *
def batch_resize(image, drawable, width):
    pdb.gimp_image_scale(image, width, width * image.height / image.width)
    new_name = pdb.gimp_image_get_filename(image) + '_resized.jpg'
    pdb.file_jpeg_save(image, drawable, new_name, new_name, 0.85, 0, 0, 0, '', 0, 1, 0, 0)

register(
    'batch_resize', 'Batch Resize Images', 'Resize all images to a width',
    'Author', 'Author', '2026', 'Batch Resize...', '*', [], [], [], batch_resize)
main()
