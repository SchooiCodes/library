🖼️ ImageMagick Guide

Image manipulation from the CLI — convert, resize, crop, color correct, batch process, and create graphics with ImageMagick's convert and magick commands.

Table of Contents

  • Installation

Installation

# Linux
sudo apt install imagemagick

# macOS
brew install imagemagick

# Verify
convert --version
magick --version

Convert & Resize

# Convert format
convert input.png output.jpg

# Resize to specific dimensions
convert input.jpg -resize 800x600 output.jpg

# Resize proportionally (width)
convert input.jpg -resize 800 output.jpg

# Resize to fit within bounds (no distortion)
convert input.jpg -resize 800x600\> output.jpg

# Convert all PNGs in folder to JPG
mogrify -format jpg *.png
The \> suffix after dimensions means "only shrink if larger" — \< means "only enlarge if smaller". This prevents accidental upscaling.

Crop & Rotate

# Crop (widthxheight+x+y)
convert input.jpg -crop 400x300+50+50 output.jpg

# Crop center gravity
convert input.jpg -gravity center -crop 500x500+0+0 output.jpg

# Rotate
convert input.jpg -rotate 90 output.jpg

# Auto-orient from EXIF
convert input.jpg -auto-orient output.jpg

# Flip / Flop
convert input.jpg -flip output.jpg   # vertical
convert input.jpg -flop output.jpg   # horizontal

Effects & Filters

convert input.jpg -colorspace Gray output.jpg                # Grayscale
convert input.jpg -sepia-tone 80% output.jpg                 # Sepia
convert input.jpg -blur 0x8 output.jpg                       # Blur
convert input.jpg -sharpen 0x3 output.jpg                    # Sharpen
convert input.jpg -brightness-contrast 20x10 output.jpg      # Brightness/Contrast
convert input.jpg -bordercolor black -border 10x10 output.jpg

Batch Processing

# Resize all JPGs in folder
for img in *.jpg; do
  convert "$img" -resize 1024 -quality 85 "resized-$img"
done

# Watermark all images
for img in *.jpg; do
  convert "$img" -font Helvetica -pointsize 36 \
    -fill white -gravity southeast -annotate +10+10 \
    "© 2026" "wm-$img"
done
Use mogrify for in-place batch operations — but always back up originals first since mogrify overwrites by default.

Format Conversion

# JPEG quality control (1-100)
convert input.png -quality 90 output.jpg

# Create PDF from images
convert page1.jpg page2.jpg page3.jpg output.pdf

# Extract frames from GIF
convert animation.gif frame%03d.png

# Create layered PSD
convert layer1.png layer2.png -composite output.psd

# Reduce PNG size (lossless)
convert input.png -strip -colors 256 output.png

Creating Graphics

convert -size 800x600 xc:"#3498db" blue.png                           # Solid colour
convert -size 800x200 xc:none -font Helvetica-Bold -pointsize 48 \
  -fill black -gravity center -annotate 0 "Hello, World!" text.png   # Add text
convert -size 400x400 gradient:blue-cyan grad.png                     # Gradient
convert background.jpg logo.png -gravity center \
  -compose over -composite final.jpg                                  # Composite
