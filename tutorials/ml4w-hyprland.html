ML4W Hyprland Configuration

ML4W (My Linux For Work) — the popular Hyprland dotfiles framework for Arch Linux.

What is ML4W?

ML4W (My Linux For Work) by Stephan Raabe is a complete Hyprland dotfiles repository. It provides a pre-configured tiling window manager setup with a graphical installer, app launcher, and modular configuration system.

ML4W includes: Hyprland config, Waybar status bar, Rofi launcher, dunst notifications, Thunar file manager, and a custom settings app.

It supports both Arch (base) and Manjaro via ml4w-hyprland package in the AUR.

Installation

Install ML4W on Arch Linux:

# Via AUR (yay/paru)
yay -S ml4w-hyprland
# Or via the official installer script
bash <(curl -s https://raw.githubusercontent.com/mylinuxforwork/dotfiles/main/setup.sh)

# After installation
ml4w-hyprland-setup  # Launch the graphical installer
# Then select: ML4W Default or ML4W Modern

The installer asks about browser, file manager, terminal, and wallpaper preferences. All configs are stored under ~/.config/ml4w/.

Modular Configuration System

ML4W uses a modular approach. Main config directory: ~/.config/ml4w/

~/.config/ml4w/
├── config/
│   ├── hypr/
│   │   ├── keybindings.conf
│   │   ├── windowrules.conf
│   │   ├── monitors.conf
│   │   └── animations.conf
│   ├── waybar/
│   ├── rofi/
│   ├── dunst/
│   └── kitty/
├── scripts/
│   ├── launcher.sh
│   ├── wallpaper.sh
│   └── screenshot.sh
└── wallpapers/

Each component has its own config directory. The ml4w-settings app provides a GUI for toggling options without editing files directly.

Keybindings & Workflow

ML4W default keybindings (Super = Windows/CMD key):

Super + Return     → Terminal (kitty)
Super + Space      → App Launcher (rofi)
Super + Q          → Close window
Super + [1-9]      → Switch workspace
Super + Shift+[1-9]→ Move window to workspace
Super + W          → Toggle fullscreen
Super + V          → Toggle floating
Super + L          → Lock screen
Super + Shift+E    → Power menu
Super + S          → Screenshot menu

ML4W also includes touchpad gestures (2-finger swipe to switch workspaces) and mouse bindings.

Customizing ML4W Themes

ML4W supports theme switching. Built-in themes: Dark, Light, Nord, Catppuccin, Tokyo Night, Dracula, Gruvbox, Everforest.

# List available themes
ls ~/.config/ml4w/themes/

# Apply a theme
ml4w-hyprland-setup --theme catppuccin-mocha

# Create custom theme
cp -r ~/.config/ml4w/themes/default ~/.config/ml4w/themes/my-theme
# Edit the colors, wallpapers, and config files in the new directory

The ml4w-settings GUI theme section lets you preview and apply themes with one click.
