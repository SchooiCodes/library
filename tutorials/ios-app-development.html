iOS App Development

Build iOS apps with Swift, SwiftUI, and Xcode for iPhone and iPad.

Table of Contents (5 sections)

Swift Fundamentals

Swift is Apple's modern language for iOS/macOS/watchOS/tvOS development. Key features: optionals (?, !), value types (structs, enums), protocols (with associated types, protocol-oriented programming), closures (lambdas), async/await (structured concurrency), and Result type.

Data types: Array, Dictionary, Set, String. Control flow: for-in, guard-let, switch (exhaustive, pattern matching). Access control: private, fileprivate, internal, public, open. Swift Package Manager (SPM) for dependencies.

SwiftUI Layout & Navigation

SwiftUI provides declarative views in Swift. Layout: VStack, HStack, ZStack (alignment, spacing). Controls: Text, Button, Image, TextField, Toggle, Slider, Picker. Modifiers (padding, font, foregroundColor, frame) chain to style views.

NavigationStack manages hierarchical navigation. TabView for bottom/ tab bars. Sheets and fullScreenCover for modal presentations. @State (local), @Binding (passed), @StateObject/@ObservedObject (reference type model), @EnvironmentObject (dependency injection) manage data flow.

Core Data & Networking

Core Data is Apple's persistence framework (object graph + SQLite). NSPersistentContainer sets up the stack. @FetchRequest in SwiftUI auto-refreshes on changes. CloudKit syncs Core Data across devices via iCloud.

Networking: URLSession (built-in, async/await), Alamofire (popular third-party). Codable protocol (Encodable/Decodable) serializes to/from JSON automatically. async let for parallel requests. Monitor network with NWPathMonitor for connectivity changes.

App Architecture (MVVM)

MVVM (Model-View-ViewModel) is the recommended iOS architecture. Model: data structures + business logic. View: SwiftUI views (pure UI). ViewModel: published properties that represent UI state, methods to handle user actions, and coordinates data flow.

Clean Architecture layers: Presentation (ViewModels, Views), Domain (Use Cases, Entities interfaces), Data (Repository implementations, API/DB data sources). Dependency injection via constructor injection. Combine framework for reactive pipelines in UIKit; SwiftUI + async/await simplifies this considerably.

App Store Connect & Publishing

Export your app: set version/build number, configure signing (Apple Developer Program, $99/year), archive and upload via Xcode. App Store Connect: app information (name, subtitle, keywords), screenshots for all device sizes (6.7", 6.5", 5.5", iPad), and promotional text.

Review process: typically 24-48 hours. Common rejections: incomplete information, crashed on review, placeholder content, UI violations (iPad apps must work in all orientations). TestFlight (external/internal testing) before production release. Phased release rolls out over 7 days.
