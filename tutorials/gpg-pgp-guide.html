GPG & PGP Key Management

Encrypt, sign, and verify with GPG. Manage keys, smartcards, and secure communication.

What is GPG?

GNU Privacy Guard (GPG) implements OpenPGP standard for encryption and signing. Used for secure email (with Thunderbird/Enigmail), file encryption, and code signing (git tag -s).

Key Generation

gpg --full-generate-key
# Select: RSA and RSA, 4096 bits
# No expiry recommended for primary key
# Set UID: Name, email address

Key Management

gpg --list-keys               # Public keys
gpg --list-secret-keys        # Private keys
gpg --export -a "Alice" > alice.pub
gpg --import alice.pub
gpg --keyserver keyserver.ubuntu.com --search-keys alice@example.com

Encryption & Decryption

# Encrypt file (to multiple recipients)
gpg -e -r alice@example.com -r bob@example.com secret.txt

# Decrypt
gpg -d secret.txt.gpg > secret.txt

Signing & Verification

# Sign a file (detached signature)
gpg --detach-sign -a file.txt

# Verify
gpg --verify file.txt.asc file.txt

# Sign git commit
git commit -S -m "Signed commit"

Smartcards & YubiKey

Store GPG private keys on YubiKey for hardware security: gpg --edit-key KEYID then "keytocard". Keys cannot be extracted from the device.
