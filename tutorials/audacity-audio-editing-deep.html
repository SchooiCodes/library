Audacity Audio Editing

Professional audio editing, restoration, mixing, and podcast production with Audacity.

Audacity Fundamentals

Audacity is a free, open-source audio editor for recording and editing sounds. Available on Windows, macOS, and Linux. Use it for podcasts, music, voice-overs, and audio restoration.

Interface: Transport toolbar (play/record/stop), Tools toolbar (selection, envelope, draw, zoom, time shift), Track control panel, and waveform display.

Keyboard shortcuts: Space (play/stop), Shift+A (record), Ctrl+C (copy), Ctrl+V (paste), Ctrl+Z (undo), and Delete key removes selected audio.

Recording & Multi-track

Set recording and playback devices in Edit → Preferences → Devices. Use WASAPI on Windows for low-latency recording. Choose mono or stereo tracks.

Multi-track recording: each microphone or instrument records to its own track. Use a USB audio interface (Focusrite, Behringer) for multiple inputs. Overdub records a new track while listening to existing ones.

Punch-in recording replaces a section of an existing track. Use Ctrl+Click to set recording start point. Timer Record schedules recordings for later.

Noise Reduction & Restoration

Noise Reduction: select a few seconds of pure noise → Effect → Noise Reduction → Get Noise Profile → select entire track → Noise Reduction → OK.

Click Removal removes mouse clicks and pops from recordings. Clip Fix repairs clipped/distorted audio. Notch Filter removes specific frequencies (50/60Hz hum).

Equalization shapes tone: graphic EQ for visual adjustment, filter curve EQ for precision. Bass and Treble for simple tone control. Compressor reduces dynamic range — essential for voice clarity.

Effects & Processing

Amplify normalizes volume. Normalize sets peak level (usually -1dB to avoid clipping). Fade In/Fade Out for smooth transitions. Crossfade Tracks blends overlapping clips.

Change Tempo (BPM) without affecting pitch. Change Pitch (semitone) without affecting tempo. Change Speed affects both. Paulstretch creates extreme time-stretching for ambient soundscapes.

Vocal Reduction and Isolation removes or extracts vocals from stereo tracks (center channel removal). Use for karaoke or acapella extraction.

Exporting & Formats

File → Export Audio (Shift+Ctrl+E) exports in various formats: WAV (lossless, large), AIFF (Apple lossless), MP3 (compressed, customizable bitrate), FLAC (lossless compressed), OGG Vorbis (open compressed), and M4A (AAC).

Export Multiple (Shift+Ctrl+L) splits a multi-track project into individual files based on labels or tracks. Tags: add metadata like title, artist, album, genre, and year.

Sample rate: 44100 Hz (CD quality), 48000 Hz (video standard), 96000 Hz (high-resolution). Bit depth: 16-bit (CD), 24-bit (studio), 32-bit float (processing headroom).
