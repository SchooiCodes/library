What is C++?

C++ is a powerful systems programming language created by Bjarne Stroustrup as an extension of C. Used in game engines, operating systems, browsers, and high-frequency trading.

Setup & First Program

// hello.cpp
#include <iostream>
int main() {
    std::cout << "Hello, C++!" << std::endl;
    return 0;
}

g++ -std=c++20 -o hello hello.cpp
./hello

Pointers & Memory

int value = 42;
int* ptr = &value;
cout << *ptr;  // Dereference: 42

// Smart pointers (modern C++)
auto smart = std::make_unique<int>(100);

Classes & OOP

class Animal {
public:
    virtual void speak() const = 0;
};
class Dog : public Animal {
    void speak() const override { cout << "Woof!"; }
};

STL & Modern C++

std::vector<int> nums = {3, 1, 4, 1, 5};
std::sort(nums.begin(), nums.end());

// C++11 range-based for
for (int n : nums) cout << n << " ";
