Calculus Fundamentals

Limits, derivatives, integrals, and optimization — the mathematical foundation of physics and machine learning.

Limits & Continuity

A limit describes the behavior of a function as its input approaches a value. Continuity means the limit equals the function value at that point.

import sympy as sp
x = sp.Symbol('x')
sp.limit(sp.sin(x)/x, x, 0)  # 1

Key limit laws: sum, product, quotient, and squeeze theorem. Understand one-sided vs two-sided limits.

Derivatives & Differentiation

The derivative measures the instantaneous rate of change. Geometrically, it is the slope of the tangent line at a point.

from sympy import diff, sin, cos
diff(x**3, x)          # 3*x**2
diff(sin(x), x)        # cos(x)
diff(cos(x), x)        # -sin(x)

Chain rule, product rule, quotient rule, and implicit differentiation are essential techniques.

Integrals & Integration

Integration computes the area under a curve. The Fundamental Theorem of Calculus links derivatives and integrals.

from sympy import integrate, oo
integrate(x**2, (x, 0, 1))  # 1/3
integrate(sp.exp(-x), (x, 0, oo))  # 1

Techniques: substitution, integration by parts, partial fractions, and numerical integration with scipy.integrate.

Multivariable Calculus

Functions of multiple variables require partial derivatives, gradients, and multiple integrals.

f = lambda x, y: x**2 + y**2
# Gradient: [2x, 2y]
# Directional derivative: ∇f · v
import sympy as sp
y = sp.Symbol('y')
f_sym = x**2 + y**2
print(sp.diff(f_sym, x))  # 2x
print(sp.diff(f_sym, y))  # 2y

Key concepts: gradient descent, Jacobian matrix, Hessian matrix, Laplacian, and curl/divergence.

Optimization & Gradient Descent

Optimization finds the minimum (or maximum) of a function. Gradient descent iteratively moves in the direction of steepest descent.

def gradient_descent(f, grad_f, x0, lr=0.1, n=100):
    x = x0
    for _ in range(n):
        x -= lr * grad_f(x)
    return x

f = lambda x: x**2 + 2*x + 1
grad = lambda x: 2*x + 2
opt = gradient_descent(f, grad, 10.0)
print(opt)  # Close to -1 (minimum)

Extensions: SGD, Adam, momentum, and constrained optimization with Lagrange multipliers.
