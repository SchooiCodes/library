Bash Scripting Mastery

Bash scripting — variables, functions, control flow, error handling, and advanced techniques.

Table of Contents (2 sections)

  • Essentials & Control Flow
  • Functions & Error Handling

Essentials & Control Flow

Shebang: #!/bin/bash. Variables: NAME="value" (no spaces), $NAME or ${NAME}. Arrays: arr=("a" "b" "c"); echo ${arr[1]}; ${#arr[@]} for length. Conditionals: if [[ condition ]]; then ... elif ...; else ...; fi. File tests: -f (file exists), -d (directory), -x (executable). String tests: [[ -z $var ]] (empty), [[ $a == $b ]], [[ $a =~ regexp ]]. Arithmetic: $(( x + y )), (( x++ )). Loops: for i in ${list[@]}; do ...; done; while read line; do ...; done < file.

Functions & Error Handling

Functions: function myfunc() { local var="scope"; return 0; }. Error handling: set -e (exit on error), set -u (undefined var error), set -o pipefail (pipe failure). Trap: trap 'cleanup' EXIT, trap 'echo "Interrupted"; exit 1' SIGINT. Debug: set -x (trace execution), bash -n script (syntax check). Subshell: (cd /tmp && command) — runs in subshell, no env change. Process substitution: diff <(cmd1) <(cmd2). Here documents: cat <<-EOF ... EOF.
