Essential Linux Applications

Curated list of essential Linux applications — productivity, media, development, and utilities.

Table of Contents (5 sections)

  • Productivity & Office
  • Media & Entertainment
  • Development Tools
  • Security & Privacy
  • System Utilities

Productivity & Office

Office suite: LibreOffice (full MS Office-compatible, Calc, Writer, Impress). OnlyOffice (cleaner UI, better docx compatibility). SoftMaker Office (commercial, fastest).

Note-taking: Obsidian (markdown, graph view), Logseq (outliner, open-source), Joplin (sync via Nextcloud/Dropbox), Standard Notes (encrypted).

PDF: Zathura (minimal, vim-keys), Okular (KDE, feature-rich), Evince (GNOME).

Finance: KMyMoney (personal finance), GnuCash (double-entry accounting), Firefly III (self-hosted web app).

Media & Entertainment

Music: Spotify (official client), mpd + ncmpcpp (terminal music player daemon), Strawberry (modern music player), Rhythmbox (GNOME).

Video: VLC (plays everything), mpv (lightweight, scriptable), Celluloid (mpv frontend).

Streaming: Freetube (Youtube without ads), Syncplay (watch videos together).

Media servers: Jellyfin (open-source, movies/TV/music), Plex (commercial, polished).

Development Tools

IDEs: VS Code via Flatpak or AUR, JetBrains Toolbox (PyCharm, IntelliJ, CLion), Neovim (terminal editor), Emacs.

Terminals: Kitty (GPU-accelerated), Alacritty (fast, config file), Ghostty (new, terminal emulator), foot (minimal Wayland).

Git: GitKraken (GUI), lazygit (TUI), GitHub CLI (gh), tig (TUI git browser).

Security & Privacy

VPN: WireGuard (modern, fast, simple), OpenVPN (widely compatible), Mullvad VPN (privacy-first).

Password managers: Bitwarden (open-source, self-hostable), KeePassXC (local, encrypted), Vaultwarden (self-hosted Bitwarden server).

Encryption: VeraCrypt (disk/container encryption), GnuPG (file/email encryption), Cryptomator (cloud encryption).

System Utilities

File managers: Thunar (fast, XFCE), Nautilus (GNOME, feature-rich), Dolphin (KDE), lf/ranger (terminal).

Screenshots: Flameshot (annotations), KSnip (KDE), grim + slurp (Wayland).

Clipboard: CopyQ (persistent history), wl-clipboard (Wayland CLI), clipmenu.

System info: Neofetch/fastfetch (distro info), btop (resource monitor), duf (disk usage), ncdu (interactive disk analysis).
