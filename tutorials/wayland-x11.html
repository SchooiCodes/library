Wayland vs X11

Display servers -- Wayland architecture, X11 comparison, migration.

Table of Contents (2 sections)

  • Wayland
  • X11 & Migration

Wayland

Wayland: modern compositor protocol. Compositors: wlroots (sway, Hyprland), Mutter (GNOME), KWin (KDE). Client renders surface directly. Security: windows cant see each other, keylog protection. XWayland: X11 apps on Wayland. Features: fractional scaling, VRR, gestures, screen capture via PipeWire. NVIDIA: driver 555+ stable, nvidia_drm.modeset=1. libinput: touchpad gestures. Sway: i3-compatible. Hyprland: animations, blur.

X11 & Migration

X11: legacy (1984), network transparency, compositor optional. Issues: tearing, no native fractional scaling, no per-pixel alpha. XWayland: compatibility layer. Migration: set GDK_BACKEND=wayland, QT_QPA_PLATFORM=wayland. Screen sharing: xdg-desktop-portal. Clipboard: wl-clipboard. Notifications: mako. Remote: pipewire-remote-desktop. Compositors: sway (tiling), Hyprland (dynamic), river (zen). xdg-desktop-portal: file picker, screen share.
