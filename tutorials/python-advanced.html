Python Advanced

Advanced Python -- decorators, generators, context managers, async, and metaprogramming.

Table of Contents (2 sections)

  • Advanced Features
  • Async & Performance

Advanced Features

Decorators: @decorator, wraps(f), decorator with args. Use: logging, timing, auth, caching. Generators: yield, generator expressions, send(), throw(), close(). Use: streaming, infinite sequences, coroutines. Context managers: with statement, __enter__/__exit__, contextlib.contextmanager. Use: file handles, locks, DB connections. Metaclasses: type() as metaclass, __new__, __init_subclass__. Use: ORMs (SQLAlchemy), validation. Descriptors: __get__, __set__, __delete__ -- property, classmethod, staticmethod. Abstract base classes: ABC, @abstractmethod. Operator overloading: __add__, __eq__, __repr__, __str__, __getitem__. Slots: __slots__ for memory optimization. Dataclasses: @dataclass (auto __init__, __repr__, __eq__). Typing: type hints, generics (List[T]), Protocol (structural subtyping), TypedDict, Literal, Final.

Async & Performance

asyncio: async/await, event loop, tasks, futures. Coroutines: async def, await. awaitable: await coroutine, task, future. Async context managers: async with. Async iterators: async for. asyncio.gather: run concurrently. aiohttp: async HTTP. asyncpg: async PostgreSQL. Celery: distributed task queue (Redis/RabbitMQ broker). Performance: cProfile, py-spy (sampling profiler), memory_profiler. C extensions: Cython, cffi, C extension modules. Numba: JIT compiler for numeric code. Multiprocessing: Process, Pool, shared memory, Queue. Threading: GIL limits CPU-bound threads, use for I/O. Subprocess: run external commands. Packaging: setuptools, pyproject.toml, wheel. uv: fast Rust-based pip. pytest: fixtures, parametrize, mock monkeypatch. GIL: Global Interpreter Lock -- use multiprocessing for CPU work.
