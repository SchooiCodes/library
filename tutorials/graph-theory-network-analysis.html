Graph Theory & Network Analysis

Graph algorithms, network metrics, community detection, and graph neural networks.

Graph Representations

Graphs can be represented as adjacency matrices, adjacency lists, edge lists, or incidence matrices. Choice affects algorithm efficiency.

import networkx as nx
import matplotlib.pyplot as plt

G = nx.karate_club_graph()
print(f'Nodes: {G.number_of_nodes()}, Edges: {G.number_of_edges()}')
# Adjacency matrix
A = nx.to_numpy_array(G)
print(f'Matrix shape: {A.shape}')

Directed vs undirected, weighted vs unweighted, simple vs multigraph. Bipartite graphs have two disjoint vertex sets.

Centrality & Importance

Centrality measures identify the most important nodes in a network: degree, betweenness, closeness, eigenvector, and PageRank.

G = nx.karate_club_graph()
deg_cen = nx.degree_centrality(G)
bet_cen = nx.betweenness_centrality(G)
eig_cen = nx.eigenvector_centrality(G)
pagerank = nx.pagerank(G)

# Top nodes by degree
sorted_nodes = sorted(deg_cen.items(), key=lambda x: x[1], reverse=True)[:5]
print(f'Top 5 by degree: {sorted_nodes}')

PageRank powers Google's search. Betweenness centrality identifies bridges and bottlenecks in networks.

Community Detection

Communities are groups of densely connected nodes. Algorithms: Louvain, Girvan-Newman, label propagation, spectral clustering.

from networkx.algorithms.community import louvain_communities
communities = louvain_communities(G)
print(f'Number of communities: {len(communities)}')
for i, comm in enumerate(communities):
    print(f'Community {i}: {len(comm)} members')

Modularity measures community quality. Overlapping communities (BigCLAM) allow nodes to belong to multiple groups.

Network Flow & Matching

Maximum flow finds the maximum throughput from source to sink. Minimum cut separates source from sink with minimal edge removal.

G = nx.DiGraph()
G.add_edge('s', 'a', capacity=10)
G.add_edge('s', 'b', capacity=5)
G.add_edge('a', 't', capacity=8)
G.add_edge('b', 't', capacity=7)
G.add_edge('a', 'b', capacity=3)

flow_value, flow_dict = nx.maximum_flow(G, 's', 't')
print(f'Max flow: {flow_value}')
# Min cut = max flow (Ford-Fulkerson theorem)
cut_value, partition = nx.minimum_cut(G, 's', 't')
print(f'Min cut: {cut_value}')

Bipartite matching, assignment problem (Hungarian algorithm), and transportation networks.

Graph Neural Networks

GNNs learn from graph-structured data by aggregating information from neighboring nodes. Used in drug discovery, social networks, and recommendation.

import torch
import torch_geometric as tg
from torch_geometric.nn import GCNConv

class GCN(torch.nn.Module):
    def __init__(self, in_dim, hidden, out_dim):
        super().__init__()
        self.conv1 = GCNConv(in_dim, hidden)
        self.conv2 = GCNConv(hidden, out_dim)
    
    def forward(self, x, edge_index):
        x = self.conv1(x, edge_index).relu()
        x = self.conv2(x, edge_index)
        return x

Types: GCN, GAT (attention), GraphSAGE (sampling), GIN (isomorphism). Applications: node classification, link prediction, graph classification.
