Audio Production

Music Production Basics

DAW fundamentals, MIDI, synthesis, mixing, and mastering — start making music at home.

DAW Setup & Workflow

A Digital Audio Workstation (DAW) is the central hub for music production. Popular free/cheap DAWs: LMMS (free), Cakewalk (free), BandLab (free), Reaper (cheap), and GarageBand (free/macOS).

Key components: timeline/arrangement view, mixer (volume, pan, FX), piano roll (MIDI editing), audio clip editor, and browser (samples, presets, plugins).

Workflow: compose (MIDI/audio) → arrange (structure song) → mix (balance levels, EQ, compression) → master (final polish). Save project files as you go — crashes happen.

MIDI & Virtual Instruments

MIDI records note data (pitch, velocity, duration) not audio. Connect a MIDI keyboard via USB or draw notes in the piano roll. Velocity affects dynamics (how hard a note is played).

Virtual instruments (VSTi/AU): synthesizers (Sylenth1, Serum, Vital free), samplers (Kontakt, Decent Sampler free), drum machines (MT Power Drum Kit free), and pianos (Keyzone free).

Vital is an excellent free wavetable synth. Piano roll features: quantization (snap to grid), velocity editing, pitch bend, modulation, and sustain pedal control.

Synthesis & Sound Design

Subtractive synthesis: oscillator → filter → amplifier → effects. Waveforms: sine (pure tone), saw (rich), square (hollow), triangle (soft). Filters: low-pass, high-pass, band-pass, notch.

Envelopes (ADSR): Attack (how fast sound starts), Decay (how fast it drops), Sustain (holding level), Release (how fast it fades). Apply to amplitude, filter cutoff, and pitch.

LFO (Low Frequency Oscillator) creates modulation: vibrato (pitch), tremolo (volume), wobble (filter). Sync LFO to tempo for rhythmic effects.

Mixing Fundamentals

Mixing balances all elements in a song. Key processors per channel: EQ (remove mud, add clarity), Compressor (control dynamics), Pan (spread across stereo field), Volume (relative balance).

EQ frequencies: sub-bass (20-60Hz), bass (60-250Hz), low mids (250-500Hz), midrange (500Hz-2kHz), upper mids (2-4kHz), presence (4-6kHz), brilliance (6-20kHz).

Compression: Threshold (when it starts), Ratio (amount), Attack (how fast), Release (how long), Makeup Gain (compensate volume). Sidechain compression creates the 'pumping' effect in electronic music.

Mastering & Export

Mastering is the final polish. Chain: EQ (subtle tonal correction) → Multiband Compression (band-specific dynamics) → Limiter (maximize loudness without clipping).

LUFS measures perceived loudness. Target: -14 LUFS (streaming), -9 LUFS (radio), -6 LUFS (loud master). Use meters to check true peak (should be ≤ -1dB).

Export: WAV 24-bit for best quality (upload to streaming services), MP3 320kbps for personal use. Leave headroom (-1dB true peak) before mastering. Reference tracks help match commercial loudness.
