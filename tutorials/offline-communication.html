Offline Communication Mesh

Communicate without internet — mesh networks, LoRa radio, and emergency broadcast tools.

Table of Contents (3 sections)

  • LoRa & Meshtastic
  • Wi-Fi Mesh Protocols
  • Emergency Broadcast

LoRa & Meshtastic

Meshtastic: open-source LoRa mesh messaging. Devices: ESP32 + SX1262 LoRa radio (Heltec, LilyGO, RAK Wireless). Range: 2-10km urban, 10-30km+ line of sight. No infrastructure needed — each node relays messages. Battery life: weeks on 18650 battery. MQTT gateway: bridge Meshtastic to the internet if available. Apps: Android, iOS, Web. Use: group chat, GPS location sharing, private channels with AES encryption. Ideal for hiking, emergencies, events.

Wi-Fi Mesh Protocols

802.11s: standard wireless mesh networking — supported by Linux and many router firmwares. BATMAN-adv: kernel-level mesh protocol (better than 802.11s in dense environments). OLSR: IPv4 mesh routing protocol. Yggdrasil: IPv6 mesh with end-to-end encryption — auto-peering, works over any connection. Setup: flash OpenWrt on compatible routers (TP-Link, Xiaomi), install mesh packages, configure. Community networks: NYC Mesh, Freifunk (Berlin), Guifi.net (Spain).

Emergency Broadcast

RTL-SDR: $20 USB dongle — receive police, fire, weather radio, aircraft (ADS-B), satellite imagery (NOAA APT), and more. Transmit requires HAM license. HAM radio: technician license (US) is 35-question multiple choice. Baofeng UV-5R: $25 handheld — 5W, 2m/70cm bands. APRS: send text messages and GPS position over HAM radio without internet. Winlink: send emails via HAM radio. Both SHARES and MARS provide government emergency communications.
