📄 CachyOS Installation Guide

CachyOS is a fast, lightweight Linux distribution based on Arch Linux.

Download ISO

Visit the official CachyOS website and download the latest ISO.

Create Bootable USB

Use dd, Rufus, or Etcher to write the ISO to a USB drive.

Installation Steps

  1. Boot from the USB and select "Install CachyOS".
  2. Follow the guided installer: partition your disk (or use automatic partitioning), set hostname, and create a user.
  3. Select your preferred desktop environment (e.g., KDE Plasma, GNOME, or i3).
  4. Complete the installation and reboot into your new system.

Post‑Installation Tweaks

  • Enable the cachyos-optimizations package for performance improvements.
  • Install additional software via pacman -Syu <package> or the cachyos‑repo for extra tools.
  • Configure the firewall with ufw enable for basic security.
  • Enable system‑wide AUR support: edit /etc/pacman.conf and uncomment [archlinuxfr] section, then run pacman -S yay.
  • Optional: install a lightweight desktop environment like i3-gaps or bspwm for a faster experience.
