🎬 FFmpeg Guide

Convert, compress, trim, and manipulate video/audio with FFmpeg — the ultimate multimedia CLI tool.

Installation

# Linux (apt)
sudo apt install ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Verify
ffmpeg -version

Format Conversion

# MP4 to WebM
ffmpeg -i input.mp4 -c:v libvpx-vp9 -b:v 1M output.webm

# MP4 to GIF
ffmpeg -i input.mp4 -vf "fps=10,scale=640:-1" output.gif

# Extract audio as MP3
ffmpeg -i input.mp4 -q:a 0 -map a output.mp3
Use -crf 23 for a good quality/size trade-off in H.264 encoding. Lower = better quality, higher = smaller file.

Compression

# Compress with H.265 (HEVC) — ~50% smaller than H.264
ffmpeg -i input.mp4 -c:v libx265 -crf 28 -c:a aac -b:a 128k output.mp4

# Target a specific file size
ffmpeg -i input.mp4 -b:v 1.5M -maxrate 1.5M -bufsize 3M output.mp4

Trimming & Concatenation

# Trim: start at 30s, take 15s
ffmpeg -i input.mp4 -ss 00:00:30 -t 15 -c copy output.mp4

# Concatenate (lossless, same codec)
echo "file 'clip1.mp4'" > list.txt
echo "file 'clip2.mp4'" >> list.txt
ffmpeg -f concat -safe 0 -i list.txt -c copy merged.mp4

Audio Extraction & Manipulation

# Extract WAV
ffmpeg -i input.mp4 -vn -acodec pcm_s16le output.wav

# Speed up audio without pitch change
ffmpeg -i input.mp3 -filter:a "atempo=1.5" output.mp3

# Merge audio and video
ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac output.mp4

Creating GIFs

# High quality palette-based GIF
ffmpeg -i input.mp4 -vf "fps=15,scale=640:-1:flags=lanczos,palettegen" palette.png
ffmpeg -i input.mp4 -i palette.png -filter_complex "fps=15,scale=640:-1[x];[x][1:v]paletteuse" output.gif

Streaming & Recording

# Screen record (Linux)
ffmpeg -f x11grab -video_size 1920x1080 -i :0.0 -f pulse -i default output.mkv

# Stream to RTMP (Twitch, YouTube)
ffmpeg -re -i input.mp4 -c:v libx264 -b:v 6000k -f flv rtmp://live.twitch.tv/app/STREAM_KEY
For hardware-accelerated encoding, try -c:v h264_amf (AMD), h264_nvenc (NVIDIA), or h264_videotoolbox (Apple).

Useful One-liners

# Remove audio track
ffmpeg -i input.mp4 -an output.mp4

# Crop video (w:h:x:y)
ffmpeg -i input.mp4 -vf "crop=640:480:0:0" output.mp4

# Add subtitles (burn-in)
ffmpeg -i input.mp4 -vf "subtitles=subtitles.srt" output.mp4

# Generate thumbnail every 10 seconds
ffmpeg -i input.mp4 -vf "fps=1/10" thumb%03d.png

# Metadata strip
ffmpeg -i input.mp4 -map_metadata -1 -c:v copy -c:a copy clean.mp4
