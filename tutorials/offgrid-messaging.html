Off-Grid Messaging

Communicate when the internet goes down. These tools let you message others over Bluetooth, LoRa radio, or mesh networking without relying on cell towers or ISPs.

Briar — Android Off-Grid Messaging

Briar is an open-source messaging app that syncs over Bluetooth, Wi-Fi Direct, or Tor. No central server required.

Setup on Android

  1. Download Briar from briarproject.org or F-Droid.
  2. Open the app and create an account (no email or phone required).
  3. Add contacts by scanning a QR code in person or sharing a Briar link via another secure channel.
  4. Messages sync automatically when devices are nearby over Bluetooth or Wi-Fi.
  5. Enable Tor bridge mode if you also want remote sync over the internet.
Tip: Briar also supports forums and blogs that sync across your contact graph — useful for community coordination.

Key Features

  • End-to-end encryption by default
  • No phone number or email required
  • Bluetooth + Wi-Fi Direct + Tor transport
  • Self-destructing messages option
  • Open source (GPLv3)

Meshtastic — LoRa Mesh Networking

Meshtastic turns inexpensive LoRa radios into a long-range, low-power mesh network for text messaging and GPS position sharing.

Flashing the Firmware

  1. Download the Meshtastic firmware from flasher.meshtastic.org.
  2. Connect your LoRa board via USB.
  3. Use the web flasher (Chrome/Edge) or the meshtastic-flasher Python tool.
  4. Select your device model and hit Flash.
  5. After flashing, configure the node via the Meshtastic mobile app or CLI.

Mesh Setup

# Install the Meshtastic Python CLI
pip install meshtastic

# Set your node's region and frequency
meshtastic --set region US
meshtastic --set lora.frequency 906.875

# Set a short name and long name
meshtastic --set "owner.short" ALICE
meshtastic --set "owner.long" AliceStation1

# Enable MQTT gateway (optional)
meshtastic --set mqtt.enabled true
meshtastic --set mqtt.address mqtt.meshtastic.org
Regulatory: LoRa frequencies vary by country. Set the correct region in Meshtastic to stay legal. US = 902-928 MHz, EU = 868 MHz, AU = 915 MHz.

Yggdrasil — IPv6 Mesh Network

Yggdrasil is an overlay network that creates a mesh IPv6 network over any existing transport (Internet, Wi-Fi, Bluetooth, etc.). Nodes discover each other automatically.

Installation

# Linux (deb/rpm)
curl -L https://yggdrasil-network.github.io/install.sh | bash

# macOS with Homebrew
brew install yggdrasil

# Windows — download from GitHub releases

Configuration

# Generate config
yggdrasil --genconf > yggdrasil.conf

# Start Yggdrasil
yggdrasil --useconf < yggdrasil.conf

# Check your IPv6 address
ip addr show tun0 | grep inet6

# Peers file (add public peers)
# yggdrasil.conf peers section:
# Peers: ["tcp://peer1.example.com:12345", "tcp://peer2.example.com:12345"]

Auto-Start with systemd

sudo cp yggdrasil.conf /etc/yggdrasil.conf
sudo systemctl enable --now yggdrasil

Once connected, you can reach any other Yggdrasil node at its 200::/7 IPv6 address. Run services (HTTP, SSH, etc.) over the mesh.
