Probability & Statistics

Probability distributions, hypothesis testing, Bayesian inference — the foundation of data science and ML.

Probability Basics

Probability measures the likelihood of events. Key rules: P(A∪B) = P(A) + P(B) − P(A∩B), and conditional probability P(A|B) = P(A∩B)/P(B).

import random, math
# Monte Carlo estimation of π
inside = 0
for _ in range(100000):
    x, y = random.random(), random.random()
    if x*x + y*y <= 1:
        inside += 1
pi_est = 4 * inside / 100000
print(pi_est)

Bayes' theorem: P(A|B) = P(B|A)P(A)/P(B) — the foundation of Bayesian statistics.

Random Variables & Distributions

A random variable maps outcomes to numbers. Common distributions: normal, binomial, Poisson, uniform, exponential.

import numpy as np, matplotlib.pyplot as plt
data = np.random.normal(loc=0, scale=1, size=10000)
plt.hist(data, bins=50, density=True)
plt.title('Standard Normal Distribution')
plt.show()

Central Limit Theorem: sample means approach a normal distribution regardless of the underlying distribution (for large n).

Hypothesis Testing

Hypothesis testing determines whether observed effects are statistically significant. Null hypothesis (H₀) vs alternative (H₁).

from scipy import stats
data = np.random.normal(5.1, 1, 100)
t_stat, p_value = stats.ttest_1samp(data, 5.0)
print(f't={t_stat:.3f}, p={p_value:.3f}')
if p_value < 0.05:
    print('Reject H₀ — significant difference')

Common tests: t-test, chi-squared, ANOVA, Mann-Whitney. Type I (false positive) and Type II (false negative) errors.

Bayesian Inference

Bayesian methods update beliefs with data: posterior ∝ likelihood × prior. This contrasts with frequentist statistics.

from scipy.stats import beta
# Prior: Beta(2, 2) — weak belief coin is fair
prior = beta(2, 2)
# Data: 8 heads, 2 tails
posterior = beta(2+8, 2+2)
print(f'Posterior mean: {posterior.mean():.3f}')
print(f'95% CI: {posterior.interval(0.95)}')

Markov Chain Monte Carlo (MCMC) enables sampling from complex posterior distributions.

Regression & Correlation

Linear regression models the relationship between variables. Correlation measures the strength of linear association.

from sklearn.linear_model import LinearRegression
X = np.random.rand(100, 1) * 10
y = 2 * X.squeeze() + 1 + np.random.randn(100)
model = LinearRegression().fit(X, y)
print(f'Slope: {model.coef_[0]:.3f}, Intercept: {model.intercept_:.3f}')
print(f'R²: {model.score(X, y):.3f}')

Multiple regression, polynomial regression, regularization (Ridge, Lasso), and logistic regression for classification.
