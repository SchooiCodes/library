Mobile UI/UX Design

Design mobile interfaces — gesture design, navigation patterns, and accessibility.

Table of Contents (2 sections)

  • Design Patterns & Gestures
  • Accessibility & Platform Guidelines

Design Patterns & Gestures

Mobile navigation: tab bar (primary), stack (drill-down), drawer (secondary), search bar (discovery). Gestures: tap, double-tap, long press, swipe (dismiss, reveal actions), pinch-to-zoom, pan/drag, rotation. Touch targets: minimum 44x44pt (Apple HIG), 48x48dp (Material). Bottom sheet: action sheet, half modal, expandable sheet. Pull-to-refresh: standard pattern for content refresh. FAB (Floating Action Button): primary action, can expand to multiple. Snackbar: non-intrusive feedback. Haptic feedback: impact, notification, selection.

Accessibility & Platform Guidelines

iOS HIG: Human Interface Guidelines — clarity, deference, depth. Safe areas, notch, Dynamic Island handling. Android Material Design 3: color system (primary, secondary, tertiary, error), typography (type scale), elevation (surface tones). Accessibility: VoiceOver/TalkBack, dynamic type (large text), sufficient contrast (4.5:1 for small text), focus indicators, reduce motion, semantic labels. Adaptive layout: size classes (iOS), screen width breakpoints (Android — compact, medium, expanded). Design tools: Figma (components, auto layout, variants), Prototyping (Smart Animate, overlay).
