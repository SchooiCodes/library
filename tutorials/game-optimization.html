Game Optimization & Tweaks

Optimize games — graphics settings, FPS tweaks, launch options, and hardware tuning.

Table of Contents (2 sections)

  • Graphics Settings
  • Launch Options & Tweaks

Graphics Settings

Key graphics settings and performance impact: Render Scale/Resolution (native 100% — biggest FPS impact). Anti-aliasing: MSAA (expensive), TAA (temporal — smears), FXAA (fast but blurry), DLSS/FSR/XeSS (AI upscaling — quality > balanced > performance). Shadows: resolution, distance, soft shadows (expensive). Textures: quality mainly affects VRAM, minimal FPS impact. Reflections: screen-space (cheap), ray-traced (very expensive). Ambient Occlusion: SSAO (cheap), HBAO (better), RTAO (expensive). Motion Blur: off for competitive. Depth of Field: off for clarity. View Distance: affects CPU draw calls. Post-processing: bloom, vignette, chromatic aberration — disable. VSync: off (eliminate input lag, can cause tearing) or G-Sync/FreeSync. Frame cap: limit FPS (reduce power/heat). Nvidia Reflex: reduce system latency. DLSS 3 Frame Generation: interpolated frames — reduces GPU load, adds latency. FSR 3 Frame Generation: AMD alternative. GPU driver settings: NVIDIA Control Panel — power management (max performance), threaded optimization (on), low latency mode (ultra).

Launch Options & Tweaks

Launch options (Steam): -high (high priority), -threads X (CPU threads), -novid (skip intro), -fullscreen, -windowed. Launch options per game: CS2 (-vulkan, +exec autoexec.cfg), Valorant (-fps max 0), Apex (-novid, -fullscreen). Config files: autoexec.cfg, custom keybinds, video.txt. Graphics driver: update regularly (Game Ready drivers). Windows gaming tweaks: Game Mode (on), Hardware-accelerated GPU scheduling, Disable fullscreen optimizations, High performance power plan. Background apps: close browsers, overlays (Discord, Xbox Game Bar) — reduce CPU contention. BIOS/UEFI: XMP/EXPO (RAM speed), Resizable BAR/SAM (GPU memory access). Monitor overclock: custom resolution higher refresh rate. FPS monitors: MSI Afterburner + RivaTuner, Steam overlay FPS. Benchmarking: 3DMark, Superposition, in-game benchmarks (Cyberpunk, Shadow of the Tomb Raider). Per-game optimizations: low settings for competitive FPS, balanced for single-player. Mouse: raw input, disable acceleration, DPI (400-1600), polling rate (1000Hz). Keyboard: NKRO (n-key rollover).
