Go Programming

Go language -- concurrency, interfaces, tooling, and web development.

Table of Contents (2 sections)

  • Concurrency & Types
  • Tooling & Web

Concurrency & Types

Goroutines: go func() -- lightweight thread. Channels: ch := make(chan int) -- communicate between goroutines. Select: wait on multiple channel operations. WaitGroup: sync.WaitGroup. Mutex: sync.Mutex. Struct: type User struct { Name string; Age int }. Methods: func (u User) Greet() string. Interfaces: type Greeter interface { Greet() string }. Satisfied implicitly. Error: type error interface { Error() string }. Multiple return: func div(a,b int) (int, error). Defer: deferred function calls (cleanup). Pointers: *T vs T. Slice: dynamic array. Map: map[string]int. Range: for k, v := range map.

Tooling & Web

Go tool: go build, test, fmt, vet, mod, get. go mod: modules (go.mod, go.sum). Testing: _test.go files, go test -v, sub tests, table-driven tests. Benchmarks: func BenchmarkXxx(b *testing.B). pprof: profiling. net/http: ServeMux, Handler, middleware. Popular frameworks: Gin, Echo, Fiber (web), GORM (ORM), sqlx (DB), chi (router). json: encoding/json. Context: request-scoped values, cancellation, deadlines. Template: text/template, html/template. io.Reader/Writer: interface patterns. CLI: os.Args, flag, cobra. HTTP client: net/http client with timeout, transport. Graceful shutdown: signal.Notify, server.Shutdown. Microservices: standard lib + protobuf/gRPC. Docker: scratch or distroless base images. Cross-compile: GOOS=linux GOARCH=amd64. Static binary: CGO_ENABLED=0.
