🐧 Essential Linux CLI Tools

Supercharge your terminal with modern replacements for classic Unix commands — faster search, better output, and tools you won't want to live without.

🐧 Why These Tools?

The standard Unix tools (grep, find, ls, cat, top) have been around for decades. These modern alternatives are faster, have better defaults, and produce much nicer output — all while being drop-in replacements.

🔍 Search & Find

ripgrep (rg)

Blazingly fast grep replacement. Recursively searches your codebase for patterns. Respects .gitignore by default.

rg "function" src/
rg -i "error" --type py
GitHub →

fd

Fast find replacement. Simple, intuitive syntax with colorized output.

fd "\.py$"
fd -e md -x cat {}
GitHub →

fzf

Fuzzy finder for anything. Pipe input to fzf for interactive filtering.

Ctrl+R (history search)
Ctrl+T (file search)
GitHub →

👁️ View & Preview

bat

cat with syntax highlighting, git integration, and line numbers. Drop-in replacement for cat.

bat file.py
bat --show-all config.yaml
GitHub →

lsd

Modern ls with icons, colors, and tree view.

lsd -la
lsd --tree
GitHub →

tealdeer

Fast tldr client. Simplified man pages with practical examples.

tldr tar
tldr ripgrep
GitHub →

duf

Better disk usage. Shows mounted devices in a clean table.

duf
duf --only local
GitHub →

📊 System Monitoring

btop

Modern resource monitor with GPU support, themes, and mouse support.

btop
GitHub →

glances

System monitoring top/htop alternative with web UI option.

glances
glances -w (web mode)
Website →

procs

Modern ps replacement with colorized, human-readable output.

procs
procs --tree
GitHub →

💻 Terminal Enhancements

fish

Friendly interactive shell with auto-suggestions, syntax highlighting, and web config.

sudo apt install fish
chsh -s /usr/bin/fish
Website →

tmux

Terminal multiplexer. Split windows, persist sessions, attach remotely.

tmux new -s work
Ctrl+B % (split)
Ctrl+B d (detach)
GitHub →

ranger

File manager with VI keybindings in the terminal. Browse files with style.

ranger
GitHub →

📦 Git Tools

lazygit

Terminal UI for git. Stage, commit, branch, rebase — all from a beautiful TUI.

lazygit
GitHub →

delta

Syntax-highlighting pager for git diff output.

git diff | delta
GitHub →

📥 Installation

Most of these tools are available in your distro's package manager:

# Debian/Ubuntu
sudo apt install bat ripgrep fzf tmux ranger duf btop fish

# Arch
sudo pacman -S bat ripgrep fzf tmux ranger duf btop fish

# macOS (Homebrew)
brew install bat ripgrep fzf tmux ranger duf btop fish lazygit

For the latest versions, check each tool's GitHub releases page.
