CSS Animations & Transitions

CSS animations, transitions, keyframes -- smooth, performant web animations.

Table of Contents (2 sections)

  • Transitions & Keyframes
  • Performance & JS

Transitions & Keyframes

Transitions: transition-property, duration, timing-function (ease, cubic-bezier), delay. Trigger: hover, focus, class change. Performance: transform + opacity only (GPU composited). will-change: hint browser. Keyframes: @keyframes name { from { } to { } }. Animation: name, duration, timing, delay, iteration, direction, fill-mode, play-state. Stagger: animation-delay on children. Houdini: Paint API, Animation Worklet. Scroll-driven: scroll(), view() CSS. Container queries.

Performance & JS

GPU properties: transform (translate, scale, rotate, skew) + opacity. Avoid: width, height, top, left, margin, padding, box-shadow. rAF: requestAnimationFrame for JS animations. Web Animations API: element.animate(). FLIP: First, Last, Invert, Play. Libraries: GSAP, Framer Motion, Anime.js. prefers-reduced-motion: respect user preference. 3D: perspective, rotateX/Y/Z. Easing: cubic-bezier for natural movement. Steps: for typewriter effect. Debug: DevTools Animations panel.
