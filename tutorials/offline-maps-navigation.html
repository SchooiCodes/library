Offline Maps & Navigation

Navigate anywhere without an internet connection. This guide covers the best tools for downloading maps, importing GPX tracks, and navigating offline.

Organic Maps

Organic Maps is a free, open-source navigation app that uses OpenStreetMap data. No ads, no tracking, fully offline.

Setup & GPX Import

  1. Install Organic Maps from organicmaps.app, F-Droid, or your app store.
  2. Open the app and download map regions for your area (Settings → Download Maps).
  3. To import a GPX track: transfer the .gpx file to your phone, then tap it in your file manager — Organic Maps will open it.
  4. Alternatively, place GPX files in OrganicMaps/gpx/ on your device storage.
  5. Tracks appear under Bookmarks → Tracks. You can follow them on the map.
Tip: Organic Maps supports offline search, turn-by-turn navigation, and public transit — all without internet.

OsmAnd — Advanced Offline Maps

OsmAnd is a feature-rich offline map app with contour lines, altitude data, POI search, and GPX recording.

Advanced Features

  • Contour lines & hillshade: Download from Plugins → Contour Lines and Hillshade.
  • GPX recording: Use the Trip Recording plugin to log your route.
  • Offline POI search: Search for shops, fuel, hospitals without data.
  • Voice navigation: Download voice packs for turn-by-turn guidance.
  • Map styles: Choose from topographic, nautical, winter ski, and more.

Google Maps Offline Download

Google Maps supports offline area downloads for navigation.

  1. Open Google Maps and search for a location.
  2. Tap the bottom info panel and select Download.
  3. Adjust the area rectangle and tap Download.
  4. Offline maps appear in your profile → Offline maps.
  5. Get driving directions within the downloaded area without data.
Limitations: Offline maps expire after 15 days unless auto-updates are enabled. Transit, cycling, and walking directions are limited offline.

Mobile Atlas Creator (MOBAC)

MOBAC creates offline atlases from online map sources (OpenStreetMap, Google Maps, etc.) for use in GPS apps.

Installation

# Requires Java 8+
wget https://mobac.org/downloads/mobac-latest.zip
unzip mobac-latest.zip
cd mobac
java -jar MobileAtlasCreator.jar

Creating a Custom Atlas

  1. Select a map source (OSM, Google Sat, OpenTopoMap, etc.).
  2. Define the area by selecting a region or entering coordinates.
  3. Choose zoom levels (1-18; higher = more detail = larger file).
  4. Select output format: OsmAnd (.obf), RMaps (.sqlitedb), Garmin (.img), or PNG tiles.
  5. Click Create Atlas and wait for the download.
Tip: Use the "OSM Mapnik (standard)" source for the most up-to-date OpenStreetMap tiles. Transfer the resulting file to your phone's map directory.

GPS Navigation Without Internet

GPS works independently of cellular or Wi-Fi. Your phone's GPS chip calculates position from satellites — no data needed.

Best Practices

  • Download maps before you leave signal range.
  • Pre-load GPX routes for trails or backroads.
  • Use airplane mode + GPS to save battery (disable mobile data).
  • Carry a portable power bank — GPS navigation drains the battery.
  • Dedicated handheld GPS units (Garmin) are more rugged and last longer.
