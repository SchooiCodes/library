Azure Cloud Services

Microsoft Azure — VMs, Blob Storage, Azure Functions, and Active Directory.

Azure Virtual Machines

Azure VMs run Windows and Linux workloads. Series: B (burstable, low-cost), D (general purpose), E (memory optimized), F (compute), N (GPU — NVIDIA A100, H100). Availability sets (fault/update domains) and availability zones protect against failures.

Azure Dedicated Host gives you physical servers for compliance. Azure Spot VMs (deep discounts, interruptible) for batch workloads. VM extensions: Azure Monitor agent, custom script, DSC (Desired State Configuration).

Azure Blob Storage

Blob Storage is Azure's object storage for unstructured data. Tiers: Hot (frequent access), Cool (30+ days), Cold (90+ days), Archive (180+ days, rehydration takes hours). Access tiers can be set at the blob level — automatically reprice based on patterns.

ADLS Gen2 (Azure Data Lake Storage) adds hierarchical namespace (directories) and POSIX permissions. Azure Files provides managed file shares (SMB/NFS) accessible from VMs and on-premises. AzCopy is the CLI tool for data transfer.

Azure Functions Serverless

Azure Functions run code on-demand with triggers (HTTP, Timer, Blob, Queue, Event Grid, CosmosDB). Plan types: Consumption (pay-per-execution, auto-scale to zero), Premium (always warm, VNet integration, higher timeouts), and Dedicated (on App Service plan).

Durable Functions extend serverless with orchestrations (stateful workflows), entities (virtual actors), and external events. Functions can be written in C#, JavaScript, Python, Java, PowerShell, and TypeScript.

Azure Active Directory

Azure AD (now Entra ID) is Microsoft's cloud identity and access management. It provides SSO for SaaS apps (Office 365, Salesforce), application proxy (on-prem apps via cloud), and Conditional Access (MFA, device compliance, location-based policies).

Key features: Privileged Identity Management (PIM — just-in-time admin access), Identity Protection (risk-based detection, automated remediation), B2B collaboration (external user access), B2C (customer identity), and managed identities (automatic Azure service auth).

Azure DevOps

Azure DevOps is Microsoft's DevOps platform. Services: Azure Repos (Git repos), Azure Pipelines (CI/CD for any platform), Azure Boards (work tracking — Kanban/Scrum), Azure Test Plans (manual/exploratory testing), and Azure Artifacts (package feeds for NuGet, npm, Maven, PyPI).

Pipelines support YAML-based definitions, multi-stage deployments, environment approvals, and Kubernetes integration. GitHub Actions is the newer alternative with deeper GitHub integration, richer ecosystem, and free minutes for public repos.
