Linux Ricing Guide

Rice your Linux desktop — themes, icons, cursors, conky, polybar, and aesthetic system-wide customization.

What is Ricing?

Ricing is the art of customizing your Linux desktop's appearance and workflow. Originating from car culture (RICER = Race Inspired Cosmetic Enhancements), it now refers to making your desktop look beautiful and efficient.

Components you can rice: window manager/bars (Polybar, Waybar, Eww), compositor (Picom), notification daemon (Dunst), app launcher (Rofi), terminal emulator (Alacritty, Kitty, foot), shell prompt (Starship, Powerlevel10k), and system monitor (Conky, bottom, btop).

Themes, Icons & Cursors

GTK themes control the look of GTK apps (Firefox, Nautilus, Thunar, Gedit). Icon themes replace system icons. Cursor themes change pointer style.

# Install popular themes
yay -S catppuccin-gtk-theme-mocha
# Install icon pack
yay -S papirus-icon-theme
# Set via ~/.config/gtk-3.0/settings.ini

[Settings]
gtk-theme-name=Catppuccin-Mocha-Compact-Blue
gtk-icon-theme-name=Papirus-Dark
gtk-cursor-theme-name=Bibata-Modern-Classic

Qt themes (for KDE apps) can match GTK via the breeze or kvantum theming engines.

Polybar & Waybar Configuration

Polybar (X11) and Waybar (Wayland) are customizable status bars. Example minimal Waybar config:

// ~/.config/waybar/config.jsonc
{
    "layer": "top",
    "position": "top",
    "modules-left": ["hyprland/workspaces", "custom/playerctl"],
    "modules-center": ["clock"],
    "modules-right": ["pulseaudio", "network", "battery", "tray"],
    "clock": {
        "format": "{:%H:%M  %A %d-%b}",
        "tooltip-format": "{:%Y-%m-%d | %H:%M:%S}"
    },
    "battery": {
        "format": "{icon} {capacity}%",
        "format-charging": "⚡ {capacity}%",
        "format-icons": ["", "", "", "", ""]
    }
}

Waybar supports CSS styling — create a style.css in the same directory to customize colors, padding, and fonts.

Rofi, Dunst & Conky

Rofi is a popup window switcher, app launcher, and dmenu replacement. Theme it via ~/.config/rofi/config.rasi:

configuration {
    modi: "drun,run,window";
    icon-theme: "Papirus-Dark";
    show-icons: true;
    drun-display-format: "{name}";
    font: "FiraCode Nerd Font 12";
}

* {
    bg-col: #1e1e2e;
    fg-col: #cdd6f4;
    accent: #89b4fa;
    width: 600;
}

Dunst notifications: ~/.config/dunst/dunstrc. Conky system monitor: ~/.config/conky/conky.conf. Both support Lua scripting for complex widgets.

Git-Based Dotfile Management

Manage your rice with a dotfiles repo. Popular approaches: bare git repo, GNU Stow, Chezmoi, or YADM.

# Bare git repo method
git init --bare ~/.dotfiles
alias dotfiles='/usr/bin/git --git-dir=$HOME/.dotfiles --work-tree=$HOME'
dotfiles config status.showUntrackedFiles no
dotfiles add ~/.config/hyprland.conf ~/.config/waybar ~/.config/rofi
dotfiles commit -m "Initial rice"
dotfiles remote add origin git@github.com:user/dotfiles.git
dotfiles push -u origin main

Tip: use .config/ symlinks for portability across distros. Share your rice on r/unixporn for community feedback!
