📄 How to Create High‑Quality Embeds (Not GIFs)

Embedding Videos (e.g., Discord)

This tutorial shows how to embed a video so that platforms like Discord display a high‑quality preview (not a GIF) with optional sound on mobile devices.

1. Choose a video

Select a short clip (max 3 minutes) that you want to share.

2. Upload to Imgur

  1. Log in or create an Imgur account.
  2. Click "New post", drag & drop your video file.
  3. Ensure the "Keep audio" option is enabled.
  4. Publish the post.

3. Copy the direct link

After upload, click "Copy link". Use the direct link (ending in .mp4 or .webm).

4. Share the link

Paste the link into Discord (or any platform that supports rich embeds). The service will automatically generate a preview.

Tips

  • For best quality, use MP4 with H.264 video and AAC audio.
  • Keep the file size under 8 MiB to avoid compression.
  • If the embed shows as a GIF, try using a .webm file instead.
