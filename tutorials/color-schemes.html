Terminal Colors 101

Terminal colors are defined using ANSI escape codes — special sequences that tell the terminal to change text attributes. The standard 16-color palette consists of 8 basic colors plus their bright variants:

# ANSI escape code structure
\033[<attribute>;<foreground>;<background>m

# Examples
\033[31m      # Red text
\033[1;32m     # Bold green text
\033[41m      # Red background
\033[1;33;44m   # Bold yellow text on blue background
\033[0m      # Reset all attributes
CodeColorCode (Bright)Bright Color
30Black90Bright Black (Gray)
31Red91Bright Red
32Green92Bright Green
33Yellow93Bright Yellow
34Blue94Bright Blue
35Magenta95Bright Magenta
36Cyan96Bright Cyan
37White97Bright White

Modern terminals support 256 colors and 24-bit true color (16.7 million colors). Use echo $TERM to check your terminal type — xterm-256color or tmux-256color indicate 256 color support. Test true color with a 24-bit color test script.

Here are the most beloved terminal color schemes in the developer community:

Tools for Managing Themes

  • pywal — Generate a terminal color scheme from an image. Extracts the dominant colors from any wallpaper and applies them to your terminal and other apps: wal -i /path/to/image.jpg.
  • themer — Generate color themes for dozens of apps (terminal, VS Code, Slack, Alfred, etc.) from a single color palette.
  • Gogh — Bash script that applies 200+ color themes to GNOME Terminal or any terminal that supports profile colors.
  • kitty-themes — A collection of kitty color schemes with a handy theme selector tool.
  • Terminal.sexy — Web-based terminal color scheme designer. Create and export themes in many formats.

Theme Setup for Popular Terminals

Kitty (kitty.conf)

# ~/.config/kitty/kitty.conf
include catppuccin-mocha.conf

background #1e1e2e
foreground #cdd6f4
color0 #45475a
color1 #f38ba8
color2 #a6e3a1
color3 #f9e2af
color4 #89b4fa
color5 #f5c2e7
color6 #94e2d5
color7 #bac2de

WezTerm (wezterm.lua)

-- ~/.config/wezterm/wezterm.lua
local wezterm = require 'wezterm'
return {
 color_scheme = 'Catppuccin Mocha',
 colors = {
  background = '#1e1e2e',
  foreground = '#cdd6f4',
 },
}

Windows Terminal (settings.json)

LS_COLORS, bat & fzf

These tools use color to make your terminal experience richer:

  • LS_COLORS — Controls the colors of ls output. Customize via $LS_COLORS environment variable or use tools like dircolors. Popular preset: trapd00r/LS_COLORS.
  • bat — A cat clone with syntax highlighting and Git integration. Themes are configured via bat --list-themes and set in ~/.config/bat/config with --theme="Catppuccin-mocha". Export theme with bat --generate-config-file.
  • fzf — Fuzzy finder with customizable colors. Set in ~/.fzfrc or via FZF_DEFAULT_OPTS environment variable: export FZF_DEFAULT_OPTS="--color=bg+:#313244,fg:#cdd6f4,hl:#f38ba8".
  • delta — A syntax-highlighting pager for git. Integrates with bat themes for beautiful diffs.
# Set LS_COLORS from a preset
eval $(dircolors ~/.config/dircolors)

# Set fzf colors for Catppuccin Mocha
export FZF_DEFAULT_OPTS=" \
 --color=bg+:#313244,bg:#1e1e2e,spinner:#f5c2e7,hl:#f38ba8 \
 --color=fg:#cdd6f4,header:#f38ba8,info:#cba6f7,pointer:#f5c2e7 \
 --color=marker:#f5c2e7,fg+:#cdd6f4,prompt:#cba6f7,hl+:#f38ba8"

# Configure bat theme
echo '--theme="Catppuccin-mocha"' >> ~/.config/bat/config
Pro tip: Once you find a scheme you love, use pywal to generate a cohesive color palette from your wallpaper. It can theme your terminal, VS Code, Spotify (via spicetify), and more with one command.
