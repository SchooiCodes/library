📺 Tmux Guide

Terminal multiplexing with tmux — sessions, windows, panes, keybindings, customisation, and pair programming.

Sessions

# Create a named session
tmux new -s myproject

# Detach: Ctrl+b d
# List sessions
tmux ls

# Attach to session
tmux attach -t myproject

# Kill session
tmux kill-session -t myproject
Tmux sessions survive SSH disconnects. Run long tasks inside tmux and never worry about broken pipes again.

Windows & Panes

# Create window: Ctrl+b c
# Rename window: Ctrl+b ,
# Next/prev window: Ctrl+b n / Ctrl+b p
# Kill window: Ctrl+b &

# Split horizontally: Ctrl+b "
# Split vertically: Ctrl+b %
# Navigate panes: Ctrl+b arrow-keys
# Resize: Ctrl+b Ctrl+arrow
# Close pane: Ctrl+b x
# Launch a pre-built session layout
tmux new -s dev \; \
 send-keys 'nvim .' C-m \; \
 split-window -h \; \
 send-keys 'npm run dev' C-m \; \
 split-window -v \; \
 send-keys 'git status' C-m

Configuration (.tmux.conf)

unbind C-b
set -g prefix C-a
bind C-a send-prefix

# Mouse support
set -g mouse on

# Increase scrollback
set -g history-limit 50000

# Status bar
set -g status-bg colour235
set -g status-fg white
set -g status-left '#[fg=green]#S '

# Vi-style navigation
setw -g mode-keys vi

# Reload config
bind r source-file ~/.tmux.conf \; display "Reloaded!"

Copy Mode

Enter with Ctrl+b [. Navigate with vi/emacs keys. Copy selection with Space then Enter. Paste with Ctrl+b ].

bind C-c run "tmux save-buffer - | pbcopy"  # macOS clipboard
bind C-c run "tmux save-buffer - | xclip -i -sel clipboard" # Linux

Pair Programming

tmux -S /tmp/pair new -s pair            # Host: create socket
ssh user@host; tmux -S /tmp/pair attach -t pair   # Guest: attach
tmux -S /tmp/pair attach -t pair -r         # Read-only mode
Use tmux new -AD -s name to auto-attach or create a session, kicking out any existing attachment.

Scripting tmux

#!/bin/bash
SESSION="monitor"
tmux new-session -d -s $SESSION
tmux send-keys -t $SESSION 'htop' C-m
tmux split-window -h -t $SESSION
tmux send-keys -t $SESSION 'watch -n 2 df -h' C-m
tmux split-window -v -t $SESSION
tmux send-keys -t $SESSION 'journalctl -f' C-m
tmux attach -t $SESSION
