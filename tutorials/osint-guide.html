What is OSINT?

OSINT (Open Source Intelligence) is the collection and analysis of publicly available information. Sources include search engines, social media, public records, domain registries, and data breaches. It's used by security researchers, penetration testers, journalists, and law enforcement.

Key principle: All information used must be legally obtained from public sources. No hacking, no unauthorized access.

Google Dorking

Advanced search operators to find hidden or specific information.

# Search within a specific site
site:example.com "confidential"

# File type search
filetype:pdf "classified"

# Search for exposed directories
intitle:"index of" "backup"

# Find login pages
inurl:login site:example.com

# Search by page title
intitle:"admin panel"

# Search cached version
cache:example.com

# Combined dorks
inurl:wp-admin site:example.com filetype:php

GHDB (Google Hacking Database) at exploit-db.com has thousands of pre-built dorks.

Tip: Use view-source:https://example.com to see the raw HTML source of any page — often reveals hidden comments, endpoints, or API keys.

Domain / WHOIS Analysis

# WHOIS lookup
whois example.com

# DNS records
dig example.com ANY
dig example.com A
dig example.com MX
dig example.com TXT

# Subdomain enumeration
dig example.com NS
for sub in $(cat subdomains.txt); do
 dig $sub.example.com A +short
done

# Reverse DNS
dig -x 8.8.8.8

# Using host
host -a example.com

# Check SSL cert transparency (crt.sh)
curl -s "https://crt.sh/?q=example.com&output=json"

Shodan

Shodan is a search engine for internet-connected devices. It scans IP ranges and indexes banners (SSH, HTTP, FTP, etc.).

# Search for a specific service
port:22 country:US

# Find webcams with default passwords
"default password" webcam

# Search by organization
org:"Example Corp"

# Find vulnerable RDP
port:3389 has_screenshot:true

# Search by ASN
asn:AS15169

Use Shodan.io with a free account or shodan CLI tool.

Social Media Searches

  • Twitter: from:username since:2025-01-01, geocode:lat,lon,radius
  • LinkedIn: Search by company, role, location. Use Google dorks: site:linkedin.com/in "Software Engineer" "Company"
  • Reddit: site:reddit.com username, use redditsearch.io
  • GitHub: site:github.com "email@example.com", search commits, gists, repos
  • Telegram: lyzem.com for channel/message search

Tools: theHarvester, Sherlock (username search across 400+ platforms), Maigret.

Email Lookup

# theHarvester — find emails associated with a domain
theHarvester -d example.com -b google,linkedin

# Hunter.io — API for email patterns
# h8mail — check email in breach data
h8mail -t target@example.com

# Have I Been Pwned — check breach status
curl -s "https://haveibeenpwned.com/api/v3/breachedaccount/target@example.com"

Staying Legal

  • Only collect publicly available data — no unauthorized access.
  • Respect robots.txt and terms of service.
  • Do not use OSINT for doxxing, harassment, or stalking.
  • Data retention laws vary by country — know your jurisdiction.
  • For penetration testing, always have written authorization.
Reminder: OSINT is a powerful defensive skill. Use it to audit your own exposure — search your own name, email, and domains to see what attackers can find.
