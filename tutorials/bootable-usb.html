📄 Creating Bootable USB Drives

Table of Contents (4 sections)

  • Tools
  • Using dd (Linux/macOS)
  • Using Rufus (Windows)
  • Using Etcher (Cross‑platform)

This guide covers creating bootable USB media for Linux distributions, Windows installers, and recovery tools.

Tools

  • dd – native Unix tool (Linux/macOS).
  • Rufus – Windows GUI tool.
  • Etcher – cross‑platform GUI tool.

Using dd (Linux/macOS)

sudo dd if=/path/to.iso of=/dev/sdX bs=4M status=progress oflag=sync

Replace /dev/sdX with your USB device identifier (check with lsblk).

Using Rufus (Windows)

  1. Download Rufus.
  2. Select your USB drive.
  3. Choose the ISO file and click "Start".

Using Etcher (Cross‑platform)

  1. Download Etcher.
  2. Open the app, select the image, select the USB, and click "Flash!".
