Differential Equations

ODE and PDE fundamentals, analytical and numerical solutions, and applications in physics and engineering.

First-Order ODEs

First-order ODEs involve only the first derivative. Forms: separable, linear (integrating factor), exact, Bernoulli.

from sympy import dsolve, symbols, Function, Eq
t = symbols('t')
y = Function('y')
# y' + 2ty = 0 → separable
equation = Eq(y(t).diff(t) + 2*t*y(t), 0)
solution = dsolve(equation, y(t))
print(solution)

Slope fields visualize solutions. Existence and uniqueness theorems guarantee conditions for a unique solution through a point.

Higher-Order Linear ODEs

Second-order ODEs are fundamental in physics (harmonic oscillator, pendulum). Characteristic equation determines solution form.

from sympy import dsolve, symbols, Function, Eq
t = symbols('t')
y = Function('y')
# y'' + 4y' + 4y = 0 → repeated root: r = -2
equation = Eq(y(t).diff(t,2) + 4*y(t).diff(t) + 4*y(t), 0)
solution = dsolve(equation, y(t))
print(solution)

Method of undetermined coefficients and variation of parameters for non-homogeneous equations. Wronskian for linear independence of solutions.

Laplace Transforms

The Laplace transform converts ODEs into algebraic equations. L{f(t)} = ∫₀^∞ e⁻ˢᵗ f(t) dt, making solving much easier.

from sympy import laplace_transform, exp, sin, symbols
t, s = symbols('t s')
print(laplace_transform(exp(2*t), t, s))  # 1/(s-2)
print(laplace_transform(sin(3*t), t, s))  # 3/(s²+9)

Properties: linearity, differentiation L{f′}=sF(s)−f(0), convolution L{f*g}=F(s)G(s). Inverse transform via partial fractions.

Systems of ODEs

Systems of ODEs model coupled phenomena. Phase plane analysis reveals stability and behavior of solutions.

from scipy.integrate import solve_ivp
import numpy as np

def lorenz(t, state, σ=10, ρ=28, β=8/3):
    x, y, z = state
    return [σ*(y-x), x*(ρ-z)-y, x*y - β*z]

sol = solve_ivp(lorenz, [0, 50], [1, 1, 1], method='RK45', max_step=0.01)

Stability analysis: fixed points, Jacobian eigenvalues, limit cycles, and chaos (Lorenz attractor).

Partial Differential Equations

PDEs involve partial derivatives of multivariable functions. Canonical types: elliptic (Laplace), parabolic (heat), hyperbolic (wave).

import numpy as np
# 1D heat equation: u_t = α·u_xx (finite difference)
α, dx, dt = 0.01, 0.1, 0.001
x = np.arange(0, 1, dx)
u = np.sin(np.pi * x)  # Initial condition
for _ in range(100):
    u[1:-1] += α * dt/dx**2 * (u[2:] - 2*u[1:-1] + u[:-2])

Separation of variables, Fourier series, method of characteristics, and numerical methods (FDM, FEM, FVM).
