Numerical Methods

Root finding, interpolation, numerical integration, ODE solvers — computational techniques for engineering.

Root Finding

Find roots of equations that lack analytic solutions. Methods: bisection, Newton-Raphson, secant, and Brent.

from scipy.optimize import newton, bisect
def f(x): return x**3 - x - 2
root_newton = newton(f, 2.0)  # 1.521
root_bisect = bisect(f, 1, 2)  # 1.521
print(root_newton, root_bisect)

Newton's method converges faster but needs derivatives. Bisection is reliable but slower. Brent's method combines both.

Interpolation & Approximation

Interpolation constructs a function passing exactly through given data points. Approximation finds the best fit with error tolerance.

from scipy.interpolate import interp1d
x = np.array([0, 1, 2, 3, 4])
y = np.array([0, 1, 4, 9, 16])
f_linear = interp1d(x, y, kind='linear')
f_cubic = interp1d(x, y, kind='cubic')
print(f_linear(2.5), f_cubic(2.5))

Polynomial interpolation (Lagrange, Newton), spline interpolation (cubic, B-spline), and least squares approximation.

Numerical Integration

Approximate definite integrals when analytic integration is impossible. Common methods: trapezoidal, Simpson, Gaussian quadrature.

from scipy.integrate import quad, simpson
f = lambda x: np.exp(-x**2)
result, error = quad(f, -np.inf, np.inf)
print(f'∫e⁻ˣ² dx from -∞ to ∞ = {result:.6f} (expected √π = {np.sqrt(np.pi):.6f})')

Monte Carlo integration scales to high dimensions. Adaptive quadrature refines the grid where the function varies rapidly.

Ordinary Differential Equations

ODEs describe how quantities change over time. Numerical solvers handle equations without closed-form solutions.

from scipy.integrate import solve_ivp
def pendulum(t, y):
    theta, omega = y
    return [omega, -9.81/1.0 * np.sin(theta)]

sol = solve_ivp(pendulum, [0, 10], [np.pi/4, 0], method='RK45')
import matplotlib.pyplot as plt
plt.plot(sol.t, sol.y[0])
plt.xlabel('Time (s)')
plt.ylabel('Angle (rad)')
plt.title('Pendulum Motion')
plt.show()

Methods: Euler, Runge-Kutta (RK4), multistep (Adams-Bashforth), and implicit methods for stiff equations.

Fast Fourier Transform

The FFT efficiently computes the discrete Fourier transform, converting signals from time to frequency domain.

from scipy.fft import fft, fftfreq
fs = 1000  # Sampling rate (Hz)
t = np.linspace(0, 1, fs, endpoint=False)
signal = np.sin(2*np.pi*50*t) + 0.5*np.sin(2*np.pi*120*t)
X = fft(signal)
freqs = fftfreq(fs, 1/fs)
# Plot magnitude spectrum
plt.plot(freqs[:fs//2], np.abs(X)[:fs//2])
plt.xlabel('Frequency (Hz)')
plt.ylabel('Magnitude')
plt.show()

FFT is O(n log n) compared to O(n²) for naive DFT. Used in audio processing, image compression, and solving PDEs.
