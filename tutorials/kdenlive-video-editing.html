Video Editing with Kdenlive

Professional non-linear video editing with Kdenlive — the free open-source video editor.

Table of Contents (5 sections)

Kdenlive Interface & Setup

Kdenlive (KDE Non-Linear Video Editor) is a powerful free video editor for Linux, Windows, and macOS. It supports unlimited tracks, proxy editing, and a wide range of effects.

Interface: Project Bin (media library), Timeline (multi-track), Monitor (preview), Effect Stack (applied effects), and Properties (clip parameters). Custom layouts save as screen profiles.

Configure: Settings → Configure Kdenlive → Timeline (default duration, transitions), Playback (preview resolution), Proxy (automatic proxy generation for 4K video).

Timeline Editing

Drag clips from Project Bin to the Timeline. Cut (Razor tool, X key), trim (drag edge), slip (Shift+click), and slide content to refine edits.

Ripple edit (default): moving/clipping a clip shifts subsequent clips. Overwrite (insert): pushes clips forward. Lift: removes clip leaving gap. Timeline markers (V key) label important points.

Keyboard shortcuts: I (set in point), O (set out point), V (split clip), Shift+R (ripple delete), Ctrl+Click (select multiple), Shift+drag (snap to nearby clip edge).

Effects & Transitions

Effects: color correction (Levels, RGB Curves, HSL adjust), blur, transform (position/scale/rotate), speed (slow-mo/time-lapse), keyframeable parameters for animation.

Transitions: dissolve, wipe, slide, composite, and Luma Wipe. Drag between overlapping clips on the same track. Default transition can be set in settings.

Keyframes animate effects over time. Click the diamond icon next to a parameter to set a keyframe. Move playhead, change parameter value, and a new keyframe is created automatically.

Audio & Color Workflow

Audio editing: separate audio from video (Shift+Alt+G), adjust volume with keyframes, apply fade in/out (audio transitions). Audio effects: compressor, equalizer, reverb, and noise reduction.

Color grading: RGB Curves for creative looks, Levels for exposure, Hue/Saturation for color intensity, and Lift/Gamma/Gain for shadows/midtones/highlights balance.

Color scopes: Waveform monitor (brightness distribution), Vectorscope (color/hue), Histogram (tonal range), and RGB Parade (individual channel levels). Essential for consistent color.

Export & Rendering

Project → Render (Ctrl+Enter) opens the rendering dialog. Presets: YouTube (1080p/4K), Vimeo, Mastodon/Pixelfed, and custom. Choose format: MP4 (H.264/AAC) for compatibility.

Advanced settings: codec (H.264/H.265/VP9), bitrate (CBR/VBR), quality slider, resolution, frame rate, and audio codec (AAC/MP3/FLAC/PCM). GPU acceleration via VA-API/NVENC for faster encoding.

Proxy editing: Kdenlive automatically creates low-res proxies for 4K footage, enabling smooth editing. Toggle proxy view on/off. Final render uses full-resolution source.
