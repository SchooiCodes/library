What is Web Scraping?

Web scraping is the automated extraction of data from websites. It involves fetching web pages, parsing the HTML content, and extracting structured information like prices, headlines, product details, or contact data.

Python is the most popular language for web scraping due to its rich ecosystem of libraries like Requests (HTTP client), BeautifulSoup (HTML parsing), Scrapy (full scraping framework), and Selenium (browser automation).

Common use cases include price monitoring, news aggregation, research data collection, lead generation, and building datasets for machine learning.

Legal & Ethical Considerations

Before scraping any website, review its robots.txt file and terms of service. The robots.txt (e.g., https://example.com/robots.txt) tells scrapers which paths are disallowed:

# Example robots.txt
User-agent: *
Disallow: /private/
Disallow: /api/
Allow: /public/

# Ethical scraping guidelines:
# 1. Check robots.txt before scraping
# 2. Rate-limit requests (1 request per second minimum)
# 3. Identify yourself with a User-Agent header
# 4. Scrape only what you need
# 5. Respect copyright and data ownership

Legal considerations vary by jurisdiction. Scraping publicly accessible data is generally legal, but bypassing login walls, scraping copyrighted content for commercial use, or exceeding rate limits may violate laws like the CFAA in the US or GDPR in Europe for personal data.

Tip: Always check the website's terms of service. Many sites explicitly allow or prohibit scraping in their ToS.

BeautifulSoup Basics

BeautifulSoup is the most widely used Python library for parsing HTML and XML documents:

from bs4 import BeautifulSoup

html_doc = """
<html><body>
  <h1>Welcome</h1>
  <p class="content">Hello, world!</p>
  <a href="https://example.com">Click here</a>
</body></html>
"""

soup = BeautifulSoup(html_doc, 'html.parser')

# Find elements
h1 = soup.find('h1')
p = soup.find('p', class_='content')
link = soup.find('a')

# Get text and attributes
print(h1.text)      # Welcome
print(p.text)      # Hello, world!
print(link['href'])   # https://example.com

Install with pip install beautifulsoup4 lxml. The lxml parser is faster than Python's built-in html.parser.

Requests & HTTP

The requests library handles HTTP communication with websites:

import requests

# Simple GET request
response = requests.get('https://example.com')
print(response.status_code) # 200
print(response.text)     # HTML content

# Custom headers (identify yourself)
headers = {
  'User-Agent': 'MyScraper/1.0 (educational scraping)',
  'Accept': 'text/html'
}
response = requests.get('https://example.com', headers=headers)

# Handle query parameters
params = {'q': 'python scraping', 'page': 1}
response = requests.get('https://search.example.com', params=params)

# Handle errors
try:
  response = requests.get('https://example.com', timeout=10)
  response.raise_for_status() # Raises on 4xx/5xx
except requests.exceptions.RequestException as e:
  print(f"Request failed: {e}")

# Use sessions for cookies and connection pooling
session = requests.Session()
session.headers.update({'User-Agent': 'MyScraper/1.0'})
response = session.get('https://example.com')

Parsing HTML

BeautifulSoup provides multiple methods for navigating and searching the parse tree:

soup = BeautifulSoup(html, 'html.parser')

# Find by tag name
all_links = soup.find_all('a')

# Find by CSS class
items = soup.find_all('div', class_='item')

# Find by attributes
images = soup.find_all('img', src=True)
inputs = soup.find_all('input', type='text')

# Navigate the tree
for div in soup.find_all('div', class_='article'):
  title = div.find('h2').text
  link = div.find('a')['href']
  summary = div.find('p', class_='summary').text.strip()
  print(f"{title}: {summary}")

# Find parent/sibling
td = soup.find('td', string='Price')
price_cell = td.find_next_sibling('td')
price = price_cell.text.strip()

# Using select (CSS selector method)
soup.select('div.article h2 a')
soup.select('#main-content p:first-child')
soup.select('ul.items > li:nth-child(2)')

Handling JavaScript

Some websites load content dynamically with JavaScript. For these, you need a browser automation tool:

# Using Selenium
from selenium import webdriver
from selenium.webdriver.chrome.options import Options
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

options = Options()
options.add_argument('--headless') # Run without GUI
driver = webdriver.Chrome(options=options)

driver.get('https://example.com')

# Wait for dynamic content to load
wait = WebDriverWait(driver, 10)
element = wait.until(
  EC.presence_of_element_located((By.CLASS_NAME, 'dynamic-content'))
)

# Get page source after JS execution
html = driver.page_source
soup = BeautifulSoup(html, 'html.parser')
print(soup.find('div', class_='dynamic-content').text)

driver.quit()

# Alternative: requests-html library
# pip install requests-html
from requests_html import HTMLSession
session = HTMLSession()
response = session.get('https://example.com')
response.html.render() # Execute JavaScript
print(response.html.find('div.dynamic-content', first=True).text)

Rate Limiting & Politeness

Always respect the website's resources. Implement delays between requests and handle rate limits gracefully:

import time
import random

def polite_delay():
  """Wait between 1-3 seconds"""
  time.sleep(random.uniform(1, 3))

        print(f"Rate limited. Waiting {retry_after}s")
        time.sleep(retry_after)

Additional best practices: respect Cache-Control headers, use conditional GETs with If-Modified-Since, and set a descriptive User-Agent with contact information.

Storing Data

Save scraped data in structured formats for later use:

import csv
import json

# Save to CSV
def save_to_csv(data, filename='scraped_data.csv'):
  if not data:
    return
  
  keys = data[0].keys()
  with open(filename, 'w', newline='', encoding='utf-8') as f:
    writer = csv.DictWriter(f, fieldnames=keys)
    writer.writeheader()
    writer.writerows(data)

# Save to JSON
def save_to_json(data, filename='scraped_data.json'):
  with open(filename, 'w', encoding='utf-8') as f:
    json.dump(data, f, indent=2, ensure_ascii=False)

# Save to SQLite
import sqlite3

def save_to_db(data, db_name='scraped.db'):
  conn = sqlite3.connect(db_name)
  cursor = conn.cursor()
  
  cursor.execute('''
    CREATE TABLE IF NOT EXISTS products (
      name TEXT,
      price TEXT,
      url TEXT UNIQUE,
      scraped_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
    )
  ''')
  
  for item in data:
    cursor.execute(
      'INSERT OR IGNORE INTO products (name, price, url) VALUES (?, ?, ?)',
      (item['name'], item['price'], item['url'])
    )
  
  conn.commit()
  conn.close()

Real-World Example

Here is a complete script that scrapes book titles and prices from a demo bookstore:

import requests
from bs4 import BeautifulSoup
import csv
import time

BASE_URL = 'https://books.toscrape.com'
HEADERS = {'User-Agent': 'EducationalScraper/1.0'}

def scrape_books():
  books = []
  page = 1
  
  while True:
    url = f"{BASE_URL}/catalogue/page-{page}.html"
    response = requests.get(url, headers=HEADERS)
    
    if response.status_code != 200:
      break
    
    soup = BeautifulSoup(response.text, 'html.parser')
    articles = soup.find_all('article', class_='product_pod')
    
    if not articles:
      break
    
    for article in articles:
      title = article.find('h3').find('a')['title']
      price = article.find('p', class_='price_color').text
      rating = article.find('p', class_='star-rating')['class'][1]
      link = article.find('h3').find('a')['href']
      
      books.append({
        'title': title,
        'price': price,
        'rating': rating,
        'url': BASE_URL + '/catalogue/' + link
      })
    
    print(f"Scraped page {page}: {len(articles)} books")
    page += 1
    time.sleep(1)
  
  return books

def main():
  print("Starting scraper...")
  books = scrape_books()
  
  with open('books.csv', 'w', newline='', encoding='utf-8') as f:
    writer = csv.DictWriter(f, fieldnames=['title', 'price', 'rating', 'url'])
    writer.writeheader()
    writer.writerows(books)
  
  print(f"Done! Scraped {len(books)} books to books.csv")

if __name__ == '__main__':
  main()
