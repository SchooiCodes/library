Introduction

JavaScript is a versatile programming language used for interactive web development.

Syntax

Core syntax rules with a simple example:

// This is a single line comment
var x = 5; // Declare a variable
var y = 6;
var z = x + y; // Perform addition
console.log(z); // Output the result

Examples

Alert Box

<script>

</script>

Changing HTML Content

<script>

</script>
