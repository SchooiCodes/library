Linux From Scratch

Build Linux from source -- LFS, BLFS, toolchain, and customization.

Table of Contents (2 sections)

  • LFS Build
  • BLFS & Custom

LFS Build

LFS: build Linux from source. Host: existing Linux, 10GB+, 4+ CPU. Steps: prepare host, build temporary toolchain (cross-compiler), build system (chroot), configure (boot scripts, kernel, grub). Toolchain: binutils, gcc, glibc. Build order: binutils pass1 -> gcc pass1 -> linux headers -> glibc -> binutils pass2 -> gcc pass2 -> coreutils. Chroot: chroot $LFS /tools/bin/env -i... /tools/bin/bash. Kernel: make menuconfig, modules_install. Boot: grub-mkconfig. LFS version 12.2.

BLFS & Custom

BLFS: packages after LFS -- networking (DHCP, SSH), X11, desktop (GNOME, KDE), multimedia (ALSA, PipeWire, ffmpeg), office, browser. Systemd or SysV. Benefits: complete control, minimal size, deep understanding. Challenges: long compile, dependency management. Gentoo: Portage similar approach. Config: /etc/fstab, hostname, hosts, resolv.conf. Users/groups: useradd, passwd. Kernel modules: essential drivers.
