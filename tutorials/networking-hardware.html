Networking Hardware

Network hardware — routers, switches, access points, cables, and home networking.

Table of Contents (2 sections)

  • Routers & Switches

Routers & Switches

Routers: connect different networks, perform NAT, firewall, routing. Home router: all-in-one (router + switch + AP + modem). Advanced: pfSense/OPNsense (DIY router on PC), Ubiquiti UniFi (Dream Machine, EdgeRouter), MikroTik (RouterBOARD, RouterOS — powerful but steep learning curve). Features: VLAN support, QoS (traffic shaping — fq_codel, CAKE), VPN server, ad-blocking (Pi-hole), parental controls. Switches: connect devices within LAN. Managed vs unmanaged: managed = VLANs, STP, LACP, SNMP, port mirroring, PoE. PoE standards: 802.3af (15.4W), 802.3at (30W — PoE+), 802.3bt Type 3 (60W), Type 4 (90W — PoE++). Switch form factors: desktop, rackmount, fanless (silent for home). L2 vs L3 switches: L2 (MAC-based switching), L3 (IP routing — inter-VLAN routing). Stacking: multiple switches as one logical unit. SFP/SFP+: fiber or DAC uplink ports (1G SFP, 10G SFP+, 25G SFP28, 100G QSFP28).

Access Points & Cabling

Access Points (APs): wireless connectivity — Ubiquiti UniFi (U6-LR, U7-Pro), TP-Link Omada (EAP670), Aruba Instant On, Ruckus. Mesh systems: eero, Google Nest WiFi, TP-Link Deco, Asus ZenWiFi — easy setup, AI roaming. Controller-based: UniFi Controller (software or CloudKey + 'gateway) — manage multiple APs centrally (roaming, band steering, RF optimization). Ethernet: Cat5e (1Gbps, 100m), Cat6 (10Gbps, 55m), Cat6a (10Gbps, 100m), Cat7 (proprietary), Cat8 (25/40Gbps, 30m). Shielded (STP) vs unshielded (UTP): STP reduces interference, needs proper grounding. Keystone jacks, patch panels, punch-down tools. Fiber optic: single-mode (OS2 — long distance, 9/125um core, LC connector) vs multi-mode (OM3/OM4/OM5 — short distance, 50/125um). Media converters: fiber to copper. Cable testing: continuity, wiremap (T568A/B), signal loss (TDR). UPS: battery backup for network equipment — runtime for modem, router, switch, AP. Rack: 19-inch server rack, patch panel, cable management.
