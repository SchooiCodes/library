📦 Linux Package Management

Master APT, Pacman, DNF, Flatpak, Snap, and AUR — install, update, search, and troubleshoot packages.

apt (Debian/Ubuntu)

APT is the package manager for Debian-based distros. It uses .deb packages and resolves dependencies automatically. Commands are run with sudo.

# Update package lists
sudo apt update

# Upgrade all packages
sudo apt upgrade

# Full distribution upgrade
sudo apt full-upgrade

# Install a package
sudo apt install nginx

# Remove a package
sudo apt remove nginx

# Purge (remove + config files)
sudo apt purge nginx

# Search for a package
apt search "web server"

# Show package info
apt show nginx

# Clean up unused dependencies
sudo apt autoremove

pacman (Arch)

Pacman is the package manager for Arch Linux and its derivatives. It uses .pkg.tar.zst packages. Pacman is fast and uses a rolling-release model.

# Sync and upgrade
sudo pacman -Syu

# Install a package
sudo pacman -S nginx

# Remove a package
sudo pacman -R nginx

# Remove with dependencies
sudo pacman -Rs nginx

# Remove with configs
sudo pacman -Rns nginx

# Search
pacman -Ss "web server"

# List installed packages
pacman -Q

# Show orphaned packages
pacman -Qdt
sudo pacman -Rns $(pacman -Qdtq)
Tip: Always run sudo pacman -Syu before installing new packages on Arch. Never use --force — use --overwrite with caution.

dnf (Fedora)

DNF is the next-generation package manager for RPM-based distros (Fedora, RHEL 8+). It's a fork of YUM with better performance and dependency resolution.

# Update
sudo dnf update

# Install
sudo dnf install nginx

# Remove
sudo dnf remove nginx

# Search
dnf search nginx

# Package info
dnf info nginx

# List installed
dnf list installed

# Group install
sudo dnf groupinstall "Web Server"

# Clean cache
sudo dnf clean all

Flatpak & Snap

Flatpak and Snap are universal package formats that bundle dependencies with the application. They run in sandboxed environments and work across all Linux distros.

# Flatpak
flatpak search "spotify"
flatpak install flathub com.spotify.Client
flatpak run com.spotify.Client
flatpak update
flatpak list

# Snap
snap find "spotify"
sudo snap install spotify
snap refresh
snap list
sudo snap remove spotify

# Where packages live
flatpak list --app  # user apps
snap list       # system snaps

AUR Helpers (yay)

The Arch User Repository (AUR) is a community-driven repository of PKGBUILD scripts. yay is a popular helper that searches both the official repos and the AUR.

# Install yay
sudo pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay-bin.git
cd yay-bin && makepkg -si

# Search AUR and repos
yay "package-name"

# Install from AUR
yay -S google-chrome

# Update AUR packages
yay -Sua

# Clean build cache
yay -Sc

# Remove orphaned deps
yay -Yc

Troubleshooting

Package managers can fail due to broken dependencies, corrupted caches, or lock files. Here's how to recover.

# APT fixes
sudo apt --fix-broken install
sudo dpkg --configure -a
sudo rm /var/lib/apt/lists/lock
sudo rm /var/cache/apt/archives/lock

# Pacman fixes
sudo rm /var/lib/pacman/db.lck
sudo pacman -Syu --overwrite="*"

# DNF fixes
sudo dnf clean all
sudo dnf distro-sync

# Clear Flatpak cache
flatpak uninstall --unused
sudo flatpak repair

# Reset Snap
snap changes
sudo snap revert
Tip: If a lock file prevents apt/pacman/dnf from running, remove it manually (rm /var/lib/.../lock), but only when no package manager is actually running.
