RAM & Memory

Memory — DDR5, DIMM, latency, dual-channel, ECC, and memory overclocking.

Table of Contents (2 sections)

  • DDR5 & Memory Technologies

DDR5 & Memory Technologies

DDR (Double Data Rate): data on both clock edges. Generations: DDR4 (2133-5333 MT/s, 1.2V), DDR5 (4800-8400+ MT/s, 1.1V — on-die ECC, two 32-bit channels per module, PMIC on board). DIMM types: DIMM (desktop), SO-DIMM (laptop), RDIMM/LRDIMM (server — registered/buffered). Latency: CAS latency (CL — cycles between column address and data). CL30-36 typical for DDR5. True latency = (CL / data rate) * 2000 — DDR5-6000 CL30 = 10ns. Dual-channel: two DIMMs in alternating channels (A2/B2) — 2x memory bandwidth. Quad-channel: high-end desktop (AMD Threadripper, Intel Xeon W). Memory ranks: single-rank (1 DRAM chip set), dual-rank (2 sets — more parallelism). XMP (Intel Extreme Memory Profile) / EXPO (AMD Extended Profiles for Overclocking): pre-configured OC profiles in SPD. Memory overclocking: increase frequency, tighten timings — stability test with MemTest86, TM5, y-cruncher. 1usmus DRAM Calculator.

ECC & Virtual Memory

ECC (Error-Correcting Code): detect and correct single-bit errors, detect double-bit. On-die ECC (DDR5): corrects internal DRAM errors. Full ECC (server): additional data bus width (72-bit vs 64-bit) — requires compatible CPU and motherboard. Registered/Buffered: RDIMM has register between DRAM and memory controller — reduces electrical load, supports more DIMMs per channel. ECC vs non-ECC: ECC critical for servers, workstations (data integrity). Virtual memory: OS uses disk as extension of RAM — page file (Windows), swap (Linux). Pages: 4KB default, 2MB huge pages (Linux), 1GB pages. TLB (Translation Lookaside Buffer): cache for virtual-to-physical mappings. NUMA (Non-Uniform Memory Access): memory closer to some CPU sockets — latency differs. Memory tiering: CXL-attached memory as slower tier. Persistent memory: Intel Optane (discontinued), CXL-attached NVDIMM.
