Windows Security

Windows security — Defender, BitLocker, Windows Hello, and attack surface reduction.

Table of Contents (2 sections)

  • Windows Defender & ASR
  • BitLocker & Windows Hello

Windows Defender & ASR

Microsoft Defender: real-time protection, cloud-delivered, MPEngine. Config: Group Policy, Intune, PowerShell (Set-MpPreference). Attack Surface Reduction (ASR): rules — block Office child processes, block executable content from email, block credential stealing from LSASS. Controlled Folder Access: protect documents from ransomware. Exploit Guard: network protection, exploit protection (DEP, ASLR, CFG). Defender for Endpoint: EDR — behavioral detection, automated investigation, threat analytics.

BitLocker & Windows Hello

BitLocker: full disk encryption — TPM + PIN or USB key. Recovery: 48-digit recovery key (back up to AD or Microsoft account). BitLocker To Go: removable drives. Layers: AES-128/256, diffuser (Elephant). Manage-bde: manage-bde -status C:, manage-bde -protectors -get C:. Windows Hello: biometric (fingerprint, face) or PIN — tied to TPM, replaces passwords. Credential Guard: isolate secrets (Kerberos, NTLM) in virtualized container — prevents pass-the-hash. Local Security Authority (LSA) protection: lsass.exe running as Protected Process Light.
