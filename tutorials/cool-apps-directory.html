Cool Apps & Tools Directory

Hand-picked collection of the most useful applications across productivity, development, media, utilities, and security.

Table of Contents

Productivity

Boost your workflow with these powerful productivity applications. From note-taking to task management, these tools help you stay organized and efficient.

Obsidian

Local-first knowledge base and note-taking app. Uses plain Markdown files. Highly extensible with 1000+ plugins, graph view, and linking. Free for personal use. Sync and publish are paid.

Platforms: Windows, macOS, Linux, iOS, Android

Site: obsidian.md

Notion

All-in-one workspace for notes, databases, wikis, and project management. Powerful blocks, templates, and collaboration features. Generous free tier for individuals.

Platforms: Web, Windows, macOS, iOS, Android

Site: notion.so

Todoist

Cross-platform task manager with natural language input, labels, filters, and project views. Clean interface with kanban boards, calendar, and recurring tasks. Excellent collaboration.

Platforms: All major platforms

Site: todoist.com

Espanso

Cross-platform text expander. Create custom shortcuts that expand into longer text. Supports dates, clipboard, shell commands, and scripts. Open-source and lightweight.

Platforms: Windows, macOS, Linux

Site: espanso.org

TickTick

Task manager with built-in Pomodoro timer, habit tracker, and calendar view. Natural language input, tags, and lists. Free tier is generous. Syncs across all devices.

Platforms: All major platforms

Site: ticktick.com

Development

Essential tools for developers — from code editors and version control to debugging and containerization. These apps form the backbone of modern software development.

VS Code

The most popular code editor. Lightweight, extensible via extensions, built-in Git, debugger, terminal, and IntelliSense. Supports every major language. Free and open-source.

Platforms: Windows, macOS, Linux

Site: code.visualstudio.com

Git / GitHub Desktop

Git is the industry-standard version control system. GitHub Desktop provides a user-friendly GUI for Git operations — commits, branches, pull requests, and merge conflict resolution.

Platforms: Windows, macOS

Site: desktop.github.com

Docker Desktop

Containerization platform for developing, shipping, and running applications in isolated environments. Docker Desktop includes Docker Engine, CLI, Compose, and a GUI dashboard.

Platforms: Windows, macOS, Linux

Site: docker.com

Windows Terminal

Modern terminal for Windows with tabs, panes, GPU acceleration, and customization. Supports PowerShell, CMD, WSL, and any shell. Open-source by Microsoft.

Platforms: Windows

Site: github.com/microsoft/terminal

Postman

API development and testing platform. Build, test, document, and monitor APIs with a clean GUI. Supports REST, GraphQL, WebSocket, and SOAP. Free tier for individuals.

Platforms: Windows, macOS, Linux

Site: postman.com

VS Code Extensions

Essential extensions: Prettier (formatter), ESLint (linting), GitLens (Git superpowers), GitHub Copilot (AI), Live Server, Thunder Client (API client), Material Icon Theme, and more.

Source: VS Code Marketplace

Site: marketplace.visualstudio.com

Media

Powerful media players, editors, and recorders for audio, video, and image content. These tools range from lightweight players to professional-grade editing suites.

MPV Player

Minimalist, lightning-fast media player controlled via keyboard. Supports nearly every format via FFmpeg. Highly scriptable with Lua. GPU-accelerated rendering. No GUI clutter.

Platforms: Windows, macOS, Linux

Site: mpv.io

VLC Media Player

The Swiss Army knife of media playback. Plays anything out of the box — DVDs, streaming URLs, damaged files. Includes conversion, streaming server, and recording. Free and open-source.

Platforms: Windows, macOS, Linux, Android, iOS

Site: videolan.org

DaVinci Resolve

Professional video editing, color grading, visual effects, and audio post-production. Free version is incredibly capable (limited to 4K and some features). Industry-standard color tools.

Platforms: Windows, macOS, Linux

Site: blackmagicdesign.com

Audacity

Free, open-source audio editor. Record, edit, splice, and apply effects to audio tracks. Supports multitrack editing, noise reduction, and a wide range of formats.

Platforms: Windows, macOS, Linux

Site: audacityteam.org

Krita

Professional open-source painting and illustration software. Packed with brushes, animation tools, and layer management. Great for digital art, concept art, and texture painting.

Platforms: Windows, macOS, Linux

Site: krita.org

GIMP

GNU Image Manipulation Program. Photoshop alternative with powerful photo retouching, composition, and image editing. Supports layers, masks, filters, batch processing, and scripting.

Platforms: Windows, macOS, Linux

Site: gimp.org

Utilities

Indispensable utility tools that optimize your workflow, manage files, clean up disk space, and automate repetitive tasks. These are the apps you will use every day.

ShareX

Advanced screenshot tool with screen recording, OCR, color picker, image editor, and file upload. Automatically uploads to Imgur, Dropbox, or custom hosts. Open-source and feature-packed.

Platforms: Windows

Site: getsharex.com

Ditto Clipboard Manager

Clipboard history manager. Copies everything you save to the clipboard, searchable history with preview. Supports images, text, HTML, and custom formats. Syncs across machines.

Platforms: Windows

Site: ditto-cp.sourceforge.io

Everything (Void Tools)

Instant file search for Windows. Indexes NTFS volumes in seconds. Type anything and get results in milliseconds. Supports regex, boolean operators, and HTTP server for remote search.

Platforms: Windows

Site: voidtools.com

TreeSize / WinDirStat

Disk space analyzers. TreeSize (free version) shows folder sizes in Explorer-like view. WinDirStat uses treemap visualization. Both show exactly what is eating your storage.

Platforms: Windows

Site: windirstat.net

Rufus

Bootable USB creator. Create installation media for Windows, Linux, and other ISOs. Supports UEFI, GPT, persistent storage, and bad block checks. Small, fast, and reliable.

Platforms: Windows

Site: rufus.ie

7-Zip

High-compression file archiver with AES-256 encryption. Supports 7z, XZ, ZIP, GZIP, TAR, and more. Built-in file manager. Significantly better compression than WinRAR in most cases.

Platforms: Windows, Linux (p7zip)

Site: 7-zip.org

Notepad++

Source code and text editor. Syntax highlighting for 80+ languages, macro recording, find/replace with regex, plugins. Tabbed interface. Lightweight and fast.

Platforms: Windows

Site: notepad-plus-plus.org

Greenshot

Lightweight screenshot tool. Capture regions, windows, or full screen. Annotate, highlight, blur, and export directly to file, printer, or clipboard. Open-source.

Platforms: Windows

Site: getgreenshot.org

OBS Studio

Free and open-source software for video recording and live streaming. High-performance real-time video/audio capture, mixing, and compositing. Industry standard for streamers.

Platforms: Windows, macOS, Linux

Site: obsproject.com

Security

Essential security tools to protect your privacy, encrypt your data, and browse safely. These tools range from password managers to disk encryption and VPN clients.

Bitwarden

Open-source password manager. Stores passwords, notes, credit cards, and identities end-to-end encrypted. Free tier is unlimited. Self-hosting option available. Browser and mobile apps.

Platforms: All major platforms + browser

Site: bitwarden.com

VeraCrypt

On-the-fly disk encryption software. Create encrypted virtual disks or encrypt entire partitions/drives. Supports hidden volumes (plausible deniability). Successor to TrueCrypt.

Platforms: Windows, macOS, Linux

Site: veracrypt.fr

ProtonVPN

Privacy-focused VPN from Proton (makers of Proton Mail). Strong encryption, no-logging policy, and a free tier (limited servers, no speed cap). Paid plans include streaming servers.

Platforms: All major platforms

Site: protonvpn.com

Tor Browser

Anonymity-focused browser based on Firefox. Routes traffic through the Tor network. Essential for accessing .onion sites. Not recommended for everyday browsing (slow).

Platforms: Windows, macOS, Linux

Site: torproject.org

SimpleWall / Portmaster

Windows firewall managers. SimpleWall provides a clean interface for Windows Filtering Platform. Portmaster is a network monitor with application-level blocking. Both are free.

Platforms: Windows

Site: safing.io

Open Source Gems

Lesser-known but incredibly useful open-source applications that deserve a spot in your toolkit. These projects are community-driven, privacy-respecting, and often outperform commercial alternatives.

LocalSend

Cross-platform file sharing over local network. No internet required. Encrypted transfer. Works like AirDrop for Windows, Linux, and Android. No accounts needed.

Platforms: Windows, macOS, Linux, Android

Site: localsend.org

Spicetify

Customize your Spotify client with themes, extensions, and CSS tweaks. Install custom themes, remove ads, add keyboard shortcuts. Works on desktop Spotify. Open-source.

Platforms: Windows, macOS, Linux

Site: spicetify.app

Scoop / Chocolatey

Command-line package managers for Windows. Scoop installs portable apps without admin rights. Chocolatey automates software installation. Both eliminate manual setup.

Platforms: Windows

Site: scoop.sh

Yazi

Blazing-fast terminal file manager written in Rust. Vim-like keybindings, image previews, code highlighting, and built-in file search. Async operations and multi-tab support.

Platforms: Linux, macOS, Windows

Site: yazi-rs.github.io

Upscayl

AI image upscaler that runs locally. Uses Real-ESRGAN models to upscale and enhance images. Free, open-source, and works offline. Supports batch processing.

Platforms: Windows, macOS, Linux

Site: upscayl.org

FlowLauncher

Open-source keystroke launcher for Windows. Search files, launch apps, run commands, calculate math, and browse the web — all from a popup. Plugin system for extensibility.

Platforms: Windows

Site: flowlauncher.com
