🚫 Ad Blocking Guide

Block ads and trackers with uBlock Origin, Pi-hole, AdGuard Home, NextDNS, and browser-based filtering.

Browser Extensions — uBlock Origin

uBlock Origin is the gold standard. Lightweight, efficient, blocks ads, trackers, and malicious domains.

# Key features:
- Default lists: EasyList, EasyPrivacy, Peter Lowe's, Fanboy's
- Element picker mode (right-click → "Block element")
- Dynamic filtering (per-site permissions)
- Advanced user mode (medium mode / hard mode)

# Recommended extra filter lists (enable in settings):
- AdGuard Annoyances (cookies, popups)
- Fanboy's Annoyances
- RUS (regional if needed)
- uBlock filters – Badware risks

# Dynamic filtering — medium mode:
- Global: block 3rd-party scripts & frames
- Per-site: allow as needed (click the shield icon)

uBO also blocks crypto miners, clickers, and trackers. No "acceptable ads" — it blocks everything unless you explicitly allow.

Tip: Enable uBlock Origin Extra (separate addon) for anti-DRM circumvention and anti-Detection blocking.

Pi-hole — DNS-Level Blocking

Runs on a Raspberry Pi (or any Linux machine). Acts as a DNS sinkhole — blocks ads at the network level for every device.

# Install on Debian/Ubuntu
curl -sSL https://install.pi-hole.net | bash

# During setup:
# - Select your network interface
# - Choose upstream DNS provider (Cloudflare, Quad9, etc.)
# - Use default blocklists (add more later)

# After install — set Pi-hole as your DHCP DNS server
# Or manually set each device's DNS to the Pi-hole IP

# Useful commands
pihole status
pihole -g       # update gravity (blocklists)
pihole -up       # update Pi-hole itself
pihole -a -q example.com # query logs
pihole -a -b ads.example.com # manually whitelist

# Web admin: http://pi.hole/admin

Combine Pi-hole with unbound (local recursive DNS) for maximum privacy — no upstream DNS logging.

AdGuard Home

Alternative to Pi-hole with a modern web UI, DHCP server, DNS-over-HTTPS/TLS, and parental controls.

# Install on Linux
curl -sSL https://raw.githubusercontent.com/AdguardTeam/AdGuardHome/master/scripts/install.sh | sh

# Docker
docker run --name adguardhome --restart unless-stopped \
 -v /opt/adguard/work:/opt/adguard/work \
 -v /opt/adguard/conf:/opt/adguard/conf \
 -p 53:53/tcp -p 53:53/udp \
 -p 80:80/tcp -p 3000:3000/tcp \
 adguard/adguardhome

# Features:
- DNS filtering + DHCP server
- Encrypted DNS (DoH, DoT, DoQ)
- Per-client rules
- Built-in AdGuard filter list
- Query log with search

NextDNS

Cloud-based DNS filtering with no hardware required. Free tier allows 300,000 queries/month.

# Setup:
1. Sign up at https://nextdns.io
2. Create a configuration
3. Use the linked IPv4/IPv6 addresses

# Linux (stubby or systemd-resolved)
# Or install the CLI:
curl -sSL https://nextdns.io/install | sh

# Recommended blocklists:
- NextDNS Ads & Trackers
- OISD Big
- 1Hosts (Lite or Pro)
- Hagezi's lists

# Also enables:
- CNAME flattening
- DNS rebinding protection
- IDN homograph protection

NextDNS is excellent for mobile devices — install the app on iOS/Android for system-wide blocking even off Wi-Fi.

Tip: For zero-config ad blocking at home, set NextDNS as your router's DNS server. Whole network protected in 2 minutes.

Hardening Firefox / Chrome

Firefox about:config:

privacy.trackingprotection.fingerprinting.enabled = true
privacy.trackingprotection.cryptomining.enabled = true
privacy.resistFingerprinting = true
network.dns.disablePrefetch = true
network.http.referer.XOriginPolicy = 2
media.peerconnection.enabled = false # disable WebRTC leak

Chrome flags: chrome://flags — not much for privacy. Use uBlock Origin + ClearURLs extension. Consider Ungoogled Chromium or Brave instead.
