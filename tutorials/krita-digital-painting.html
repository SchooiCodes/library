Krita Digital Painting

Digital painting, concept art, and illustration with Krita — the professional free painting software.

Krita Basics & Workspace

Krita is a free, open-source painting program designed for artists. It excels at digital painting, concept art, textures, comics, and matte painting.

Interface: Canvas, Toolbox (left), Tool Options (top), Brush Presets (top), Layers (right), Color Selector. Workspaces: Painting, Animation, Illustration, and Custom.

Configure canvas with Ctrl+Alt+R (wrap-around mode for seamless textures). Use the Navigator docker for zoom preview. Multiple document support with tabs.

Brushes & Brush Engines

Krita has 9 brush engines: Pixel, Sketch, Color Smudge, Shape, Bristle, Spray, Chalk, Hatching, and Particle. Each offers unique behavior.

Brush settings: size, opacity, flow, spacing, rotation, color, and scatter. Brush tips can be imported from PNG/GIMP brushes. Tag presets for organization.

Stabilizer smooths strokes: basic stabilization, smoother, weighted smooth, and stabilizer (predictive). Pressure sensitivity requires a graphics tablet (Wacom, Huion, XP-Pen).

Wrap-around mode in brush settings creates seamlessly tiling textures — perfect for game assets and pattern design.

Layers & Blending Modes

Krita's layer system supports paint layers, group layers, vector layers, filter layers, clone layers, and file layers. Each has specific use cases.

Blending modes: Normal, Multiply (shadows), Screen (highlights), Overlay (contrast), Soft Light (gentle), Color Dodge (bright), Linear Burn (dark), and more.

Filter masks apply effects non-destructively. Transparency masks control visibility. Transformation masks animate position/scale/rotation. Use layer styles for drop shadow, bevel, and glow.

Color Management

Krita fully supports color management with ICC profiles. Work in sRGB (web), Adobe RGB (print), or Rec.709 (video). Choose the right color space in Settings → Configure Krita → Color Management.

Color selectors: Default (circle), Small HSV, Triangle, Ring, and the innovative Pop-up Palette (right-click on canvas). The Advanced Color Selector shows complementary, split-complementary, and analogous harmonies.

Gamut masks constrain your palette to specific color ranges — essential for color theory practice and consistent illustration styles.

Animation in Krita

Krita includes a full 2D animation system. Enable Animation workspace. The Timeline docker manages frames, onion skinning, and playback.

Creating frame-by-frame animation: draw on each frame, use onion skin to see previous/next frames as ghosts. Auto frame creation mode simplifies workflow.

Export animation as GIF, MP4, AVI, or PNG sequence. The FFmpeg export option gives the best quality and compression. Use the Animation Curves docker for precise timing control.
