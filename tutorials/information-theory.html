Information Theory

Entropy, mutual information, KL divergence, and channel capacity — the mathematical theory of communication.

Entropy & Information Content

Entropy measures the average information content of a random variable. H(X) = −∑p(x)log₂p(x) in bits.

def entropy(probs):
    return -sum(p * np.log2(p) for p in probs if p > 0)

fair_coin = [0.5, 0.5]
biased = [0.9, 0.1]
print(f'Fair coin: {entropy(fair_coin):.3f} bits')
print(f'Biased coin: {entropy(biased):.3f} bits')

Joint entropy H(X,Y), conditional entropy H(X|Y), and the chain rule: H(X,Y) = H(X) + H(Y|X).

Mutual Information

Mutual information I(X;Y) measures the reduction in uncertainty about X given knowledge of Y. I(X;Y) = H(X) − H(X|Y).

from sklearn.feature_selection import mutual_info_classif
# Feature selection with mutual information
mi_scores = mutual_info_classif(X, y)
print('MI scores:', mi_scores)
# Higher MI = more relevant for prediction

MI is zero iff X and Y are independent. Unlike correlation, it captures non-linear relationships. Uses: feature selection, clustering evaluation.

KL Divergence & Cross-Entropy

Kullback-Leibler divergence measures how one probability distribution diverges from another. D_KL(P‖Q) = ∑P(i)log(P(i)/Q(i)).

def kl_divergence(p, q):
    return sum(p[i] * np.log2(p[i]/q[i]) for i in range(len(p)))

P = [0.5, 0.5]
Q = [0.9, 0.1]
print(f'D_KL(P‖Q): {kl_divergence(P, Q):.3f}')
print(f'D_KL(Q‖P): {kl_divergence(Q, P):.3f}')  # Asymmetric!

Cross-entropy H(P,Q) = H(P) + D_KL(P‖Q). Used as loss function in classification (categorical cross-entropy) and model distillation.

Channel Capacity & Coding

Channel capacity is the maximum mutual information over all input distributions. Shannon's noisy channel coding theorem defines the fundamental limit.

# Binary symmetric channel capacity
def bsc_capacity(p_error):
    if p_error == 0.5: return 0
    H = -p_error*np.log2(p_error) - (1-p_error)*np.log2(1-p_error)
    return 1 - H

for p in [0, 0.1, 0.25, 0.4, 0.5]:
    cap = bsc_capacity(p)
    print(f'p={p}: C={cap:.3f} bits/use')

Source coding (compression) and channel coding (error correction). Huffman coding, LZ77, Reed-Solomon, LDPC codes.

Applications: ML & NLP

Information theory underpins many ML techniques: decision tree splitting (information gain), word embeddings (PMI), and variational autoencoders (ELBO).

# Information gain for decision tree
def info_gain(parent, left, right):
    n = len(parent)
    nl, nr = len(left), len(right)
    return entropy(parent) - (nl/n)*entropy(left) - (nr/n)*entropy(right)

Mutual information for feature selection, cross-entropy loss for neural networks, and KL divergence for variational inference and model compression.
