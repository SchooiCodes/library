Password Reset & Recovery

Reset forgotten passwords, bypass lock screens, and recover credentials — legally and safely.

Table of Contents (3 sections)

  • Windows Password Reset
  • Linux Password Reset
  • macOS & Mobile Recovery

Windows Password Reset

chntpw: reset Windows local passwords from Linux live USB. Boot Ubuntu live, mount Windows partition, sudo apt install chntpw, cd /mnt/Windows/System32/config, chntpw SAM. Interactive menu: clear password, promote to admin, or blank. Works on all Windows up to Windows 11. Limitations: Microsoft account (online) passwords can't be reset — you need to reset the Microsoft password online. Utility: Hiren's Boot CD includes chntpw GUI.

Linux Password Reset

Single user mode: boot GRUB, press 'e' on kernel line, add init=/bin/bash or rd.break at end, Ctrl+X to boot, mount -o remount,rw /, passwd, new password. Live USB: boot Ubuntu, chroot into system: mount /dev/sda1 /mnt, mount --bind /dev /mnt/dev, chroot /mnt, passwd username. LUKS encryption: you need the passphrase — no bypass. Emergency: if LUKS header is corrupt, backup header (if you made one).

macOS & Mobile Recovery

macOS: boot into Recovery Mode (Cmd+R), open Terminal, resetpassword command resets login password. Apple ID: if FileVault is enabled, you need recovery key or Apple ID. iOS: recovery mode (Volume Up, Volume Down, Power button combo) — connect to computer, restore via Finder/iTunes — erases data. Android: Google account recovery (after multiple failed attempts). Samsung: Find My Mobile — unlock remotely. ADB: if USB debugging was enabled, adb shell to bypass lock.
