💽 Ventoy Advanced Guide

Multi-boot USB mastery — run dozens of ISOs from a single drive without reformatting.

What is Ventoy?

Ventoy is an open-source tool that creates a bootable USB drive by simply copying ISO/WIM/IMG/VHD(x)/EFI files onto it. Unlike traditional tools (Rufus, balenaEtcher) that reformat the entire drive every time you change an ISO, Ventoy installs itself once and then treats the data partition as a normal storage device. Drop an ISO in, boot it. Delete an ISO, drop another — no reformatting needed.

Ventoy was first released in 2020 and has rapidly become the gold standard for multi-boot USB drives. It supports Legacy BIOS, UEFI, Secure Boot, and works with more than 1,100+ verified ISOs across Windows, Linux, and BSD.

Installation

Download Ventoy2Disk from the official site (ventoy.net). Extract the archive and run the appropriate installer for your OS.

Windows

Run Ventoy2Disk.exe. Select the target USB drive, choose partition style (MBR or GPT), and click Install. This destroys all data on the selected drive.

Linux

Extract the tarball, then run the installer script:

tar -xzf ventoy-x.y.z-linux.tar.gz
cd ventoy-x.y.z
sudo bash Ventoy2Disk.sh -i /dev/sdX   # -i = install, /dev/sdX = your USB device

Replace /dev/sdX with your actual USB device (e.g., /dev/sdb). Use lsblk or fdisk -l to identify it. Double-check — selecting the wrong device will wipe your data.

Ventoy GUI

The Ventoy2Disk GUI provides all essential options:

  • Install / Update: Fresh install or upgrade Ventoy without touching the data partition.
  • Secure Boot Support: Enable Secure Boot enrollment (required for many UEFI systems).
  • Partition Style: Choose MBR (Legacy BIOS + UEFI) or GPT (UEFI-only, required for Windows 11).

On Linux, the same options are available via flags: -s for Secure Boot, -g for GPT.

Plugins (ventoy.json)

Ventoy's true power lies in its plugin system. Create a file named ventoy.json in the ventoy/ directory on the Ventoy partition (the small EFI partition, not the large data partition). Here are the most useful plugins:

🎨 Theme Plugin

Apply a custom GRUB theme to Ventoy's boot menu. Download a theme (e.g., Vimix or any Ventoy-compatible theme) and place it in the ventoy/theme/ directory:

{
  "theme": {
    "file": "/ventoy/theme/vimix/theme.txt",
    "gfxmode": "1920x1080"
  }
}

⚙️ Auto Install (Unattended)

Skip interactive installers by providing answer files:

  • Windows: Place an autounattend.xml in the ISO root or in ventoy/.
  • Ubuntu: Use a preseed.cfg with the auto_install plugin.
  • Fedora: Use a kickstart.ks file.
{
  "auto_install": {
    "ubuntu": "/ventoy/autoinstall/preseed.cfg",
    "windows": "/ventoy/autoinstall/autounattend.xml"
  }
}

💾 Persistence

Create persistent storage for live Linux sessions (Ubuntu, Mint, Kali). This keeps files, settings, and installed packages across reboots:

# Create a 4GB persistence image
dd if=/dev/zero of=persistence.img bs=1M count=4096
mkfs.ext4 persistence.img

Place persistence.img in the ventoy/ directory and configure the plugin:

{
  "persistence": [
    {
      "image": "/ventoy/persistence/ubuntu.img",
      "iso": "/ubuntu-24.04-desktop.iso"
    }
  ]
}

🔧 Conf Replace

Replace the GRUB boot configuration for a specific ISO — useful for adding kernel parameters or fixing boot issues:

{
  "conf_replace": [
    {
      "iso": "/ubuntu-24.04-desktop.iso",
      "file": "/ventoy/grub/ubuntu-custom.cfg"
    }
  ]
}

💉 Injection

Inject drivers, firmware, or scripts into an ISO at boot time without modifying the original ISO file:

{
  "injection": [
    {
      "iso": "/windows-11.iso",
      "file": "/ventoy/inject/drivers/nvme.ko"
    }
  ]
}

📛 Menu Alias

Rename ISOs in the boot menu without renaming the files themselves:

{
  "menu_alias": [
    {
      "iso": "/ubuntu-24.04-desktop.iso",
      "alias": "Ubuntu 24.04 LTS Desktop"
    },
    {
      "iso": "/kali-linux-2024.1.iso",
      "alias": "Kali Linux 2024.1"
    }
  ]
}

VentoyPlugson

VentoyPlugson is a GUI configuration tool for all Ventoy plugins. It runs a local web server and provides a point-and-click interface to edit ventoy.json:

# On Linux:
sudo bash VentoyPlugson.sh /dev/sdX

# On Windows:
VentoyPlugson.exe

Open the URL shown in the terminal (http://127.0.0.1:XXXX) to access the web UI. Every plugin option — theme, persistence, auto-install, injection, aliases — is available through the dashboard.

Ventoy on Windows

The Windows version (Ventoy2Disk.exe) provides all the same features via the GUI. Updating Ventoy on Windows is as simple as running the installer again — it upgrades the EFI partition without touching your ISO collection. Secure Boot enrollment is handled through a one-time MOK enrollment process.

Security — ISO Verification

Ventoy can verify ISO checksums directly from the boot menu. Place a checksum file alongside your ISO:

  • myiso.iso.md5 — MD5 checksum
  • myiso.iso.whirlpool — Whirlpool checksum
  • myiso.iso.sha256 — SHA-256 checksum

When you select an ISO that has a matching checksum file, Ventoy will verify the hash before booting. This prevents corruption or tampered ISOs from booting.

Ventoy vs Rufus vs balenaEtcher

FeatureVentoyRufusbalenaEtcher
Multi-ISO✅ Native — just copy files❌ One ISO per drive❌ One ISO per drive
Persistence✅ Plugin-based✅ Built-in (Linux only)❌ Not supported
Windows To Go⚠️ Limited✅ Full support❌ Not supported
Secure Boot✅ MOK enrollment✅ Native support⚠️ Varies by ISO
Legacy BIOS✅ Full support✅ Full support✅ Full support
Reformat per ISO❌ Never✅ Required each time✅ Required each time
Plugin system✅ Themes, auto-install, injection❌ No plugins❌ No plugins
Ease of use⭐⭐⭐⭐⭐ (once installed)⭐⭐⭐⭐⭐⭐⭐⭐⭐

Ventoy is the undisputed king of multi-ISO setups. Use Rufus when you need fine-grained control over a single ISO (partition scheme, cluster size, Windows To Go). Use balenaEtcher when you want the simplest possible one-click flash for a single ISO — but understand you'll reformat every time.

Pro Tip: Keep both Ventoy and Rufus handy. Use Ventoy for day-to-day multi-boot. Keep Rufus for Windows To Go installations or when you need non-standard partition layouts.

Keeping Ventoy Updated

Updating Ventoy is the same process as installing — download the latest release and run the installer, selecting Update instead of Install. Your data partition (all your ISOs) will not be touched. Only the EFI/Ventoy boot partition is rewritten. If you ever need a truly clean slate, select Install again; this will wipe the entire drive, so back up your ISOs first.

Getting Started: Download Ventoy from ventoy.net, install it to a USB drive, and start copying ISOs into the data partition. That's it — you now have a multi-boot USB drive.
