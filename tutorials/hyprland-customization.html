Hyprland Desktop Customization

Deep dive into Hyprland — configuring animations, blur, windows rules, and eye candy.

Hyprland Basics

Hyprland is a dynamic tiling Wayland compositor with smooth animations, rounded corners, and blur effects. Written in C++, it's performance-focused and highly configurable.

Config file: ~/.config/hypr/hyprland.conf. Hyprctl controls the compositor at runtime: hyprctl monitors, hyprctl clients, hyprctl workspaces.

Hyprland uses a declarative config format. Changes take effect immediately with hyprctl reload.

Animations & Eye Candy

Hyprland's animation system is one of its standout features. Configure global and per-window animations:

animations {
    enabled = yes
    bezier = wind, 0.05, 0.9, 0.1, 1.05
    bezier = linear, 0.0, 0.0, 1.0, 1.0

    animation = windows, 1, 7, wind, slide
    animation = windowsOut, 1, 7, wind, slide
    animation = fade, 1, 7, wind
    animation = workspaces, 1, 6, wind, slidevert
    animation = border, 1, 10, default
    animation = borderangle, 1, 8, wind
}

Bezier curves control easing. Speeds (1-10) control duration. Styles: slide, slidevert, fade, popin.

Blur, Shadows & Corner Settings

Blur and transparency effects require the blur decoration layer:

decoration {
    # Blur
    blur {
        enabled = true
        size = 8
        passes = 3
        ignore_opacity = false
        new_optimizations = true
        noise = 0.01
        contrast = 1.0
        brightness = 1.0
        vibrancy = 0.2
        vibrancy_darkness = 0.0
    }

    # Drop shadows
    drop_shadow = yes
    shadow_range = 30
    shadow_render_power = 3
    col.shadow = rgba(1a1a2eee)

    # Rounded corners
    rounding = 12
    active_opacity = 0.98
    inactive_opacity = 0.9
    fullscreen_opacity = 1.0
}

Higher passes = more quality but lower performance. On integrated GPUs, keep passes ≤ 2 and size ≤ 6.

Window Rules & Workspace Management

Window rules automatically apply settings to matching windows using class names, titles, or initial titles:

windowrulev2 {
    # Float certain apps
    float = class:(pavucontrol|blueman-manager|xdg-desktop-portal)
    # Tile Firefox, don't float
    tile = class:(firefox)
    # Fullscreen games
    fullscreen = class:(Steam|steam_app*)
    # Pin apps to workspace
    workspace = 1 silent, class:(firefox)
    workspace = 2 silent, class:(Alacritty|kitty)
    workspace = 5 silent, class:(code|Code)
    # Opacity rules
    opacity = 0.9 override, class:(firefox)
    # No border for some windows
    noborder = class:(xdg-desktop-portal*)
}

Use hyprctl clients to find class names. Window rules use PCRE regex.

Monitors, Input & Bindings

Multi-monitor setup and input configuration:

monitor = DP-1, 2560x1440@165, 0x0, 1.0
monitor = HDMI-A-1, 1920x1080@60, 2560x0, 1.0
monitor = ,preferred, auto, 1.0  # Laptop display

input {
    kb_layout = us
    kb_variant =
    kb_options = ctrl:nocaps
    follow_mouse = 1
    sensitivity = 0.5
    accel_profile = flat
    touchpad {
        natural_scroll = yes
        tap-to-click = yes
        drag_lock = yes
    }
}

bind = SUPER, Q, killactive
bind = SUPER, Return, exec, kitty
bind = SUPER, Space, exec, rofi -show drun
bind = SUPER, W, togglefloating
bind = SUPER ALT, L, exec, loginctl lock-session
