Figma UI Design Basics

Design interfaces, prototypes, and design systems with Figma — the industry-standard collaborative design tool.

Getting Started with Figma

Figma is a browser-based collaborative interface design tool. Create a free account at figma.com and start designing immediately without any installation.

Figma uses vector-based design with a pen tool similar to Illustrator. Key workspace areas: canvas, layers panel, properties panel, and toolbar.

// Figma plugin example
figma.showUI(__html__)
figma.ui.onmessage = msg => {
  if (msg.type === 'create-shapes') {
    const nodes = []
    for (let i = 0; i < msg.count; i++) {
      const rect = figma.createRectangle()
      rect.x = i * 100
      rect.fills = [{type: 'SOLID', color: {r: 0.2, g: 0.4, b: 0.8}}]
      nodes.push(rect)
    }
    figma.currentPage.selection = nodes
    figma.viewport.scrollAndZoomIntoView(nodes)
  }
  figma.closePlugin()
}

Components & Variants

Components are reusable design elements. Changes to the master component propagate to all instances. Variants group related component states.

Create components with Ctrl+Alt+K (Cmd+Opt+K). Properties: boolean (show/hide), text swap, instance swap, and variant.

Use component properties to make flexible, reusable UI elements without creating dozens of separate components. Auto layout makes responsive design effortless.

Auto Layout & Constraints

Auto Layout creates responsive containers that adapt to content. Think of it as CSS flexbox inside Figma. Set padding, spacing between items, and alignment.

Constraints define how elements behave when the parent resizes: left/right/top/bottom/center/scale. Combine with auto layout for truly responsive designs.

Use hugs content vs fixed width to control sizing behavior. Nest auto layout frames to build complex, responsive cards, lists, and navigation bars.

Prototyping & Interactions

Figma prototypes connect frames with interactions. Add connections between frames to create clickable prototypes that simulate the real app experience.

Interaction types: navigate to, open overlay, swap with, scroll to, change to. Triggers: on click, on drag, while hovering, while pressing, key/gamepad.

Smart animate detects matching layers between frames and animates the transition. Use this for micro-interactions, loading states, and screen transitions.

Design Systems & Team Libraries

Design systems enforce consistency across products. Figma supports shared team libraries with components, styles, and variables.

Publish components as a team library. Other team members can use them and receive updates automatically. Style variables include colors, text, effects, and grid.

Figma variables support modes (light/dark theme), and can be bound to component properties. Design tokens export to CSS, iOS, and Android code formats.
