Set up Jenkins for continuous integration and delivery.

Jenkins Overview

Jenkins is the leading open-source automation server for CI/CD. Supports thousands of plugins for building, deploying, and automating any project.

Installation

# Docker
sudo docker run -p 8080:8080 -p 50000:50000 -v jenkins_home:/var/jenkins_home jenkins/jenkins:lts

# Access: http://localhost:8080
# Initial password:
sudo cat /var/lib/jenkins/secrets/initialAdminPassword

Declarative Pipeline

pipeline {
    agent any
    stages {
        stage("Build") { steps { sh "npm ci && npm run build" } }
        stage("Test") { steps { sh "npm run test" } }
        stage("Deploy") { when { branch "main" }; steps { sh "./deploy.sh" } }
    }
}

Distributed Builds

Scale with agents: permanent agents (VMs), cloud agents (AWS EC2, Kubernetes), Docker agents.
