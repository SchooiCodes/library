F-Droid & Aurora Store

Open-source Android app stores that give you freedom, privacy, and control — no Google Play Services required.

Overview

F-Droid is a curated catalog of free and open-source Android applications. Every app in the main F-Droid repository is built from source by the F-Droid build server, ensuring no proprietary blobs, trackers, or malware. Aurora Store is a privacy-friendly Google Play Store client that lets you download apps from Google Play without a Google account or Google Play Services.

Together, they cover both open-source apps (F-Droid) and mainstream apps (Aurora Store) — giving you complete app access without depending on Google.

Safety Warning. F-Droid's main repository is safe by design — all apps are built from source. However, third-party repositories (repos) may host apps that are not verified. Only add repos from trusted sources like IzzyOnDroid, Guardian Project, or the developer's official F-Droid repo.

F-Droid Basics

Installation

# Download the F-Droid APK from the official website:
https://f-droid.org/FDroid.apk

# Or get it from the official F-Droid repository on GitHub.
# Enable "Install from unknown sources" for your browser/file manager.
# Install the APK.

F-Droid does not need Google Play Services. It works on any Android device, including those without Google apps (e.g., Huawei, Amazon Fire tablets with Google-free ROMs).

Basic Usage

  1. Open F-Droid. The app list will populate after it syncs the repository metadata.
  2. Use the search bar to find apps or browse categories.
  3. Tap an app to view its description, source code link, version history, and permissions.
  4. Tap Install to download and install. F-Droid handles the download and installation.
  5. Updates appear in the Updates tab. You can update individual apps or all at once.

What You Will Find on F-Droid

  • NewPipe — YouTube frontend without ads or tracking.
  • K-9 Mail — Full-featured email client (now being merged with Thunderbird).
  • Organic Maps — Offline maps with no ads or tracking.
  • Simple Mobile Tools — Suite of privacy-respecting apps (gallery, calendar, contacts, etc.).
  • Syncthing — Peer-to-peer file synchronization.
  • Bitwarden — Password manager (F-Droid version has no Google Play dependency).
  • KeePassDX — KeePass-compatible password manager.
  • OsmAnd~ — Offline navigation with OpenStreetMap data.
  • Termux — Terminal emulator with Linux package environment.
  • Fennec F-Droid — Firefox build with Mozilla telemetry removed.

Adding Repositories

F-Droid's power comes from its repository system. The default repo is the main F-Droid repository. You can add more:

IzzyOnDroid

The largest third-party F-Droid repo. It hosts apps that are open-source but do not meet F-Droid's strict build requirements (e.g., apps that use proprietary libraries or are hard to build from source).

# Add IzzyOnDroid repo:
# Open F-Droid > Settings > Repositories > Add repository
# URL: https://apt.izzysoft.de/fdroid/repo
# Fingerprint: (available on the IzzyOnDroid website)

# Or scan the QR code from https://apt.izzysoft.de

Guardian Project

Official repository of the Guardian Project, which develops security and privacy apps.

# Add Guardian Project repo:
# URL: https://guardianproject.info/fdroid/repo
# Fingerprint: (available on the Guardian Project website)

Developer Repos

Many open-source developers host their own F-Droid repo so you get updates faster than the main F-Droid catalog. For example:

# NewPipe repo:
https://archive.newpipe.net/fdroid/repo/

# MicroG repo:
https://microg.org/fdroid/repo

# Bitwarden (direct F-Droid):
https://mobileapp.bitwarden.com/fdroid/repo
Repository Fingerprints. When adding a repository, always verify its fingerprint matches the one shown on the official website. This prevents man-in-the-middle attacks where someone could serve you malicious apps.

Aurora Store

Aurora Store is a fully open-source Google Play Store client. It lets you browse, download, and update apps from Google Play without a Google account or Google Play Services.

Features

  • Anonymous Login — Aurora Store provides a guest account that lets you download free apps without signing in. This uses shared anonymous accounts maintained by the Aurora project.
  • Account Spoofing — You can spoof a device ID to access region-restricted apps or apps that require a specific device.
  • App Blacklist — Hide certain apps from appearing in your updates list.
  • Export APKs — Save downloaded APKs to your device for sharing or backup.
  • No Google Play Services — Works on de-Googled ROMs, microG devices, and Huawei phones.
  • Material You Design — Modern UI that matches Android 12+ design language.

Installation

# Download Aurora Store from F-Droid:
# Search for "Aurora Store" in F-Droid and install it.

# Or get it from the official GitLab:
# https://gitlab.com/AuroraOSS/AuroraStore/-/releases

Privacy Benefits

Using F-Droid and Aurora Store instead of the Play Store provides several privacy advantages:

  • No Google Play Services required. Google Play Services is a proprietary blob that has extensive access to your device (location, device identifiers, app usage). Eliminating it is a major privacy win.
  • F-Droid apps are built from source. Every app in the default repo is compiled by F-Droid's build servers from publicly available source code. You can verify the build yourself.
  • No tracking. F-Droid does not collect analytics. Aurora Store's anonymous mode does not tie downloads to your identity.
  • Permission warnings. F-Droid shows you exactly what permissions each app requests before you install.
  • Anti-features. F-Droid labels apps with "anti-features" like NonFreeNet (depends on non-free network services), Tracking, or Ads so you can make informed choices.
  • No forced updates. You control when and what to update.
microG Alternative. If you need Google Play Services for certain apps but want privacy, consider microG — it is an open-source reimplementation that does not send your data to Google. Aurora Store + microG is a powerful combo for de-Googled devices.

Unattended Updates

Both F-Droid and Aurora Store can update apps automatically in the background (with the right setup):

F-Droid Unattended Updates

F-Droid's "Unattended Updates" feature uses the system's session installer (Android 12+) or the Privileged Extension (requires root or installing it as a system app via recovery).

# Option 1: Android 12+ Session Installer
# Go to F-Droid Settings > Unattended updates
# Select "Session installer (no root)"
# This works on Android 12+ without any special setup.

# Option 2: Privileged Extension (root/recovery)
# Install the F-Droid Privileged Extension from the repo.
# It must be flashed via recovery or installed as a system app.
# Once installed, F-Droid can update apps in the background silently.

Aurora Store Auto-Updates

Aurora Store can auto-update apps via its built-in services (requires notification access):

# Aurora Store > Settings > Updates
# Enable "Auto-update"
# Select "Use foreground service" for better reliability.

# Note: Aurora auto-updates work best when using a personal Google account.
# Anonymous accounts have rate limits.

Troubleshooting

F-Droid "Pending download" Stuck

Clear the F-Droid app cache and data, then re-sync repositories. Also ensure you have a stable internet connection.

Aurora Store "Login failed"

  • Anonymous accounts get rate-limited after ~20 downloads per day. Wait 24 hours or use a different anonymous account.
  • If using a personal account, ensure you have enabled "Less secure app access" or use an App Password if you have 2FA enabled.
  • Try clearing Aurora Store's data and starting fresh.

Repository Sync Fails

  • Check your internet connection.
  • The repository may be temporarily down. Try again later.
  • If you manually entered a repo URL, double-check the URL and fingerprint.
  • Some repos require a specific F-Droid client version. Update F-Droid.

F-Droid Privileged Extension Not Working

  • It must be installed as a system app. If you flashed it via recovery, ensure your recovery supports the installation.
  • On Android 12+, you can use the session installer instead — it works without root.

Tips & Best Practices

  • Use both F-Droid and Aurora Store. F-Droid for open-source apps, Aurora for everything else. This maximizes both privacy and app availability.
  • Verify repository fingerprints. When adding a third-party repo, visit the official website to confirm the fingerprint.
  • Disable Google Play Store. If you are fully de-Googling, disable or uninstall the Play Store and Play Services via ADB or a debloater.
  • Prefer F-Droid builds over Aurora downloads. Apps from F-Droid are verified open-source builds, whereas Aurora downloads the same binary as the Play Store (which may contain proprietary components).
  • Use Aurora's Export APK feature. Download APKs on Wi-Fi and keep local backups — useful if you need to install the same app on multiple devices.
  • Check app anti-features. Before installing an app from F-Droid, scroll down to see if it has any anti-feature labels.
  • Keep old app versions. F-Droid keeps previous versions in its repository. If an update breaks something, you can downgrade easily.
  • Join the community. F-Droid has a forum and Matrix chat for support. Aurora Store has a Telegram group.
