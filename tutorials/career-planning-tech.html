Career Planning in Tech

Plan your tech career — learning paths, resumes, networking, and interviewing.

Table of Contents (2 sections)

  • Learning Paths & Skills
  • Resumes & Interviewing

Learning Paths & Skills

Tech roles: Frontend (React, Vue, CSS), Backend (Node, Python, Go, Java, Rust, C#), Full-Stack, DevOps (AWS, Docker, K8s, CI/CD), Data Science (Python, ML, SQL), Mobile (Swift, Kotlin, React Native, Flutter), Security (pen testing, AppSec, GRC), Cloud (AWS, Azure, GCP — Solutions Architect). Learning resources: full courses (freeCodeCamp, The Odin Project), structured (Frontend Masters, Pluralsight, Udemy), bootcamps (intensive 12-24 weeks), degree (CS degree — comprehensive, 4 years). Certifications: AWS (SA, Developer, DevOps), Google Cloud, Azure, CompTIA (A+, Network+, Security+), Cisco (CCNA). Portfolio: GitHub projects, personal website, blog. Open source: contribute to projects — learn code review, collaboration, git workflows. Internships: apply early (junior year), tech-specific internships, networking. Freelance: Upwork, Fiverr — build experience. University vs self-teach: degree helps (filter for HR), self-teach + portfolio can work. Job search: LinkedIn, Indeed, Glassdoor, company career pages, networking referrals. Continuous learning: tech changes fast — set aside time weekly for learning.

Resumes & Interviewing

Resume: one page (junior-max 2), quantifiable achievements ("improved performance 30%"), keywords from job description, action verbs (developed, implemented, optimized, designed). Sections: contact, summary (optional), skills, experience, education, projects, certifications. ATS (Applicant Tracking System): machine-readable format (PDF), standard fonts, no columns/graphics, include keywords. Cover letter: specific to company, why you, what you bring. Behavioral interviews: STAR method (Situation, Task, Action, Result) — examples of leadership, conflict, failure, success. Technical interviews: LeetCode (medium/hard), system design (whiteboard), take-home projects. System design interview: requirements > estimation > data model > high-level design > deep dive > trade-offs. Pair programming interview: share screen, solve problem, explain thinking. Negotiation: know market rate, total compensation (salary + bonus + equity). Offer evaluation: base salary, equity (RSUs vs options), benefits (health, 401k match, PTO), growth potential. First job: prioritize learning/growth over money. Career growth: mentor, network, find sponsors, change jobs for advancement every 2-3 years. Soft skills: communication, teamwork, empathy, adaptability. Self-promotion: blog, speak at meetups/conferences, share achievements. Remote work: async communication, self-discipline, home office setup.
