App Management

Orion Store

A powerful Android app manager for batch APK installation, app backup and restore, APK export, and app freezing — all in one tool.

Overview

Orion Store is a free, open-source Android application manager that goes far beyond what a typical app store offers. It focuses on batch operations: installing multiple APKs at once, backing up and restoring apps with their data, exporting installed apps as APK files, and freezing/unfreezing apps without root (via Shizuku) or with root.

It is particularly useful for developers, ROM flashers, people who reset their phones frequently, and anyone who manages multiple devices. Orion Store does not require Google Play Services and works fully offline for management tasks. For app discovery, it also has a built-in catalog of open-source apps (powered by the F-Droid ecosystem).

Safety Warning. Always download Orion Store from the official GitHub repository (https://github.com/anddea/orion-store) or from F-Droid. Third-party builds may contain malware. The app requires no special permissions for basic features, but features like app freezing require Shizuku or root.

Installation

# From F-Droid:
# Search for "Orion Store" in F-Droid and install it.

# From GitHub Releases:
# Download the latest APK from:
# https://github.com/anddea/orion-store/releases
# Enable "Install from unknown sources" and install.

# From Aurora Store:
# Search for "Orion Store" and install via anonymous session.

After installation, you may want to set up Shizuku for advanced features like app freezing and silent installation (see the Shizuku & Hail tutorial for setup instructions).

Batch APK Installation

One of Orion Store's standout features is the ability to install multiple APK files at once — both regular APKs and split APKs (APKS/XAPK).

How to Batch Install

  1. Open Orion Store and tap the "Install APKs" button on the home screen.
  2. Navigate to the folder containing your APK files.
  3. Select multiple APK files (tap each one, or use Select All).
  4. Tap the install button in the bottom bar.
  5. Orion Store will install them sequentially. Each app's installation progress is shown.

Supported Formats

  • .apk — Standard Android package files.
  • .apks — Split APK bundles (commonly used by Aurora Store exports).
  • .xapk — APKPure's split APK format (Orion handles these transparently).
  • .apkm — APK Mirror's bundle format.

Silent Installation (with Shizuku)

If you have Shizuku running, Orion Store can install APKs silently in the background without showing the system installer dialog for each app:

# Enable Shizuku integration:
# Orion Store > Settings > Installation > Install method > Shizuku

# Once enabled, batch installations happen silently.
# You can continue using your phone while apps install.

App Backup & Restore

Orion Store can back up both APK files and app data (on rooted devices or using Shizuku). This is essential before factory resets or when switching ROMs.

Backing Up Apps

  1. Go to the Installed Apps tab.
  2. Tap the checkbox icon in the top bar to enter selection mode.
  3. Select the apps you want to back up.
  4. Tap the backup icon (cloud with arrow up) in the bottom bar.
  5. Choose whether to back up APK only, or APK + data (requires Shizuku/root).
  6. Backups are saved to Internal Storage/OrionStore/backups/ by default.

Restoring Apps

  1. Tap the "Restore" button on the home screen.
  2. Browse to the backup folder or select individual backup files.
  3. Select the backups to restore.
  4. Tap Restore. Data will be restored if it was included in the backup.
Data Backup Note. App data backup requires Shizuku or root because Android's backup system restricts access to app-private data. Without these, Orion can still back up the APK itself, which is useful for reinstalling the same version later.

Export APKs

You can extract APK files from installed apps — useful for sharing apps without internet, creating local archives, or archiving specific versions before an update.

Exporting a Single APK

  1. Go to Installed Apps and tap an app.
  2. Tap the three-dot menu and select "Export APK".
  3. The APK will be saved to Internal Storage/OrionStore/exported_apks/.

Exporting Multiple APKs

  1. Enter selection mode in Installed Apps.
  2. Select multiple apps.
  3. Tap the export icon (box with arrow up).
  4. All selected APKs will be saved to the export folder.

Exported APKs are named using the format: PackageName_versionCode_versionName.apk (e.g., com.example.app_123_1.2.3.apk).

App Freezer

Orion Store includes an app freezer that can disable apps without uninstalling them. Frozen apps are hidden from the app drawer and cannot run. This is useful for disabling bloatware that cannot be uninstalled.

How Freezing Works

  • With Shizuku — Freeze/unfreeze apps using the pm disable/pm enable shell commands. Works on non-rooted devices.
  • With Root — Same commands, executed directly as root.
  • Device Owner — Some devices support freezing via Device Owner mode (no root, set up via ADB).

Freezing an App

  1. Go to Installed Apps and long-press an app.
  2. Tap Freeze from the context menu.
  3. The app will be disabled. It will disappear from the app drawer and cannot send notifications or run in the background.

Unfreezing

To restore a frozen app, go to the Frozen Apps tab (or filter Installed Apps by status), select the app, and tap Unfreeze.

Freezing System Apps. Be careful when freezing system apps. Freezing critical components like com.android.phone or com.android.systemui can break your device. Only freeze apps you are sure are safe to disable (e.g., Facebook, Chrome, YouTube — if you use alternatives).

UI Overview

Orion Store's interface is divided into several tabs:

  • Home — Quick actions: Install APKs, Restore, Recent apps, suggested open-source apps.
  • Installed Apps — List of all installed apps (user and system). Sort by name, size, or install date. Filter by user/system/frozen.
  • Updates — Shows apps that have updates available (checks F-Droid repos or Aurora Store if integrated).
  • Queue — Active and pending batch operations.
  • Settings — Backup location, install method (system/Shizuku), theme (Material You / light / dark), language, export settings.

Use Cases

Migrating to a New Device

  1. On your old phone: Use Orion Store to export all user apps as APKs.
  2. Transfer the OrionStore/exported_apks/ folder to your new phone.
  3. On your new phone: Use Orion Store's batch install to install all apps at once.
  4. If you have Shizuku, the installation will be silent and fast.

Clean ROM Reinstall

  1. Before wiping your device, back up all user apps (APK + data) with Orion Store.
  2. Copy the backup folder to a PC or SD card.
  3. Flash the new ROM.
  4. Install Orion Store.
  5. Restore all apps from backup.

Freezing Bloatware

  1. Open Orion Store and go to Installed Apps.
  2. Filter by "System apps" to see pre-installed bloatware.
  3. Select unwanted apps (Facebook, LinkedIn, Xbox Game Pass, etc.).
  4. Freeze them all at once.
  5. They will no longer appear in your app drawer or consume resources.

Archiving App Versions

When an app updates and the new version has unwanted changes (UI redesign, removed features, new permissions), use Orion Store to export the old version's APK before updating. You can then downgrade by installing the old APK later.

Troubleshooting

Batch Install Fails for Some APKs

  • The APK may be corrupted or incomplete. Check its size.
  • The APK may require a newer Android API level. Check the manifest.
  • Split APKs need all parts in the same folder. Ensure you have all .apks parts or the complete .xapk file.

Shizuku Not Detected

  • Ensure Shizuku is running. Open the Shizuku app and check its status.
  • Restart the Shizuku service (via ADB or root).
  • Grant Orion Store permission to use Shizuku in the Shizuku app's "Granted apps" list.

Freeze Button Grayed Out

  • Freezing requires Shizuku or root. Set up one of these first.
  • Some critical system apps cannot be frozen by Android's security policies.
  • Device Owner mode may work but requires ADB setup.

Backup Fails for Some Apps

  • App data backup requires root or Shizuku. Without these, only APK backup is possible.
  • Some apps use Splite (split APKs) that Orion cannot recombine for backup. Export them instead.
  • Protected system apps may refuse backup at the OS level.

Tips & Best Practices

  • Use Shizuku for the best experience. It enables silent install and data backup without root. Setting it up once takes 2 minutes with ADB.
  • Keep your APK archives organized. Orion Store's default export naming is good, but consider creating subfolders by app category.
  • Export before updating. If an app update breaks something, having the old APK saved locally lets you downgrade instantly.
  • Batch install in waves. Installing 50+ apps at once can be slow. Split into groups of 10-15 for faster feedback.
  • Check for Orion Store updates. The app is actively developed. Update via F-Droid or GitHub to get new features and bug fixes.
  • Use with Aurora Store. Export APKs from Aurora Store's download list, then install them with Orion Store's batch installer.
  • Do not freeze system launcher. If you freeze your device's launcher, you may lose access to the app drawer. Always have a backup launcher installed.
  • Verify APKs from unknown sources. If you download APKs from the web, scan them with Hypatia or another malware scanner before installing via Orion.
