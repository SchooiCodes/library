📄 How to BSOD your computer/laptop in 2023

Warning: This will force‑close all programs without saving. Save any work before proceeding.

Method 1 – Killing wininit

  1. Press the Windows logo key.
  2. Search for taskmgr and run as Administrator.
  3. In Task Manager, go to the Details tab.
  4. Find csrss.exe or wininit.exe.
  5. Right‑click and choose End Task, then confirm.

Method 2 – Editing the Registry

  1. Press the Windows logo key and search for regedit, run it.
  2. Determine if your keyboard is USB or PS/2:
    • USB – rectangular connector.
    • PS/2 – round connector.
  3. Navigate to the appropriate registry key:
    • USB: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\kbdhid\Parameters
    • PS/2: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\i8042prt\Parameters
  4. Create a new DWORD (32‑bit) value named CrashOnCtrlScroll and set it to 1.
  5. Reboot, then hold the right Ctrl key and press Scroll Lock twice.
