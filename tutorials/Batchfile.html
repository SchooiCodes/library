📄 Creating a Basic Batch File

Introduction

A batch file is a script for Windows command‑line interpreter, used to automate tasks.

Syntax

@echo off
echo Hello, World!
pause

@echo off hides command echo, echo prints, pause waits for a key.

Common Commands

  • echo – display text
  • pause – wait for a key press
  • cls – clear screen
  • rem – comment line
  • set – set environment variable
  • if – conditional execution
  • for – loop over files
  • goto – jump to label
  • call – call another batch file

Examples

Display Message

@echo off
echo This is a batch file example.
pause

Create Directory

@echo off
mkdir MyNewFolder
echo Folder created.
pause

Delete Temporary Files

@echo off
del /Q C:\Temp\*.tmp
echo Temporary files deleted.
pause
