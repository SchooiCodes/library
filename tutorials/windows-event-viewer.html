Windows Event Viewer & Troubleshooting

Master Event Viewer for system diagnostics and troubleshooting.

Table of Contents (5 sections)

Event Viewer Basics

Event Viewer (eventvwr.msc) logs system events in categories: Application, Security, System, Setup, Forwarded Events. Event levels: Info, Warning, Error, Critical.

Custom Views & Filtering

Filter by Event ID, Level, Source. Key IDs: 4624 (logon), 4625 (failed logon), 4688 (process creation), 41 (unclean shutdown), 1001 (crash).

PowerShell Queries

# Recent errors
Get-WinEvent -LogName System -MaxEvents 50 | Where-Object { $_.LevelDisplayName -match "Error|Critical" }

# Failed logins
Get-WinEvent -LogName Security -FilterXPath "*[System[EventID=4625]]"

Common Event IDs

41 — Unexpected shutdown | 1074 — Shutdown/restart | 4624 — Successful logon | 4625 — Failed logon | 4688 — Process creation | 6008 — Unexpected shutdown

Diagnostic Scenarios

Use Event Viewer to diagnose BSODs (critical level), app crashes (1001), driver failures, service hangs, and login failures.
