🛡️ Browser Privacy Guide

How to lock down your browser for maximum privacy — ad blocking, tracker blocking, secure DNS, and essential settings.

🛡️ Why Browser Privacy Matters

Every website you visit can track you using cookies, fingerprinting, IP logging, and third-party scripts. Ad networks build profiles of your browsing habits, ISPs log your DNS queries, and data brokers sell your information. A privacy-hardened browser stops most of this cold.

🚫 uBlock Origin — The Gold Standard

uBlock Origin is the best ad blocker for any browser. It's lightweight, efficient, and blocks ads, trackers, and malicious domains.

Installation

  • Firefox: Add to Firefox
  • Chrome/Edge/Brave: Add to Chrome

Recommended Settings

  1. Open uBlock Origin dashboard (click the icon → gear icon)
  2. Under "Filter lists", enable:
    • Built-in: uBlock filters, EasyList, EasyPrivacy
    • Privacy: AdGuard Tracking Protection, Fanboy's Annoyance List
    • Malware: Malware Domain Blocklist, Curated Malicious Sites
  3. Under "Settings", check:
    • ✓ Block large media elements
    • ✓ Disable pre-fetching
    • ✓ Block CNAME cloaking
  4. Click "Apply changes"

⚙️ Browser Privacy Settings

Firefox

  1. Go to Settings → Privacy & Security
  2. Enhanced Tracking Protection: Strict
  3. Check "HTTPS-Only Mode" → Enable in all windows
  4. DNS over HTTPS: Increased Protection (use NextDNS or Cloudflare)
  5. Uncheck "Allow Firefox to send technical data"
  6. Uncheck "Allow Firefox to install studies"

Brave

  1. Go to Settings → Shields
  2. Set "Trackers & ads blocking" to Aggressive
  3. Enable "Block fingerprints"
  4. Enable "Block phishing" and "Block scripts"
  5. Go to Settings → Security and Privacy → Use secure DNS → Cloudflare (1.1.1.1) or NextDNS

Chrome/Edge

  1. Go to Settings → Privacy and security
  2. Enable "Use secure DNS" → Choose Cloudflare (1.1.1.1) or NextDNS
  3. Send "Do Not Track" requests
  4. Disable "Allow sign-in to Chrome"
  5. Install uBlock Origin (Chrome's built-in blocker is weak)

🌐 DNS over HTTPS (DoH)

DNS over HTTPS encrypts your DNS queries so your ISP can't see what websites you visit.

  • NextDNS: Setup Guide → — Custom blocklists, analytics, multi-device
  • Cloudflare: https://1.1.1.1/dns-query — Fast, privacy-focused
  • Quad9: https://dns.quad9.net/dns-query — Blocks malicious domains

Most browsers have DoH built into settings. Enable it for system-wide protection.
