Variables & Types

JavaScript has three ways to declare variables. Use const by default, let when you need to reassign, and avoid var in modern code.

// Strings, Numbers, Booleans
const name = "Alice";
const age = 25;
const isStudent = true;

// Template literals (backticks)
const greeting = `Hello, my name is ${name} and I am ${age}.`;

// null vs undefined
let empty = null;    // Intentional absence
let notDefined;     // undefined (never assigned)

// Type checking
console.log(typeof name);  // "string"
console.log(typeof age);   // "number"

Functions

Functions are reusable blocks of code. Modern JavaScript prefers arrow functions, but classic function declarations are still widely used.

// Function declaration (hoisted)
function greet(name) {
 return `Hi, ${name}!`;
}

// Arrow function (preferred for callbacks)
const add = (a, b) => a + b;

// Arrow function with block body
const multiply = (a, b) => {
 const result = a * b;
 return result;
};

// Default parameters
const sayHello = (name = "World") => `Hello, ${name}!`;

// Rest parameters
const sum = (...numbers) => numbers.reduce((acc, n) => acc + n, 0);
console.log(sum(1, 2, 3, 4)); // 10
Arrow functions don't have their own this binding. They inherit this from the enclosing scope, which makes them ideal for event handlers and array methods.

DOM Manipulation

The Document Object Model (DOM) is JavaScript's interface to the HTML page. You can select, modify, create, and remove elements.

// Selecting elements
const header = document.getElementById("header");
const cards = document.querySelectorAll(".card");
const firstBtn = document.querySelector("button");

// Modifying content
header.textContent = "New Title";
header.innerHTML = "<span>Bold text</span>";

// Modifying styles
header.style.color = "#ff6b6b";
header.classList.add("active");
header.classList.remove("hidden");
header.classList.toggle("dark-mode");

// Creating elements
const newDiv = document.createElement("div");
newDiv.textContent = "I'm new!";
document.body.appendChild(newDiv);

// Removing elements
const old = document.querySelector(".old");
old.remove();

Events

Events let your page react to user actions — clicks, keystrokes, form submissions, scrolls, and more.

// Click event
button.addEventListener("click", (event) => {
 console.log("Button clicked!", event.target);
});

// Form submission
form.addEventListener("submit", (event) => {
 event.preventDefault(); // Stop page reload
 const data = new FormData(form);
 console.log(Object.fromEntries(data));
});

// Keyboard events
document.addEventListener("keydown", (e) => {
 if (e.key === "Escape") {
  closeModal();
 }
});

// Event delegation — listen on parent
list.addEventListener("click", (e) => {
 if (e.target.matches("li")) {
  console.log("List item clicked:", e.target.textContent);
 }
});

Arrays & Objects

Arrays hold ordered lists of data; objects hold key-value pairs. Modern JavaScript provides powerful methods to work with both.

// Array methods
const numbers = [1, 2, 3, 4, 5];

const doubled = numbers.map(n => n * 2);  // [2, 4, 6, 8, 10]
const evens = numbers.filter(n => n % 2 === 0); // [2, 4]
const sum = numbers.reduce((acc, n) => acc + n, 0); // 15
const found = numbers.find(n => n > 3);  // 4
const hasBig = numbers.some(n => n > 10); // false

// Objects
const user = {
 name: "Alice",
 age: 25,
 skills: ["JS", "Python"],
 greet() {
  return `Hi, I'm ${this.name}`;
 }
};

// Destructuring
const { name, age } = user;
const [first, second] = numbers;

// Spread operator
const cloned = { ...user, location: "NYC" };
const combined = [...numbers, 6, 7, 8];

Async/Await

Async functions let you write asynchronous code that looks synchronous. Use async to declare an async function, then await promises inside it.

// Fetch API — get data from an endpoint
async function getUsers() {
 try {
  const response = await fetch("https://api.example.com/users");
  if (!response.ok) throw new Error("Network error");
  const data = await response.json();
  return data;
 } catch (error) {
  console.error("Fetch failed:", error);
 }
}

// Multiple parallel requests
async function getDashboardData() {
 const [users, posts, comments] = await Promise.all([
  fetch("/api/users").then(r => r.json()),
  fetch("/api/posts").then(r => r.json()),
  fetch("/api/comments").then(r => r.json())
 ]);
 return { users, posts, comments };
}

// Async arrow function
const loadData = async () => {
 const data = await fetch("/api/data");
 return data.json();
};
