📄 RethinkDNS Setup Guide

Table of Contents (2 sections)

  • Installation
  • Advanced Configuration

RethinkDNS is a privacy‑focused DNS resolver that offers easy configuration across devices.

Installation

  1. Visit rethinkdns.com and download the client for your platform (Android, iOS, Linux, Windows).
  2. Install the application following the platform‑specific instructions.
  3. Open the app and enable the DNS service. The default configuration provides ad‑blocking and tracker protection.

Android

  1. Install the "RethinkDNS" app from Google Play.
  2. Open, select a server or use the default, and enable the VPN‑based DNS service.

iOS

  1. Install the app from the App Store.
  2. Configure a server and activate the DNS filter.

Linux (systemd)

  1. Run curl -s https://rethinkdns.com/install.sh | sudo bash (script adds a systemd service).
  2. Enable and start with sudo systemctl enable rethinkdns && sudo systemctl start rethinkdns.

Windows

  1. Download the Windows installer from the website.
  2. Run the installer, which sets up a background service.
  3. Open the app from the start menu and toggle the DNS filter on.

Advanced Configuration

  • Import custom blocklists via the Settings → Blocklists menu.
  • Enable DNS‑over‑HTTPS for encrypted query transport.
  • Use the built‑in analytics to monitor query volume and blocked domains.
