Statistical Modeling

Linear models, generalized linear models, mixed effects, and model selection for data analysis.

Linear Models & Assumptions

Linear regression models the conditional expectation of Y given X. Key assumptions: linearity, independence, homoscedasticity, normality of errors.

import statsmodels.api as sm
X = sm.add_constant(X)  # Intercept
model = sm.OLS(y, X).fit()
print(model.summary())  # R², p-values, F-statistic

Diagnostics: residual plots, Q-Q plots, Cook's distance for influential points, VIF for multicollinearity.

Generalized Linear Models

GLMs extend linear models to non-normal distributions via link functions. Common families: binomial (logistic), Poisson (log), gamma.

import statsmodels.formula.api as smf
# Logistic regression
data = pd.DataFrame({'y': [0,1,0,1,1], 'x': [1,2,3,4,5]})
model = smf.glm('y ~ x', data, family=sm.families.Binomial()).fit()
print(model.summary())
print(model.predict(pd.DataFrame({'x': [6]})))

Logistic regression for binary outcomes, Poisson for count data, and multinomial for categorical outcomes.

Mixed Effects Models

Mixed models account for both fixed effects (population-level) and random effects (group-level). Essential for hierarchical/grouped data.

import statsmodels.formula.api as smf
# Random intercept for subjects
model = smf.mixedlm('y ~ x', data, groups=data['subject']).fit()
print(model.summary())

Within-subject correlations, repeated measures, and multi-level modeling. Linear mixed models (LMM) and generalized linear mixed models (GLMM).

Model Selection & Regularization

Select the best model while avoiding overfitting. Methods: AIC, BIC, cross-validation, stepwise selection, and regularization.

from sklearn.linear_model import LassoCV
lasso = LassoCV(cv=5).fit(X, y)
print(f'Optimal alpha: {lasso.alpha_:.4f}')
print(f'Number of features used: {np.sum(lasso.coef_ != 0)}')

Ridge (L2), Lasso (L1), Elastic Net (L1+L2). Information criteria: AIC = 2k − 2ln(L̂), BIC = k·ln(n) − 2ln(L̂).

Bayesian Statistical Modeling

Bayesian models incorporate prior knowledge and quantify uncertainty through posterior distributions.

import pymc as pm
with pm.Model() as model:
    α = pm.Normal('α', mu=0, sigma=10)
    β = pm.Normal('β', mu=0, sigma=5)
    σ = pm.HalfNormal('σ', sigma=5)
    μ = α + β * X
    y_obs = pm.Normal('y_obs', mu=μ, sigma=σ, observed=y)
    trace = pm.sample(2000, tune=1000)
print(pm.summary(trace))

MCMC sampling (NUTS, Metropolis), posterior predictive checks, and model comparison with WAIC/LOO.
