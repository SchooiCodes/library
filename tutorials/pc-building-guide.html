PC Building Guide

Build your own PC — component selection, assembly, and benchmarking.

Table of Contents (2 sections)

  • Component Selection
  • Assembly & Benchmarking

Component Selection

PC components: CPU (Intel Core i5/i7/i9 — LGA1700/1851, AMD Ryzen 5/7/9 — AM5), GPU (NVIDIA GeForce RTX 40/50 series, AMD Radeon RX 7000/9000), motherboard (chipset — B760, Z790, B650, X670E — socket, PCIe lanes, DDR5 support), RAM (DDR5 — 6000-8000 MT/s, 32GB recommended), storage (NVMe M.2 SSD — PCIe 4.0/5.0, 1-2TB), PSU (power supply — 80+ Gold/Platinum, 750-1000W, modular), case (ATX, mATX, ITX — airflow). Cooling: air (tower cooler, dual tower), AIO liquid (240/360mm). Monitor: resolution (1440p, 4K), refresh rate (144Hz, 240Hz), panel type (IPS, OLED, VA). Peripherals: mechanical keyboard (Cherry MX, optical switches), mouse (DPI, wireless). Budget vs performance: prioritize GPU for gaming, CPU for productivity. Bottleneck: balance components.

Assembly & Benchmarking

Assembly: anti-static wrist strap, clear workspace. Steps: 1) CPU into socket (triangle alignment, no force). 2) RAM into slots A2+B2 (dual channel). 3) M.2 SSD into motherboard slot. 4) CPU cooler (thermal paste — pea size, mounting pressure). 5) PSU in case, route cables. 6) Motherboard in case (standoffs). 7) GPU into PCIe x16 slot. 8) Connect all cables (24-pin ATX, 8-pin CPU, PCIe power, SATA, front panel). 9) Cable management. 10) First boot: enter BIOS, check CPU temp, RAM speed (enable XMP/EXPO), boot drive order. OS install: Windows 11 (requires TPM 2.0, Secure Boot) or Linux. Drivers: GPU driver (NVIDIA/AMD), chipset drivers, network drivers. Benchmarking: Cinebench (CPU), 3DMark/Unigine (GPU), CrystalDiskMark (SSD), MemTest86 (RAM stability). Stress test: Prime95 + FurMark (temperature). Monitoring: HWMonitor, HWiNFO, MSI Afterburner.
