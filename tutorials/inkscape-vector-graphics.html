Inkscape Vector Graphics

Create logos, illustrations, and scalable vector graphics with Inkscape — the free SVG editor.

Inkscape Fundamentals

Inkscape is a free, open-source vector graphics editor for SVG files. It competes with Illustrator and CorelDRAW. The SVG format is web-native and resolution-independent.

Interface: canvas area, toolbar (left), tool controls (top), commands bar (top), snap controls (right), and color palette (bottom). Customize with View → Show/Hide.

Basic shapes: rectangle (R), ellipse (E), star/polygon (*), spiral, and 3D box. Each has adjustable parameters in the tool controls bar.

Pen Tool & Path Editing

The Pen (Bezier) tool creates smooth curves. Click to create corner nodes, click-and-drag for smooth curves. Node tool (N) edits individual nodes.

Path operations: Union (Ctrl++), Difference (Ctrl+−), Intersection (Ctrl+*), Exclusion (Ctrl+^), Division (Ctrl+/), and Cut Path. These combine shapes in powerful ways.

Simplify paths (Ctrl+L) reduces node count while preserving shape. Dynamic Offset and Linked Offset create outlines. Use Path → Inset/Outset for precise sizing.

Text & Typography

Inkscape handles text with SVG's text elements. Text tool (F8) creates point text or flowed text frames. Install additional fonts on your system to use them.

Convert text to path (Shift+Ctrl+C) for complete control over individual glyph shapes — essential for logo design. SVG fonts and web fonts can be referenced in exported files.

Text on path: select text, shift-select a path, then Text → Put on Path. Kerning and tracking adjusted with Alt+arrows or Text → Text and Font dialog.

Gradients & Filters

Gradients create smooth color transitions. Gradient tool (G) places gradient handles directly on the canvas. Linear, radial, and conical gradients supported.

Filters add effects: drop shadows, blurs, bevels, color adjustments, and distortions. The Filter Editor creates custom filters using SVG filter primitives (feGaussianBlur, feOffset, etc.).

Mesh gradients enable complex multi-color surfaces. Use the Create Mesh Gradient extension for advanced color transitions.

SVG Export & Web Use

Inkscape exports clean SVG code suitable for web use. Save As → Optimized SVG reduces file size. Copy SVG paths directly for use in CSS background-images or inline SVG.

Extensions → Export → Save as HTML+CSS creates web-ready output. Use the SVG Cleaner extension to remove Inkscape-specific metadata.

For responsive SVGs, remove absolute width/height and add viewBox. Embed SVGs inline for styling control with CSS — change fill colors, stroke widths, and animations.
