💿 Hiren's Boot CD PE

A complete guide to Hiren's Boot CD PE — the modern Windows PE rescue environment for recovery, diagnostics, and system repair.

What is Hiren's Boot CD PE?

Hiren's Boot CD PE is a bootable Windows PE (Preinstallation Environment) rescue environment based on Windows 10. Unlike the original Hiren's Boot CD (which was based on DOS with a collection of ancient tools), the PE edition boots into a full graphical Windows desktop with access to a vast array of modern system tools, recovery utilities, and diagnostic software.

It replaces the old Windows PE-based "Hiren's Boot CD" (versions around 15.x) with a more modern, regularly updated build. It is invaluable for IT professionals, technicians, and anyone who needs to repair or recover a computer that won't boot properly.

Hiren's PE vs Old Hiren's 15.2 (DOS)

FeatureHiren's Boot CD PEHiren's Boot CD 15.2

Simply put: if you're still using the old 15.2, it's time to upgrade. The PE edition is faster, more capable, and compatible with modern hardware including NVMe SSDs and USB 3.0 ports.

Getting Hiren's Boot CD PE

  1. Download the ISO from the official site at hirensbootcd.org. The file is roughly 2–3 GB.
  2. Verify the download — check the SHA-256 hash provided on the site to ensure the ISO is not corrupted.
  3. Add to Ventoy — simply copy the ISO file to your Ventoy USB drive. No extraction or special setup needed. Ventoy will pick it up automatically.

Alternatively, you can use Rufus to write the ISO directly to a USB drive if you prefer a dedicated boot stick.

Booting into Hiren's

  1. Insert your Ventoy USB and boot from it (press F12, F2, Del, or Esc to enter the boot menu, depending on your motherboard).
  2. Select Ventoy from the boot menu.
  3. Choose Hiren's Boot CD PE from the Ventoy ISO list.
  4. Wait for the Windows PE environment to load — this takes 30–90 seconds depending on your hardware.
  5. You will land on a full Windows 10 PE desktop with the Hiren's Start Menu already open.
Tip: If your system uses Secure Boot, make sure your Ventoy drive is configured for Secure Boot support (Ventoy has a Secure Boot option in its menu).

Navigation

The Hiren's PE desktop behaves like a stripped-down Windows 10. The Start Menu is organized into categorized folders, making it easy to find the tool you need:

  • Security — Antivirus scanners, malware removers, and secure deletion tools.
  • System Tools — Registry editors, system information, restore tools.
  • Hard Disk Tools — Partition managers, disk imagers, benchmarking, SMART monitoring.
  • Data Recovery — File recovery and undelete utilities.
  • Browsers — Pre-loaded web browsers for online scanning and research.
  • Password — Windows password reset and product key recovery tools.
  • Diagnostics — CPU, GPU, RAM, and hardware monitoring utilities.
  • Utilities — File managers, archivers, PDF readers, and general-purpose tools.

How To: Reset a Forgotten Windows Password with NTPWedit

  1. Boot into Hiren's Boot CD PE.
  2. Open the Start Menu → Password → NTPWedit.
  3. Click the Open button — it will automatically target the SAM file from your installed Windows (usually at C:\Windows\System32\config\SAM).
  4. NTPWedit will parse the SAM file and display all local user accounts.
  5. Select the user account whose password you want to reset.
  6. Click Change Password — leave the new password field blank to remove the password entirely, or enter a new one.
  7. Click Save Changes. NTPWedit will write the modified SAM file back to disk.
  8. Reboot the system and remove the USB drive. The user should now be able to log in (with no password or the new one you set).
Important: NTPWedit modifies the SAM file directly. If you have BitLocker encryption enabled, you will need the recovery key to access the drive first. Always create a backup of the SAM file before making changes (NTPWedit can do this for you).

How To: Clone a Failing Drive with Macrium Reflect

  1. Boot into Hiren's and connect your replacement drive (via USB SATA adapter or internal connection).
  2. Open the Start Menu → Hard Disk Tools → Macrium Reflect.
  3. Select the failing source drive (watch for SMART warnings or bad sectors).
  4. Click Clone this disk → select the target drive (the replacement).
  5. Critical: Enable "Intelligent Sector Copy" — this skips bad sectors and logs them, rather than hanging the entire clone process.
  6. Choose Copy exact layout (or let Macrium auto-fit if the target is smaller).
  7. Click Finish → confirm the warning that all data on the target will be destroyed.
  8. Wait for the clone to complete. If the source has many bad sectors, this could take hours — be patient.
  9. Once done, shut down, swap the drives, and boot from the new drive.
Pro Tip: If the drive is making clicking noises or is very slow, use HDTune or CrystalDiskInfo first to check the SMART status. If the drive is critically failing, consider using ddrescue (available via the command line in Hiren's) which is more resilient with failing media.

How To: Scan for Malware in Offline Mode

  1. Boot into Hiren's Boot CD PE — since Windows PE is a read-only environment, malware cannot run or hide.
  2. Connect to the internet if needed (for online scanners that download definitions):
    • Open the Start Menu → System Tools → Network Configuration or use the network icon in the system tray.
    • For Wi-Fi, select your network and enter the password.
  3. Open ESET Online Scanner from the Start Menu → Security.
  4. Let ESET download the latest virus definitions (requires internet).
  5. Select Full Scan and target all local drives (typically C:\, D:\, etc.).
  6. ESET will scan files directly — because the infected Windows is not running, malware cannot resist removal.
  7. Alternatively, use McAfee Stinger for a quick targeted scan against known threats.
  8. After the scan completes, review the detected items and choose to clean or delete them.
  9. Reboot back to your normal Windows — the malware should now be removed.
Why offline scanning works: Most malware relies on being loaded into memory and actively running to defend itself. By booting into a clean Windows PE environment, the malware's files sit inert on disk and can be safely deleted without resistance.
