Offline Developer Knowledge Bases

Access API documentation, cheat sheets, and developer references without an internet connection. Essential for remote work, travel, or disaster scenarios.

DevDocs.io — Docker Setup

DevDocs.io combines hundreds of API documentations into a single, fast search interface. You can run it locally via Docker.

Docker Deployment

# Pull the DevDocs image
docker pull ghcr.io/akash-akya/devdocs:latest

# Run with persistent storage
docker run -d \
  --name devdocs \
  -p 3000:3000 \
  -v devdocs_data:/opt/devdocs/public/docs \
  ghcr.io/akash-akya/devdocs:latest

Downloading Docsets

# Access the web UI at http://localhost:3000
# Click Preferences → select documentations to download
# Python, JavaScript, React, Docker, Linux — all available offline
Tip: The Docker image pre-downloads popular docsets. For a full offline setup, let it run overnight — downloading all docsets takes time.

Zeal — Docset Management via CLI

Zeal is an offline API documentation browser for Linux and Windows. It uses the same docset format as Dash (macOS).

Installation

# Ubuntu/Debian
sudo apt install zeal

# Arch Linux
sudo pacman -S zeal

# Windows — download from https://zealdocs.org

CLI Docset Management with zeal-cli

# Install the zeal-cli tool
pip install zeal-cli

# List available docsets
zeal-cli list

# Search for a specific docset
zeal-cli search python

# Download a docset
zeal-cli install python_3
zeal-cli install javascript
zeal-cli install react
zeal-cli install docker

# Update all installed docsets
zeal-cli update

Dash/Zeal Offline API Docs

Dash (macOS) and Zeal (Linux/Windows) use the same docset format. Most docsets are hosted at kapeli.com/docsets.

Manual Docset Installation

# Download a docset (example: Python 3)
wget https://kapeli.com/feeds/zzz/user-contributions/Python_3/Python_3.tgz

# Extract to Zeal's docset directory
tar -xzf Python_3.tgz -d ~/.local/share/Zeal/Zeal/docsets/

# Or for Dash on macOS
# ~/Library/Application Support/Dash/DocSets/

tldr Pages & cheat.sh

Community-maintained simplified man pages for common commands.

tldr (Node.js client)

# Install
npm install -g tldr

# Look up a command (requires first-time cache download)
tldr tar
tldr ffmpeg
tldr docker-compose

# Update local cache
tldr --update

cheat.sh

# Query from the command line (needs internet for live queries)
curl cheat.sh/tar
curl cheat.sh/python/list
curl cheat.sh/go/fmt

# Offline use with cheat.sh local server (Docker)
docker run -d -p 9111:9111 --name cheatsh tromer/cheat.sh
curl localhost:9111/tar
Tip: tldr --update downloads all pages to ~/.tldr/ — afterward, tldr works fully offline.

Docset Auto-Update Script

Keep your offline docsets fresh with a cron job or systemd timer.

#!/bin/bash
# ~/bin/update-docsets.sh

echo "Updating tldr pages..."
tldr --update

echo "Updating Zeal docsets..."
zeal-cli update

echo "Updating DevDocs (Docker)..."
docker exec devdocs bundle exec thor docs:update

echo "All docsets updated."
# Make it executable
chmod +x ~/bin/update-docsets.sh

# Cron — run weekly on Sunday at 3 AM
crontab -e
0 3 * * 0 /home/user/bin/update-docsets.sh

# Or systemd timer (see systemd-services.html)
