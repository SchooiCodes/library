Large Language Models

Large Language Models (LLMs) are the backbone of modern generative AI. They can answer questions, write code, generate text, summarize documents, and hold conversations. Below are the most notable LLMs available today, both proprietary and open-source.

Claude (Anthropic)

Claude 4 Opus is Anthropic's most capable model, excelling at reasoning, coding, and long-context tasks (up to 200K tokens). Known for safety-conscious design and strong performance on benchmarks.

Tiers: Free (Sonnet), Pro ($20/mo), Max ($100/mo)

Site: claude.ai

GPT-4 / GPT-4o (OpenAI)

GPT-4o is OpenAI's multimodal flagship. It understands text, images, and audio. GPT-4 Turbo offers lower pricing with high performance. Powers ChatGPT and API integrations worldwide.

Tiers: Free (GPT-4o mini), Plus ($20/mo), Pro ($200/mo)

Site: chatgpt.com

Gemini (Google)

Gemini 2.5 Pro is Google's most intelligent model. Natively multimodal, it processes text, images, audio, video, and code. Deep integration with Google ecosystem. 1M+ token context window.

Tiers: Free (Gemini), Advanced ($19.99/mo)

Site: gemini.google.com

Llama 3 / 4 (Meta)

Meta's open-weight Llama series. Llama 4 (released 2026) includes Scout (17B MoE), Maverick (17B MoE), and larger variants. Strong performance rivaling proprietary models. Self-hostable.

Tiers: Free (open-weight, commercial terms)

Site: llama.meta.com

Mistral AI

Mistral Large 2, Mistral Small, and open-weight Mistral 7B/Mixtral 8x22B. French AI lab known for efficient, high-quality models. Their le Chat app offers a free ChatGPT alternative.

Tiers: Free (le Chat), API pay-as-you-go

Site: mistral.ai

DeepSeek

DeepSeek-R1 is a powerful open-weight reasoning model that rivals GPT-4 and Claude on math and coding benchmarks. Extremely cost-effective API pricing. Self-hostable via Ollama.

Tiers: Free (web chat), API ($0.14/M tokens output)

Site: chat.deepseek.com

Grok (xAI)

Elon Musk's xAI Grok models. Grok 3 offers strong reasoning with real-time X/Twitter integration. Available via X Premium+ subscription or standalone API.

Tiers: X Premium+ ($16/mo), API pay-as-you-go

Site: grok.com

Image Generation

AI image generators have exploded in capability, from photorealistic renders to artistic styles. Here are the leading platforms and open-source models.

Midjourney

The gold standard for artistic AI imagery. Known for stunning aesthetics, style consistency, and active community. Discord-based with a web alpha in development.

Tiers: Basic ($10/mo), Standard ($30/mo), Pro ($60/mo)

Site: midjourney.com

Stable Diffusion 3.5

Open-source image generation model by Stability AI. SD3.5 and SDXL offer high-quality generation with full local control. Requires a GPU. Flux is the new state-of-the-art.

Tiers: Free (open-source), cloud options available

Site: stability.ai

Flux (Black Forest Labs)

Created by former Stability AI researchers. Flux.1 Pro is currently the highest-quality open-ish image model, rivaling Midjourney. Runs locally via ComfyUI or through API.

Tiers: Free (open weights, non-commercial), API tiers

Site: blackforestlabs.ai

DALL-E 3 (OpenAI)

Integrated into ChatGPT Plus. Excellent at following complex prompts, generating text within images, and producing coherent scenes. Less artistic control than Midjourney.

Tiers: Included with ChatGPT Plus ($20/mo)

Site: openai.com/dall-e

Google ImageFX

Google's Imagen 3 powering ImageFX. Free to use with a Google account. Strong photorealism and text rendering. Part of Google Labs suite.

Tiers: Free

Site: labs.google.com

Adobe Firefly

Adobe's generative AI integrated into Photoshop and Express. Legally safe for commercial use. Good for design workflows and photo editing with generative fill.

Tiers: Free (limited), Premium ($4.99/mo)

Site: firefly.adobe.com

Audio & Music Generation

AI audio tools can clone voices, generate music from text prompts, create sound effects, and master tracks. Here are the best in each category.

ElevenLabs

Industry-leading voice synthesis and cloning. Supports 29+ languages, emotion control, voice design, and real-time API. Used for audiobooks, dubbing, and content creation.

Tiers: Free (10K chars/mo), Starter ($5/mo), Creator ($22/mo)

Site: elevenlabs.io

MusicGen (Meta)

Open-source music generation model. Generate songs from text descriptions or melody references. Runs locally with adequate hardware. Good quality for instrumental and simple compositions.

Tiers: Free (open-source)

Site: github.com/facebookresearch/audiocraft

Stable Audio (Stability AI)

Generates music, sound effects, and audio from text prompts. Uses latent diffusion for high-quality stereo audio up to 90 seconds. Clean interface.

Tiers: Free (20 gen/mo), Pro ($11.99/mo)

Site: stabledudio.com

Suno AI

Leading AI music generation platform. Creates full songs with lyrics, vocals, and instruments. V4 model produces radio-quality tracks. Very popular for creative experimentation.

Tiers: Free (10 gen/day), Pro ($10/mo), Premier ($30/mo)

Site: suno.com

OpenAI Whisper

State-of-the-art open-source speech recognition (STT). Supports 99+ languages. Can run locally. Powers many transcription tools and is the backbone of voice interfaces.

Tiers: Free (open-source), API pay-as-you-go

Site: github.com/openai/whisper

Video Generation

AI video generation is evolving rapidly. These platforms can create short video clips from text prompts, animate images, and even generate full scenes with coherent motion.

Runway Gen-3 / Gen-4

Professional AI video generation platform. Gen-4 offers high consistency, camera control, and longer clips. Used in commercial film production. Web-based with API access.

Tiers: Free (125 credits), Standard ($15/mo), Pro ($35/mo)

Site: runwayml.com

Pika Labs

User-friendly AI video generator. Pika 2.0 supports video-to-video, image-to-video, lip sync, and various aspect ratios. Strong community and Discord integration.

Tiers: Free (limited), Pro ($10/mo), Unlimited ($30/mo)

Site: pika.art

Kling (Kuaishou)

Kling 1.6 from Kuaishou (China) produces high-quality 1080p videos with realistic physics and motion. Very competitive with Runway and Pika. Free tier available.

Tiers: Free (66 credits/day), Paid plans available

Site: klingai.com

Sora (OpenAI)

OpenAI's video generation model. Sora creates photorealistic 60-second videos from text prompts with impressive scene coherence. Available through ChatGPT Plus/Pro subscription.

Tiers: ChatGPT Plus ($20/mo), Pro ($200/mo)

Site: openai.com/sora

Code Assistants

AI code assistants boost developer productivity by autocompleting code, generating functions, explaining codebases, and debugging. Here are the best tools currently available.

GitHub Copilot

The most widely used AI code assistant. Integrates into VS Code, JetBrains, Neovim, and more. Offers autocomplete, chat, and agent modes. Powered by OpenAI models.

Tiers: Free (2000 completions/mo), Pro ($10/mo), Enterprise ($39/mo)

Site: github.com/features/copilot

Cursor

AI-first code editor (fork of VS Code). Built-in chat, inline editing, agent mode, and deep codebase understanding. Supports Claude, GPT-4, and custom models. Very popular.

Tiers: Free (2000 completions/mo), Pro ($20/mo)

Site: cursor.sh

Codeium / Windsurf

Free-first AI code assistant by Codeium. Windsurf is the standalone IDE variant. Supports multi-line autocomplete, chat, and commands. Unlimited completions on free tier.

Tiers: Free (unlimited), Pro ($15/mo)

Site: codeium.com

Tabnine

AI code completion that runs locally or in the cloud. Supports over 90 programming languages. Enterprise-focused with customizable models. Strong privacy options for on-prem deployment.

Tiers: Free (basic), Pro ($12/mo), Enterprise (custom)

Site: tabnine.com

Continue (Open Source)

Open-source AI code assistant for VS Code and JetBrains. Connect to any LLM (local or API-based). Supports chat, edit, and autocomplete. Fully customizable and free.

Tiers: Free (open-source)

Site: continue.dev

Local AI Tools

Running AI models locally gives you privacy, no censorship, and no ongoing costs. These tools let you run LLMs, image generators, and other models on your own hardware.

Ollama

Simplest way to run LLMs locally. Supports Llama, Mistral, DeepSeek, Gemma, Qwen, and hundreds more. One-command install and model pull. REST API for integrations.

Platforms: Windows, macOS, Linux

Site: ollama.com

LM Studio

User-friendly GUI for running local LLMs. Browse and download models from Hugging Face, chat with them, run a local OpenAI-compatible server. Great for beginners.

Platforms: Windows, macOS, Linux

Site: lmstudio.ai

ComfyUI

Powerful node-based UI for Stable Diffusion, Flux, and other image/video models. Visual workflow builder for advanced image generation. Steep learning curve but extremely capable.

Platforms: Windows, macOS, Linux

Site: github.com/comfyanonymous/ComfyUI

llama.cpp

High-performance C++ inference for LLMs. Runs on CPU and GPU. Quantized models (GGUF format) allow running on modest hardware. Powers Ollama and many other tools.

Platforms: Windows, macOS, Linux

Site: github.com/ggerganov/llama.cpp

Open WebUI

Self-hosted web interface for LLMs compatible with Ollama and OpenAI APIs. Features chat management, document RAG, model switching, user management. Like a local ChatGPT.

Deploy: Docker, pip, or bare metal

Site: openwebui.com

LocalAI

Open-source, self-hosted OpenAI API alternative. Supports LLMs, image generation, audio transcription, and text-to-speech. Drop-in replacement for OpenAI endpoints.

Deploy: Docker, Kubernetes, binary

Site: localai.io
