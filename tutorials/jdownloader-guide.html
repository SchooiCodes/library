⬇️ JDownloader 2 Guide

A comprehensive guide to JDownloader 2 — the powerful open-source download manager that handles link extraction, captcha solving, and automated downloads from hundreds of file hosts.

💿 Installation

Windows

  1. Download from jdownloader.org
  2. Run the installer — it installs both JDownloader 2 and Java (required)
  3. Launch from the desktop shortcut

Linux

# Download the installer
wget https://d1g1q00v0gqjbh.cloudfront.net/JD2Setup_x64.sh
chmod +x JD2Setup_x64.sh
./JD2Setup_x64.sh

# Or use the portable version (no installer)
wget https://d1g1q00v0gqjbh.cloudfront.net/JDownloader.jar
java -jar JDownloader.jar

macOS

brew install --cask jdownloader

🚀 First Run & Setup

When you first launch JDownloader, it will update itself and prompt you to create a MyJDownloader account (free, optional). You can skip this and register later.

Recommended: Create a MyJDownloader account. It lets you control downloads from your phone/web browser and syncs settings across devices.

🔗 Using the Link Grabber

JDownloader's Link Grabber automatically monitors your clipboard for supported URLs. Here's how to use it:

  1. Copy a link (e.g., from a file host like MediaFire, Mega, or Zippyshare)
  2. JDownloader automatically detects it and adds it to the Link Grabber tab
  3. Review the files, select/deselect what you want
  4. Right-click → "Add Links" or press Ctrl+V to manually add
  5. Click "Start All Downloads" (green play button)

Supported Services

JDownloader supports 1000+ file hosts, including Mega, MediaFire, Zippyshare, Google Drive, Dropbox, RapidGator, Uploaded, FileFactory, and most premium link generators.

🔑 Adding Premium Accounts

If you have premium accounts to file hosts, add them for faster downloads:

  1. Go to Settings → Account Manager
  2. Click "Add" and select the host from the list
  3. Enter your username and password
  4. JDownloader will verify and display the account status

🤖 Captcha Solving

Many free file hosts require captchas. JDownloader offers several solutions:

  • 9kw.eu: Paid captcha solving service (~1€ per 1000 captchas). Configure in Settings → Captcha
  • Manual: JDownloader pops up a window when a captcha needs solving — type it and continue
  • Auto: Some simple captchas are solved automatically by JD's built-in OCR

🛡️ Proxy & VPN Setup

Route your downloads through a proxy or VPN directly within JDownloader:

  1. Go to Settings → Advanced Settings
  2. Search for "Proxy"
  3. Add your proxy: Protocol (HTTP/SOCKS5), IP, Port
  4. Optionally set different proxies per host in Account Manager
Some file hosts block known VPN IPs. If downloads fail, try a different server or use a proxy instead.

☁️ MyJDownloader Remote

MyJDownloader lets you control your downloads from anywhere:

  1. Create a free account at my.jdownloader.org
  2. In JDownloader, go to Settings → MyJDownloader
  3. Enter your email and password
  4. Access your downloads via the web interface or mobile app

💻 Headless / CLI Mode

JDownloader can run on a headless server (no GUI) for always-on downloading:

# Download the headless installer
wget https://d1g1q00v0gqjbh.cloudfront.net/JD2SilentSetup_x64.sh
chmod +x JD2SilentSetup_x64.sh
./JD2SilentSetup_x64.sh

# Run headless
cd JDownloader/jd
java -jar JDownloader.jar -headless

Control it via the MyJDownloader web interface or the JDownloader 2 API.
