Typography & Color Theory

Typography and color theory — typefaces, color harmony, contrast, and accessibility.

Table of Contents (2 sections)

  • Typography
  • Color Theory & Accessibility

Typography

Type anatomy: baseline, x-height, cap height, ascender, descender, serif, stem, bowl, counter, aperture, ligature. Classifications: Serif (old-style, transitional, didone, slab), Sans-serif (grotesque, geometric, humanist), Script, Display, Monospace. Typeface vs font: typeface is design (Helvetica), font is file (Helvetica Bold 16pt). Font formats: TTF, OTF, WOFF, WOFF2 (web), variable fonts (single file with adjustable weight/width/slant). Pairing: complementary typefaces — sans-serif heading + serif body, or similar x-heights. Readability: line length (45-75 characters), line height (1.5x text), font size (16px body minimum). Hierarchy: heading sizes, weight, color, spacing. Font licensing: OFL (open-source), Google Fonts, Adobe Fonts (subscription), commercial licenses (Hoefler&Co, FontFont). Web fonts: @font-face, font-display (swap, optional, block). Loading: preload, subset fonts, variable fonts reduce HTTP requests. Icon fonts: Font Awesome, Material Icons — scalable vector icons.

Color Theory & Accessibility

Color model: RGB (additive — screens), CMYK (subtractive — print), HSL (hue, saturation, lightness — intuitive). Color wheel: primary (red, yellow, blue), secondary (green, orange, purple), tertiary. Harmonies: complementary (opposite — high contrast), analogous (adjacent — harmonious), triadic (120° apart — vibrant), split-complementary, tetradic (double complementary), monochromatic (one hue, variations). Saturation vs lightness: saturated = vivid, desaturated = muted. Cultural context: colors have different meanings (red = luck in China, danger in West). WCAG contrast: AA (4.5:1 normal text, 3:1 large), AAA (7:1 normal, 4.5:1 large). Tools: Stark plugin, Contrast Ratio checker, Color Safe. Color blindness: deuteranopia (green), protanopia (red), tritanopia (blue) — 8% of males. Don't rely solely on color — use icons, patterns, text labels. Color palettes: Coolors, Adobe Color, Material Design colors. Dark mode: desaturated colors, avoid pure black (#121212 recommended surface).
