Minecraft Server Hosting (Aternos)

Aternos is a free Minecraft server hosting service. This guide covers setup, cross-version support, Bedrock crossplay, cracked mode, and common workarounds.

Note: Aternos servers go to sleep when idle and take 1-2 minutes to start. This is expected for a free service.

Account Creation

  1. Go to aternos.org and click Register.
  2. Enter a username, email, and password. No payment info required.
  3. Verify your email via the confirmation link.
  4. Log in — you'll land on the server dashboard.
Note: Aternos queues free users during peak hours. Log in a few minutes early to secure your slot.

Server Setup & Configuration

Creating a Server

  1. Click Create a server on the dashboard.
  2. Select Minecraft: Java Edition or Bedrock Edition.
  3. Choose your server software: Vanilla, Paper (recommended), Spigot, Fabric, or Forge.
  4. Select the Minecraft version (1.20, 1.21, etc.).
  5. Click Create.

Configuration

# Common server.properties tweaks on Aternos:
# In the server settings panel:
max-players=20
difficulty=normal
pvp=true
spawn-protection=0
view-distance=10

Installing Plugins/Mods

  1. Go to Plugins or Mods tab.
  2. Search for the desired plugin (e.g., EssentialsX, LuckPerms).
  3. Click Install — Aternos downloads and applies it.
  4. Restart the server to activate.

ViaVersion — Cross-Version Support

ViaVersion allows clients on different Minecraft versions to connect to the same server.

Installation

  1. In the Plugins tab, search for ViaVersion.
  2. Install ViaVersion, ViaBackwards, and ViaRewind (for older clients).
  3. Restart the server.

GeyserMC — Bedrock Crossplay

GeyserMC lets Bedrock Edition (mobile, console, Windows 10) players join your Java server.

Installation on Aternos

  1. Go to Plugins and search for GeyserMC.
  2. Also install Floodgate (allows Bedrock players to join without a Java account).
  3. Restart the server.

Connecting from Bedrock

# In Geyser's config.yml (auto-generated):
# bedrock:
#   port: 19132
#   clone-remote-port: true

# Tell Bedrock players to connect using:
# Address: yourserver.aternos.me
# Port: 19132 (default Bedrock port)
Note: Bedrock players on mobile/console may need a DNS workaround to connect to Aternos. See the workarounds section below.

Cracked Mode Setup

Cracked mode allows players without a paid Minecraft account to join your server.

  1. In the Aternos server settings, enable Cracked (also called "Online Mode" = OFF).
  2. Save and restart the server.
  3. Players can now join with any username using a non-premium launcher.
Security: Cracked mode disables account authentication. Anyone can join using any username. Use plugins like AuthMe or LoginSecurity for password protection.

Bedrock Workarounds

Common issues and fixes for Bedrock players on Aternos.

DNS Resolution for Mobile/Console

# Bedrock on mobile/console often can't resolve Aternos DDNS.
# Workaround: Use a free DNS service like DuckDNS or No-IP.

# On your Bedrock device, set a static DNS entry:
# Server Address: your-aternos-server.aternos.me
# Port: 19132

# If that fails, use a local proxy:
# On a PC on the same network, run:
socat UDP-LISTEN:19132,fork UDP:yourserver.aternos.me:19132

Connection Timeout

  • Start the Aternos server first — it takes ~1-2 minutes to boot.
  • Bedrock players should wait until the server status shows "Online".
  • Check that Geyser and Floodgate are both installed and up to date.

Port Forwarding Alternative

If Aternos's built-in Bedrock port doesn't work, use a free tunnel service like Playit.gg or Ngrok to tunnel the Geyser port.

# Playit.gg (free, no account required on older version)
# Install the Playit agent plugin on Aternos, then
# connect via the Playit tunnel address on port 19132
