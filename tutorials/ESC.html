📄 How to Use and Type the ESC Character

Table of Contents (5 sections)

Tip: Save any unsaved work before experimenting with the ESC key, as some programs may react unexpectedly.

What is the ESC Character?

The ESC (Escape) character is an ASCII control code (27 decimal, 0x1B hex). It begins escape sequences that control terminals, printers, and other devices.

How to Type the ESC Character

  1. Ensure Num Lock is enabled.
  2. Hold the Alt key.
  3. On the numeric keypad, type 027.
  4. Release the Alt key – the ESC character appears.

Common Uses

  • Exiting insert mode in Vim or other modal editors.
  • Initiating ANSI escape sequences for cursor movement or color changes in terminals.
  • Representing non‑printable characters in strings (e.g., "\x1b" in many languages).

Code Example (Python)

print("Hello, world!\x1b[31m This text is red.\x1b[0m")

The \x1b[31m makes the following text red; \x1b[0m resets the color.

Conclusion

Understanding and being able to generate the ESC character helps you work with terminal control codes, scripting, and various developer tools.
