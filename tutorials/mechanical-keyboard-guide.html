Mechanical Keyboard Buyer's Guide

Find the perfect mechanical keyboard — switch types, form factors, keycaps, and vendors for Linux.

Table of Contents (5 sections)

Switch Types & Feel

Mechanical switches define keyboard feel. Three main types:

  • Linear: Smooth, consistent press. Cherry MX Red, Gateron Yellow, Kailh Box Black. Best for gaming.
  • Tactile: Bump mid-press for feedback. Cherry MX Brown, Gateron Brown, Holy Panda. Best for typing.
  • Clicky: Tactile bump + audible click. Cherry MX Blue, Kailh Box White, NovelKeys Cream. Best for typing (but noisy).

Factors: actuation force (35-80g), pre-travel (1.2-2.0mm), total travel (3.0-4.0mm). Hot-swappable sockets allow changing switches without soldering.

Form Factors & Layouts

Common keyboard sizes:

  • Full-size (100%): All keys including numpad. Heavy, desk space hog. Low-spec Cherry G80-3000.
  • TKL (80%): No numpad. Most popular among enthusiasts. Keychron Q3, Leopold FC750R.
  • 75%: TKL with nav cluster condensed. Keychron V1, Q1. Very popular.
  • 65%: Arrows + a few nav keys. Keychron V2, K6. Good balance of space and function.
  • 60%: No arrows or function row. Ducky One 3 Mini, Anne Pro 2. Ultra-portable.
  • 40%: Minimal — requires layers for numbers/symbols. Planck, Vortex Core. Niche.

Keycaps & Materials

Keycap materials: ABS (smooth, shiny over time, cheaper), PBT (textured, durable, fade-resistant, preferred). Profile: Cherry (sculpted, low), OEM (taller), SA (high, retro), DSA (flat), XDA (flat, wide).

Manufacturing: double-shot (legends never fade), dye-sublimated (good color, PBT only), laser-etched (cheap, wears off).

Popular sets: GMK (high-end ABS), ePBT (PBT), Tai-Hao (budget PBT), Drop + MiTo (custom designs).

Linux Compatibility & Programming

Most mechanical keyboards work on Linux via generic HID. Key concerns:

  • Via/Vial support — web-based configurator for QMK keyboards (Keychron Q series, Glorious GMMK Pro). Works perfectly on Linux via Chrome/Chromium.
  • QMK Firmware — open-source keyboard firmware. Compile with qmk setup then qmk flash. Full remapping, macros, combos, layers, tapping terms.
  • KMonad — software keyboard remapper for Linux when keyboard isn't QMK-compatible.

Recommended Linux-friendly keyboards: Keychron Q/V/K series, Drop CTRL/ALT, ZSA Voyager/Moonlander, Dygma Raise 2.

Budget-to-Premium Recommendations

Entry ($30-60): Keychron C3 Pro (wired, hotswap), Redragon K552 (outemu switches).

Mid-range ($80-150): Keychron V1/V2 (PBT keycaps, QMK/Via, hotswap), NuPhy Air75 v2 (low-profile, wireless), Akko 3068B Plus.

Premium ($150-250): Keychron Q1/Q3 (full aluminum, gasket mount, QMK), Glorious GMMK Pro, Monsgeek M1W.

Luxury ($300+): ZSA Voyager (split ergo), Mode Sonnet, Satisfaction 75 (group buy), custom builds (solder + lube + mod).
