Linear Algebra Fundamentals

Matrices, vectors, eigenvalues, and linear transformations — essential math for ML and graphics.

Vectors & Vector Spaces

A vector is an ordered tuple of numbers representing magnitude and direction. Vector spaces are collections of vectors closed under addition and scalar multiplication.

# Python example
import numpy as np
v = np.array([3, 4])
w = np.array([1, 2])
print(v + w)   # [4 6]
print(np.dot(v, w))  # 11

Key operations: addition, scalar multiplication, dot product, cross product (3D), and norm (magnitude).

Matrices & Linear Transformations

Matrices are rectangular arrays of numbers that represent linear transformations. An m×n matrix maps vectors from Rⁿ to Rᵐ.

A = np.array([[2, 0], [0, 3]])
v = np.array([1, 1])
print(A @ v)  # [2 3] — scales x by 2, y by 3

Matrix multiplication, transpose, inverse, and determinant are fundamental operations. The identity matrix I leaves vectors unchanged.

Eigenvalues & Eigenvectors

An eigenvector of a matrix A is a non-zero vector v such that Av = λv, where λ is the eigenvalue. Eigenvectors reveal the intrinsic directions of a transformation.

A = np.array([[2, 1], [1, 2]])
eigvals, eigvecs = np.linalg.eig(A)
print(eigvals)  # [3. 1.]

Eigendecomposition is used in PCA, PageRank, quantum mechanics, and dimensionality reduction.

Systems of Linear Equations

Solving Ax = b is central to engineering. Methods include Gaussian elimination, LU decomposition, and iterative methods like conjugate gradient.

A = np.array([[3, 2], [1, 2]])
b = np.array([7, 5])
x = np.linalg.solve(A, b)
print(x)  # [1. 2.] — solution

Use np.linalg.solve for direct solving. For large sparse systems, use iterative solvers from scipy.sparse.linalg.

Singular Value Decomposition

SVD factors any matrix A into UΣVᵀ. It is the fundamental theorem of linear algebra, used for compression, recommendation, and latent semantic analysis.

U, S, Vt = np.linalg.svd(A, full_matrices=False)
# Truncated SVD for dimensionality reduction
k = 2
A_reduced = U[:,:k] @ np.diag(S[:k]) @ Vt[:k,:]

SVD is the backbone of principal component analysis (PCA) and collaborative filtering.
