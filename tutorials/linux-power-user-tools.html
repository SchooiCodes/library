Linux Power User Tools

Essential Linux utilities every power user should know — fd, ripgrep, fzf, bat, jq, and more.

Modern CLI Replacements

Modern alternatives to classic Unix tools. Install via your package manager:

# Install modern CLI tools (Arch)
sudo pacman -S fd bat ripgrep fzf jq htop neofetch duf

# Comparisons:
# find → fd (faster, intuitive syntax)
# grep → ripgrep (rg) (git-aware, faster)
# cat → bat (syntax highlighting, git integration)
# ls → exa/eza (colorized, icons, tree view)
# du → duf (colorized disk usage)
# top → htop/btop (interactive process viewer)

Examples: fd pattern finds files; rg 'class.*Widget' src/ searches code; bat file.py shows syntax-highlighted output.

fzf — Fuzzy Finder

fzf is a general-purpose fuzzy finder that integrates with your shell for history search, file navigation, and more:

# Search and change directories
zoxide + fzf: z dirname

# Ctrl+R — fuzzy history search
# Ctrl+T — fuzzy file search
# Alt+C — fuzzy cd

# Custom fzf usage
kill -9 $(ps aux | fzf | awk '{print $2}')
# Preview files while searching
fd --type f | fzf --preview 'bat --color=always {}'

Integrate fzf with vim/neovim via :FZF or use fzf.vim plugin.

zoxide & Shell Enhancements

zoxide is a smarter cd that learns your directory preferences:

# Install
eval "$(zoxide init zsh)"  # for zsh
eval "$(zoxide init bash)" # for bash

# Usage
z projects       # cd to most relevant path matching 'projects'
zi Downloads     # interactive selection
z foo bar        # narrows to path containing 'foo' and 'bar'

Other shell enhancements: thefuck (corrects previous command), tldr (simplified man pages), ncdu (interactive disk usage), and tmux (terminal multiplexer).

System Monitoring Tools

Monitor system resources with modern tools:

# btop — terminal resource monitor with mouse support
btop

# bottom (btm) — Rust-based resource monitor
btm

# glances — cross-platform monitoring with web interface
glances

# duf — disk usage with colorized output
duf

# procs — modern ps replacement
procs | grep firefox

# bandwhich — per-process network usage
bandwhich -i wlan0

Combine with watch for continuous monitoring: watch -n1 nvidia-smi.

File Managers & Editors

Terminal file managers: lf (Go, minimal), ranger (Python, extensive), nnn (C, fastest), joshuto (Rust, ranger-like).

Text editors: neovim (extensible), micro (simpler, mouse-support), helix (modal with built-in LSP), emacs (everything-in-one).

Quick viewing: glow (markdown), chafa (images in terminal), pistol (auto file preview), ffmpeg (video/audio conversion).
