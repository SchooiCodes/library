Home Lab Hardware Guide

Build a home lab — servers, networking, storage, and virtualization hardware for self-hosting.

Home Lab Tiers

Home labs scale from a single Raspberry Pi to a full rack setup:

  • Tier 1 (~$100): Raspberry Pi 5 (8GB) — Pi-hole, Homebridge, Jellyfin, lightweight web services, low power (~5W).
  • Tier 2 (~$300): Used mini PC (HP EliteDesk 800 G6, Lenovo ThinkCentre M720q, Dell OptiPlex 7080) — Proxmox host, 4-8 containers, 32-64GB RAM.
  • Tier 3 (~$600): Custom build or used server (Intel N100/N305, Ryzen 5 7600) — TrueNAS, multiple VMs, Docker swarm.
  • Tier 4 ($1500+): Used enterprise gear — Dell PowerEdge R730xd, SuperMicro, Ubiquiti networking, Synology NAS, UPS.

Used Enterprise Hardware

Buying used enterprise gear saves money but has trade-offs:

Popular servers: Dell PowerEdge R730/R740 (13th/14th gen), HP ProLiant DL380 Gen10, SuperMicro SYS-5029C-T. Prices: $200-800 for dual Xeon, 64-256GB RAM.

Key considerations: noise (rack servers are LOUD — not suitable for home unless soundproofed), power draw (200-400W idle), heat output, and drive caddies (often sold separately).

Where to buy: r/homelabsales, eBay (LabGopher to search), ServerMonkey, Uniserv.

Networking Gear

Switches: Uniquiti Switch Lite 16 PoE (managed), MikroTik CSS326-24G-2S+RM (huge 24-port, SFP+ 10GbE), TP-Link TL-SG108E (basic managed).

Routers/Firewalls: OPNsense/PFSense on Protectli Vault (4x2.5GbE), Uniquiti Dream Machine SE, MikroTik hAP ax3.

Access points: Uniquiti U6 Pro (Wi-Fi 6, meshing), Grandstream GWN7660, TP-Link EAP670.

Cabling: CAT6A for 10GbE (shielded for interference). Patch panels and keystone jacks for tidy racks.

Storage Strategy

NAS OS: TrueNAS Scale (ZFS, containers, VMs), Unraid (flexible parity, Docker, easy expansion), OMV (lightweight, plugins).

Disk layout: 4× HDD (RAID-Z1 = ~12TB usable with 4×4TB) + 2× NVMe (mirrored for VM/DB storage) + 1× SSD (boot).

Backup strategy: 3-2-1 rule (3 copies, 2 media, 1 off-site). Use BorgBackup or Kopia for versioned backups. Sync to Backblaze B2 or Wasabi for off-site.

Virtualization Stack

Hypervisor: Proxmox VE (KVM + LXC, web UI, clustering). Host on Debian, manage via web interface. Features: live migration, HA, backups, ZFS.

Orchestration: Docker Swarm (simpler) or Kubernetes (k3s, MicroK8s) for container orchestration. Portainer for Docker UI.

Services to self-host: AdGuard Home (DNS blocking), Jellyfin (media server), Nginx Proxy Manager (reverse proxy), Vaultwarden (password manager), GitLab/Gitea (code hosting), Home Assistant (home automation).
