Discord Modding

Vencord & Vesktop

Mod your Discord client with plugins, themes, and quality-of-life improvements using Vencord and the standalone Vesktop client.

Overview

Vencord is a free, open-source client mod for Discord that injects plugins and themes directly into the official Discord desktop or web client. Unlike older mods like BetterDiscord, Vencord is built for performance, uses Discord's own internals cleanly, and ships with a built-in plugin store.

Vesktop is a standalone desktop Discord client that bundles Vencord natively. It wraps Discord's web app in a lightweight Electron shell with additional features like screenshare audio, better Linux support, and Wayland integration. You do not need to install Vencord separately when using Vesktop — it is included.

No Account Risk. Discord's policy on client modding is ambiguous. Vencord does not send your token to third parties, but using any mod could technically violate ToS. In practice, bans are extremely rare and typically only happen if you use self-bot plugins for abuse. Use at your own discretion.
Safety Warning. Always download Vencord from the official GitHub repository (https://github.com/Vendicated/Vencord) and Vesktop from https://github.com/Vencord/Vesktop. Third-party builds may contain malware or token stealers.

Installation — Vencord (Browser Extension)

Vencord offers a browser extension that works with Discord's web client. This is the safest method because it runs in a sandboxed browser environment.

Vencord Web Installer

# Visit the official Vencord site in your browser:
https://vencord.dev

# Click "Download" and follow the instructions for your browser.
# Supported: Chrome, Firefox, Edge, Opera, Brave

After installing the extension, open https://discord.com/app and you will see a Vencord settings icon in the user menu (gear icon next to your name).

Vencord Desktop (Manual Injection)

For the desktop client, run the installer script. This modifies the official Discord installation to load Vencord's code.

# Windows (PowerShell as Administrator):
iwr -useb "https://raw.githubusercontent.com/Vendicated/Vencord/main/installer.ps1" | iex

# Linux/macOS (terminal):
sh -c "$(curl -s https://raw.githubusercontent.com/Vendicated/Vencord/main/installer.sh)"

# The script detects your Discord installation and patches it.
# Re-launch Discord after installation.

On Linux, the script supports Discord Stable, Discord Canary, and Discord PTB. It also works with Flatpak installations if you select the correct path.

Updating Vencord

Vencord updates automatically when Discord updates. However, after a major Discord update, you may need to re-run the installer. The browser extension updates through your browser's extension store.

Vesktop — Standalone Client

Vesktop is a full Discord client replacement that bundles Vencord. It is built on Electron and provides features not available in the official client.

Key Features

  • Screenshare Audio — Share your screen with system audio on all platforms (Windows, Linux, macOS). The official client restricts this to Windows only.
  • Wayland Support — Native Wayland support on Linux for smooth rendering and screen sharing under GNOME/KDE Wayland sessions.
  • Vencord Built-In — No separate installation needed. All Vencord plugins and themes work out of the box.
  • Lightweight — Less memory usage than the official client in many cases.
  • Open-Source — Full source available; no telemetry or proprietary blobs.
  • Multiple Accounts — Support for multiple Discord accounts natively.
  • Custom CSS — Inject your own CSS without a separate theme.

Installation

# Linux — AppImage (recommended):
# Download from https://github.com/Vencord/Vesktop/releases
chmod +x Vesktop-*.AppImage
./Vesktop-*.AppImage

# Linux — AUR (Arch Linux):
yay -S vesktop

# Linux — Flatpak:
flatpak install flathub dev.vencord.Vesktop

# Windows:
# Download the installer from the releases page.
# Run the .exe and follow the setup wizard.

# macOS:
# Download the .dmg from the releases page.
# Drag Vesktop to Applications.

Configuration

Vesktop stores its config in ~/.config/Vesktop/ on Linux/macOS and %APPDATA%/Vesktop/ on Windows. You can edit settings.json for advanced options like disabling hardware acceleration or changing the Discord API endpoint.

Recommended Plugins

Vencord ships with dozens of plugins accessible from User Settings > Vencord > Plugins. Here are the most useful ones:

PluginFunction
SpotifyCrackBypasses Spotify "listening along" restrictions and shows full song titles in your status.
MessageLoggerSaves deleted and edited messages locally. Only you can see them.
NoTrackBlocks Discord's tracking and analytics endpoints.
ImageUtilitiesAdds image zoom, copy, save-as, and reverse image search to all media embeds.
FakeNitroUnlocks many Nitro features: emojis, stickers, screenshare quality, upload size limit. Client-side only — others will not see your custom emoji unless they have the same plugin.
VoiceMessagesLets you send voice messages from desktop (normally a mobile-only feature).
GameActivityToggleQuickly toggle your game activity status on/off from a button next to the mic.
ShowHiddenChannelsDisplays hidden/locked channels (you cannot read them unless you have access, but you can see they exist).
CallTimeCounterShows how long you have been in a voice call.
ReadAllNotificationsButtonAdds a "Mark all read" button to your server list.
Plugins Are Client-Side. All Vencord plugins run only on your machine. Other users cannot see your custom emoji, read deleted messages you logged, or see plugins you have enabled. Do not use self-bot plugins — they violate ToS and can get your account banned.

Themes

Vencord supports CSS themes that completely change the look of Discord. You can find themes on the official Vencord theme repository or sites like https://betterdiscord.app/themes (many BD themes work with Vencord after minor tweaks).

Installing a Theme

  1. Open Discord and go to User Settings > Vencord > Themes.
  2. Click "Open Themes Folder". This opens the themes directory in your file manager.
  3. Download a .theme.css file (or a .css file) and place it in the themes folder.
  4. Go back to Discord and enable the theme from the themes list.

Popular Themes

  • ClearVision — A highly customizable theme with color sliders and many presets.
  • Midnight — Dark, sleek theme with rounded corners and blur effects.
  • Monstercat — Inspired by Monstercat's branding; vibrant colors.
  • Simply Dark — Minimal dark theme — no frills, just clean dark UI.
  • Slate — macOS-inspired theme with frosted glass effects.
  • Discord Recolored — Lets you recolor every UI element via a CSS file.

Custom CSS Snippets

You can also inject small CSS snippets directly in Vencord settings without creating a full theme file:

/* Hide the "Gift Nitro" button */
[aria-label="Send a gift"] { display: none !important; }

/* Hide the "Shop" icon in the toolbar */
[href="/shop"] { display: none !important; }

/* Make the channel list darker */
.sidebar_ded4b5 { background: var(--background-secondary) !important; }

/* Compact mode: reduce padding between messages */
.message_d5deea { margin-bottom: 2px !important; }

To use these, go to Vencord Settings > Vencord > Custom CSS and paste your snippet there. Changes apply immediately.

Troubleshooting

Vencord Not Loading After Discord Update

Discord updates often break mods. Re-run the Vencord installer script — it will re-patch the new Discord installation.

Vesktop Won't Launch on Linux

If Vesktop fails to launch, try running it from the terminal to see error output:

# Run from terminal:
./Vesktop-*.AppImage

# If you get a FUSE error, extract and run:
./Vesktop-*.AppImage --appimage-extract
./squashfs-root/vesktop

# Or install the FUSE library:
sudo apt install fuse   # Debian/Ubuntu
sudo pacman -S fuse     # Arch

Hardware Acceleration Issues

If you experience screen tearing or high GPU usage, disable hardware acceleration:

# In Vesktop, edit the config file:
# Linux: ~/.config/Vesktop/settings.json
# Windows: %APPDATA%/Vesktop/settings.json

# Set:
{
  "hardwareAcceleration": false
}

Plugins Tab Grayed Out

Ensure you are running the latest version of Vencord. If the plugins tab is grayed out, Vencord failed to inject. Re-install using the installer script.

Screenshare Audio Not Working in Vesktop

On Linux, ensure you have pipewire and pipewire-pulse installed. Vesktop uses PipeWire for audio loopback. On Windows, make sure your audio drivers are up to date.

Tips & Best Practices

  • Do not enable every plugin. Only enable what you need. Some plugins (like MessageLogger) store data locally and can consume disk space over time.
  • Backup your settings. Copy ~/.config/Vencord/settings.json (or the Vesktop equivalent) if you reinstall.
  • Use Vesktop for better Linux support. Wayland screen sharing and audio are significantly better than the official client.
  • Check for updates regularly. Both Vencord and Vesktop receive frequent updates. Follow their GitHub repos for release announcements.
  • Do not install random plugins. Only install plugins from Vencord's built-in plugin store. Third-party plugins could be malicious.
  • Hide your modded status. If you are concerned about detection, avoid using plugins that visibly modify Discord's UI in ways that could be detected (like showing Nitro features in tooltips that others could screenshot).
  • Use multiple accounts safely. Vesktop allows multiple accounts, but do not use self-bot automation on any of them.
  • Join the Vencord Discord. The community is active and helpful for troubleshooting. Link is on the GitHub README.
