SMT — System Multitool Guide

SMT (System Multitool) is a batch-powered Windows utility collection with over 90 tools for networking, system configuration, activation, security research, and more.

What is SMT?

SMT started as a small collection of batch scripts and grew into a massive toolbox with 90+ integrated tools. It's designed for Windows 10 & 11 and provides a clean console interface for common system tasks.

Categories of Tools

  • Networking: IP geolocation, DNS changer, network diagnostics, bandwidth tests
  • System Utilities: CLI task manager, system info, bootable USB creator, disk cleaner
  • Windows & Office Activation: Multiple activation methods via Microsoft Activation Scripts
  • Security Research: Credential logger, file integrity checker, info stealer generator (educational)
  • App Installers: One-click installs for popular software
  • School Utilities: Tools for educational environments
  • Auto-Update System: Built-in update mechanism
Disclaimer: Some tools (activation, security research) are for educational purposes only. Use responsibly and only on systems you own.

Installation via PowerShell

SMT is installed using a one-line PowerShell command that downloads and runs the latest version.

# Open PowerShell as Administrator and run:
irm https://raw.githubusercontent.com/SchooiCodes/smt/main/install.ps1 | iex

This command:

  1. Downloads the installer script from GitHub
  2. Fetches the latest SMT release
  3. Extracts it to your chosen directory
  4. Creates a shortcut for easy access
Requirements: Windows 10/11 64-bit, PowerShell 5.1+. Administrator privileges are required for some tools.

Alternative: Manual Download

# Download the latest release from GitHub
# https://github.com/SchooiCodes/smt/releases/latest

# Extract and run SMT.bat as Administrator

Feature Walkthrough

IP Geolocator

Enter any IP address or domain and get detailed geographic information — country, city, ISP, coordinates, and more. Uses free geo-IP APIs.

Windows & Office Activator

Activates Windows 10/11 and Office products using Microsoft Activation Scripts (MAS). Supports HWID, KMS, and Ohook methods.

Bootable USB Creator

Creates bootable USB drives for OS installation. Integrates with Ventoy for multi-boot support.

DNS Changer

Quickly switch between DNS providers (Google, Cloudflare, OpenDNS, Quad9) or set custom DNS servers. Useful for bypassing restrictions or improving privacy.

Example Workflow

1. Launch SMT as Administrator
2. Select [1] Networking Tools
3. Select [1] IP Geolocator
4. Enter "8.8.8.8" to look up Google's DNS
5. View results (country, ISP, coordinates)
6. Press 0 to return to the main menu
7. Select [3] System Utilities
8. Select [5] Bootable USB Creator
9. Follow the prompts to create a bootable drive

Quick Tips

  • Always run SMT as Administrator for full functionality.
  • Use the auto-update option to keep SMT current.
  • Some tools require an internet connection (IP geolocation, activation).
  • Check the full tool list for all 90+ tools.
