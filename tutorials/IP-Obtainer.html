📄 IP Obtainer (Legacy)

Table of Contents (1 section)

  • Overview

Overview

This legacy tool extracts the public IP address from a generated image file.

Setup

  1. Download the Funny Image Generator.zip and extract its contents.
  2. Run funney image generator.bat (the batch file inside the archive).
  3. After execution, open the resulting image file in a text editor (e.g., Notepad) to view the embedded IP address.

Note: This project is superseded by SMT, which provides more advanced information‑stealing utilities.
