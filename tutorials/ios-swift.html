iOS Development (Swift)

iOS development with Swift — SwiftUI, UIKit, and App Store deployment.

Table of Contents (2 sections)

  • SwiftUI
  • UIKit & App Lifecycle

SwiftUI

SwiftUI: declarative UI — struct ContentView: View { var body: some View { Text("Hello").font(.title) } }. Layout: VStack, HStack, ZStack, List, ScrollView, LazyVGrid, NavigationStack, TabView. State: @State private var count = 0, @Binding (parent to child), @ObservedObject / @StateObject (ObservableObject), @EnvironmentObject (shared). Modifiers: .padding(), .foregroundColor(), .frame(), .background(), .clipShape(), .shadow(). Navigation: NavigationLink, .navigationDestination(). Previews: #Preview { ContentView() }. Swift Charts: Chart { BarMark(x: , y:) }.

UIKit & App Lifecycle

UIKit: legacy but still essential. ViewController: UIViewController — viewDidLoad, viewWillAppear, viewDidAppear. Auto Layout: NSLayoutConstraint, Visual Format Language, SnapKit (third-party). Storyboards vs XIBs vs code-only UI. App lifecycle: AppDelegate (didFinishLaunching, scene configuration), SceneDelegate (multi-window iPadOS). Codable: Encodable/Decodable for JSON. Core Data: persistent storage, NSPredicate queries, CloudKit sync. Combine: reactive framework — Publisher, Subscriber, @Published. async/await: Swift concurrency, URLSession async, Task, Actor.
