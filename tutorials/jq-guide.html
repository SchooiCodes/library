🔧 jq Guide

Parse, filter, and transform JSON from the command line using jq — queries, pipes, arrays, and practical API data extraction.

Table of Contents

  • Filtering

Basic Queries

# Pretty-print JSON
curl -s https://api.github.com/users/SchooiCodes | jq .

# Extract a field
curl -s https://api.github.com/repos/jqlang/jq | jq '.name'

# Nested field
curl -s https://api.github.com/repos/jqlang/jq | jq '.owner.login'
Use jq -r to output raw strings without quotes — essential for piping into other shell commands.

Filtering

# Select objects by condition
curl -s https://api.github.com/search/repositories?q=topic:jq \
 | jq '.items[] | select(.stargazers_count > 1000) | {name: .name, stars: .stargazers_count}'

# Multiple conditions
echo '[{"name":"a","active":true},{"name":"b","active":false}]' \
 | jq '.[] | select(.active == true) | .name'

Array Operations

# Length of array
curl -s https://api.github.com/repos/jqlang/jq/issues | jq 'length'

# Map over array
curl -s https://api.github.com/repos/jqlang/jq/issues \
 | jq '[.[] | {title: .title, state: .state, number: .number}]'

# First / last / slice
jq '.[0]' file.json
jq '.[-1]' file.json
jq '.[:5]' file.json

# Unique values
jq '[.[].user.login] | unique' file.json

Object Construction

# Build new objects
curl -s https://api.github.com/repos/jqlang/jq \
 | jq '{repo: .full_name, stars: .stargazers_count, issues: .open_issues_count, url: .html_url}'

# Add computed fields
curl -s https://api.github.com/repos/jqlang/jq \
 | jq '{name, age_days: (now - (.created_at | fromdateiso8601)) / 86400 | floor}'

Pipes & Functions

# Pipe operator
jq '.items | group_by(.language) | to_entries[] | {key, count: (.value | length)}'

# Built-in functions
jq 'map(select(.name | test("jq|json"; "i")))' file.json

# Reduce
jq 'reduce .[] as $item (0; . + $item.stars)' file.json
Store a jq filter in a variable or file (jq -f myfilter.jq data.json) for complex, reusable transformations.

Practical API Examples

# Get latest release tag
curl -s https://api.github.com/repos/jqlang/jq/releases/latest | jq -r '.tag_name'

# List open PRs with author and title
curl -s https://api.github.com/repos/jqlang/jq/pulls \
 | jq -r '.[] | "\(.user.login): \(.title)"'

# Tree of repo languages
curl -s https://api.github.com/repos/jqlang/jq/languages \
 | jq -r 'to_entries | .[] | "\(.key): \(.value) bytes"'

# CSV output
curl -s https://api.github.com/repos/jqlang/jq/issues \
 | jq -r '.[] | [.number, .title, .state, .user.login] | @csv'
