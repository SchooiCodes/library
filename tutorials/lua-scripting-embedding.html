Lua Scripting & Embedding

Lightweight embeddable scripting language — Lua syntax, C API, and game scripting.

Table of Contents (2 sections)

  • Lua Syntax & Tables
  • C API & Embedding

Lua Syntax & Tables

Lua is minimal (~200KB) and fast. Tables are the universal data structure: t = {key = "value", [1] = "array"}. Metatables: setmetatable(t, {__index = other, __call = f}) enables OOP, operator overloading. Functions are first-class: local fn = function(x) return x*2 end. Coroutines: co = coroutine.create(fn); coroutine.resume(co). Multiple return values: function f() return 1, 2, 3 end.

C API & Embedding

Embed Lua in C/C++: luaL_newstate(), luaL_openlibs(L), luaL_dofile(L, "script.lua"). Stack-based communication: lua_pushnumber(L, 42), lua_getglobal(L, "myvar"). Calling Lua from C: lua_pcall(L, nargs, nresults, 0). Calling C from Lua: lua_register(L, "cfunc", c_function). LuaJIT: JIT compiler for Lua — 10-50x faster, FFI library for direct C binding.
