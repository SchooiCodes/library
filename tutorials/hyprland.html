📄 Hyprland Configuration Guide

Hyprland is a dynamic tiling Wayland compositor. This guide covers installation, basic configuration, and useful tweaks.

Installation

  1. On Arch‑based distros: pacman -S hyprland
  2. On Debian/Ubuntu: add the Hyprland PPA or build from source following the official docs.
  3. Ensure you have a Wayland‑compatible compositor and a compatible graphics driver.

Configuration Files

Create ~/.config/hypr/hyprland.conf and include the following sections:

# basic.conf
      monitor=eDP-1,1920x1080@60,0x0,1
      workspace=1
      exec=foot
      bind=SUPER,H,movefocus,left
      bind=SUPER,J,movefocus,down
      bind=SUPER,K,movefocus,up
      bind=SUPER,L,movefocus,right
      # optional: set wallpaper
      exec=~/.config/hypr/wallpaper.sh

Common Tweaks

  • Enable animations = yes for smoother transitions.
  • Use exec-once = nm-tray to show network manager.
  • Install Waybar for a status bar and configure ~/.config/waybar/config.
  • For a compositor‑wide cursor theme, add cursor_theme = "Bibata-Modern-Ice" in the config.

Basic Configuration

Create ~/.config/hypr/hyprland.conf and add a minimal config:

# Example minimal hyprland.conf
exec = foot
monitor = eDP-1, 1920x1080@60, 0x0, 1
workspace = 1
bind = SUPER, H, movefocus, left
bind = SUPER, J, movefocus, down
bind = SUPER, K, movefocus, up
bind = SUPER, L, movefocus, right

Useful Tweaks

  • Enable animations = yes for smoother transitions.
  • Configure exec-once = nm-tray to have network manager in the tray.
  • Set up a status bar like waybar for workspace and system info.
