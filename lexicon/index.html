Tech Lexicon

100+ tech slang terms, acronyms, and jargon — organized by category. Click any term to see its definition.

General Tech

API
Application Programming Interface — a set of rules for interacting with software
CLI
Command-Line Interface — text-based interface for running commands
CRUD
Create, Read, Update, Delete — the four basic operations for persistent storage
DRY
Don't Repeat Yourself — a principle that aims to reduce repetition in code
GUI
Graphical User Interface — visual interface with windows, icons, and buttons
IDE
Integrated Development Environment — a software suite for writing and testing code
SDK
Software Development Kit — a collection of tools for building software on a platform
TLD
Top-Level Domain — the last segment of a domain name (e.g. .com, .org, .io)
WYSIWYG
What You See Is What You Get — editors where content appears as it will in the final output
YAML
YAML Ain't Markup Language — a human-readable data serialization format
ASCII
American Standard Code for Information Interchange — a character encoding standard
FOSS
Free and Open-Source Software — software with publicly accessible source code
RFC
Request for Comments — a publication by the IETF defining internet standards

Programming

Abstraction
Hiding complex implementation details behind a simple interface
Async
Asynchronous programming — non-blocking execution that allows concurrent operations
Callback
A function passed as an argument to another function to be executed later
Closure
A function that retains access to its outer scope even after the outer function returns
Compiler
A program that translates source code into machine code or bytecode
Dependency
An external library or module that a project requires to function
EOF
End of File — a marker indicating no more data can be read from a file or stream
Framework
A pre-built structure of code that provides common functionality for applications
Garbage Collection
Automatic memory management that reclaims unused memory at runtime
Interpreter
A program that executes source code line by line without prior compilation
JSON
JavaScript Object Notation — a lightweight data interchange format
Linter
A tool that analyzes code for potential errors, bugs, and style violations
Namespace
A container for identifiers that prevents naming conflicts in code
OOP
Object-Oriented Programming — a paradigm based on objects containing data and methods
Polymorphism
The ability of different objects to respond to the same method call in different ways
Recursion
A technique where a function calls itself to break a problem into smaller subproblems
Refactoring
Restructuring existing code without changing its external behavior
Regex
Regular Expression — a pattern used to match character combinations in strings
REPL
Read-Eval-Print Loop — an interactive shell for executing code in real time
Serialization
Converting data structures into a format that can be stored or transmitted
Syntax
The set of rules defining how code must be structured in a language
Type Coercion
Implicit or explicit conversion of a value from one data type to another
Unicode
A universal character encoding standard that supports most writing systems
Variable
A named container that stores a value in memory for use in a program

Web

AJAX
Asynchronous JavaScript and XML — technique for updating parts of a page without reloading
CORS
Cross-Origin Resource Sharing — a security mechanism that controls cross-domain requests
DOM
Document Object Model — a programming interface for HTML and XML documents
HTTPS
HyperText Transfer Protocol Secure — encrypted HTTP over TLS
REST
Representational State Transfer — an architectural style for designing networked APIs
SEO
Search Engine Optimization — practices to improve a site's visibility in search results
SPOF
Single Point of Failure — a component whose failure brings down the whole system
SSR
Server-Side Rendering — generating HTML on the server rather than in the browser
CSR
Client-Side Rendering — rendering pages in the browser using JavaScript
PWA
Progressive Web App — a web app that behaves like a native app with offline support
SPA
Single Page Application — a web app that loads one page and updates content dynamically
WebSocket
A protocol providing full-duplex communication channels over a single TCP connection
Lazy Loading
Delaying the loading of resources until they are needed to improve performance
TTFB
Time to First Byte — the time between a request and the first byte of the response
Web Vitals
A set of metrics for measuring user experience on the web (LCP, FID, CLS)

Security

2FA
Two-Factor Authentication — a second layer of security beyond just a password
Backdoor
A hidden way to bypass normal authentication in a system
Botnet
A network of compromised computers controlled remotely for malicious purposes
CVE
Common Vulnerabilities and Exposures — a catalog of publicly known security flaws
DDoS
Distributed Denial of Service — overwhelming a server with traffic to make it unavailable
E2EE
End-to-End Encryption — encryption where only the communicating parties can read messages
MITM
Man-in-the-Middle — an attack where the attacker secretly intercepts communications
RCE
Remote Code Execution — a vulnerability allowing execution of code on a remote system
SOC
Security Operations Center — a team that monitors and responds to security incidents
XSS
Cross-Site Scripting — injecting malicious scripts into otherwise trusted websites
CSRF
Cross-Site Request Forgery — tricking a user into performing unwanted actions on a site
SQL Injection
Injecting SQL queries through input fields to manipulate a database
Zero-Day
A vulnerability unknown to the vendor with no available fix
OWASP
Open Web Application Security Project — a nonprofit for improving software security
JWT
JSON Web Token — a compact token format for securely transmitting claims between parties
OAuth
Open Authorization — an open standard for token-based authentication and authorization
HSTS
HTTP Strict Transport Security — forces browsers to connect over HTTPS only
SSRF
Server-Side Request Forgery — exploiting a server to make requests to internal resources

Networking

CDN
Content Delivery Network — a distributed network of servers that deliver content closer to users
DHCP
Dynamic Host Configuration Protocol — automatically assigns IP addresses on a network
DNS
Domain Name System — translates human-readable domain names into IP addresses
LAN
Local Area Network — a network connecting devices within a limited area
NAT
Network Address Translation — maps private IPs to a public IP for internet access
TCP
Transmission Control Protocol — a reliable, connection-oriented transport protocol
UDP
User Datagram Protocol — a fast, connectionless transport protocol without reliability guarantees
VPN
Virtual Private Network — encrypts traffic and routes it through a remote server
ARP
Address Resolution Protocol — maps IP addresses to MAC addresses on a local network
CIDR
Classless Inter-Domain Routing — a method for allocating IP addresses and routing
ICMP
Internet Control Message Protocol — used for diagnostics (ping, traceroute)
MTU
Maximum Transmission Unit — the largest packet size that can be transmitted over a network
OSI
Open Systems Interconnection — a conceptual model for network communication (7 layers)
SSL
Secure Sockets Layer — a deprecated cryptographic protocol superseded by TLS
TLS
Transport Layer Security — the modern cryptographic protocol for secure communication
TTL
Time to Live — a limit on the lifespan of data in a network or cache
WAN
Wide Area Network — a network that spans a large geographical area

Linux / CLI

chroot
Changes the root directory for a process, creating an isolated filesystem environment
cron
A time-based job scheduler in Unix-like systems for running tasks at intervals
daemon
A background process that runs continuously and handles service requests
FIFO
First In, First Out — a named pipe used for inter-process communication
grep
A command-line utility for searching text using regular expression patterns
kernel
The core of an operating system that manages hardware and system resources
pipe
A mechanism for passing output of one command as input to another (using |)
symlink
Symbolic link — a special file that points to another file or directory
AWK
A scripting language for pattern scanning and text processing
Bash
Bourne Again SHell — the default command shell on most Linux distributions
chmod
Changes file permissions (read, write, execute) for owner, group, and others
cURL
A command-line tool for transferring data with URL syntax
fork
A system call that creates a new process by duplicating the calling process
inode
A data structure that stores metadata about a file on a Unix filesystem
nohup
Runs a command immune to hangups, allowing it to continue after logout
PATH
An environment variable listing directories where executables are searched for
rsync
A fast file-syncing and transfer tool that copies only differences
shebang
The #! sequence at the start of a script indicating the interpreter to use
SSH
Secure Shell — a cryptographic network protocol for secure remote access
sudo
Superuser Do — allows executing commands with elevated privileges
systemd
A system and service manager for Linux, used as the init system
tar
Tape Archive — a utility for bundling files into a single archive file
tty
Teletypewriter — a terminal device for text input and output
xargs
Builds and executes command lines from standard input
zsh
Z Shell — an extended Bourne shell with advanced features and plugin support

Hardware

BIOS
Basic Input/Output System — firmware that initializes hardware during boot
DDR
Double Data Rate — a type of memory that transfers data on both clock edges
GPU
Graphics Processing Unit — a processor optimized for rendering graphics
I/O
Input/Output — communication between a system and external devices
LED
Light-Emitting Diode — a semiconductor light source used as indicators
NIC
Network Interface Card — hardware that connects a computer to a network
NVMe
Non-Volatile Memory Express — a fast protocol for SSDs over PCIe
PCIe
Peripheral Component Interconnect Express — a high-speed expansion bus standard
PSU
Power Supply Unit — converts AC power to DC power for a computer's components
SATA
Serial ATA — an interface for connecting storage devices like HDDs and SSDs
SSD
Solid-State Drive — a storage device using flash memory with no moving parts
CPU
Central Processing Unit — the primary processor that executes instructions
DRAM
Dynamic Random Access Memory — volatile memory used as main system RAM
ECC
Error-Correcting Code — memory that can detect and correct data corruption
HDD
Hard Disk Drive — a storage device with spinning magnetic platters
HDMI
High-Definition Multimedia Interface — a digital video and audio interface
RAM
Random Access Memory — fast volatile memory for active data and applications
ROM
Read-Only Memory — non-volatile memory that stores firmware
UEFI
Unified Extensible Firmware Interface — modern replacement for BIOS
USB
Universal Serial Bus — a standard interface for connecting peripherals
VRAM
Video RAM — memory dedicated to storing graphics data for the GPU

Gaming

FPS
Frames Per Second — measures how many frames are rendered each second
GG
Good Game — a sportsmanlike phrase said at the end of a match
Hitbox
The invisible collision area around a character that determines if hits land
Lag
A delay between a player's action and the game's response due to network issues
Ping
The round-trip time for data to travel from a client to a server and back
RNG
Random Number Generator — used for randomized outcomes like loot drops
Skins
Cosmetic items that change the appearance of characters, weapons, or items
Spawn
The point where a player or entity appears in the game world
AFK
Away From Keyboard — a player who is not actively playing
Buff
An improvement to a character, weapon, or ability to make it stronger
Nerf
A reduction in the power of a character, weapon, or ability for balance
DLC
Downloadable Content — additional content released after a game's launch
FOV
Field of View — the extent of the observable game world visible at any moment
Glitch
A temporary bug or error in a game that causes unexpected behavior
HP
Hit Points / Health Points — a measure of how much damage a character can take
HUD
Heads-Up Display — the on-screen interface showing health, ammo, map, etc.
Latency
The time delay between a player's action and the server's acknowledgment
Meta
Most Effective Tactic Available — the optimal strategies in a game
MMO
Massively Multiplayer Online — games with large numbers of players in the same world
NPC
Non-Player Character — characters controlled by the game, not by players
PvP
Player versus Player — combat between human players
PvE
Player versus Environment — combat against AI-controlled enemies
XP
Experience Points — points earned for progressing and leveling up
