PC Gaming Optimization

  • Update drivers — Keep GPU, chipset, and audio drivers up to date
  • Game Mode — Enable Windows Game Mode for background process optimization
  • Disable overlays — Turn off Discord, Xbox, and Steam overlays for FPS gains
  • Lower graphics settings — Reduce shadow quality, anti-aliasing, and draw distance
  • Close background apps — Kill unnecessary processes before launching games
  • Use performance power plan — Set Windows to "High Performance" or "Ultimate Performance"
