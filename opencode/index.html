Introduction

What is OpenCode?

OpenCode is an open-source, provider-agnostic AI coding agent that lives in your terminal (or IDE, or desktop). It understands natural language and can perform complex software engineering tasks — reading and writing files, running bash commands, searching codebases, browsing the web, and even delegating work to sub-agents.

Think of it as an AI pair programmer that never sleeps. You type what you need in plain English (or 20+ other languages), and OpenCode figures out the rest — creating entire projects, fixing bugs, refactoring code, writing tests, and deploying applications.

Open source (MIT license)
75+ LLM providers supported
Provider-agnostic — bring your own model
Runs in terminal, IDE, or desktop app
LSP-enabled for smart code context
Multi-session parallel agents
Capabilities

Key Features

Interactive Chat

Terminal Chat

Conversational interface right in your terminal. Ask questions, request changes, and get instant responses with code diffs and explanations.

File Operations

Create, read, write, edit, and delete files. OpenCode handles exact string replacements, multi-file edits, and preserves your code style.

Command Execution

Run bash commands directly — install packages, run scripts, start dev servers, execute tests. OpenCode sees output and reacts accordingly.

Code Search

Grep and glob searches across your codebase. Find functions, classes, patterns, and files instantly without leaving the conversation.

Web & External Access

Web Search

Search the web and fetch content from URLs. OpenCode can research APIs, documentation, and current events to inform its work.

Web Fetch

Download and read web pages, documentation, or any URL. Extracts content as markdown for easy reference and incorporation.

Git Integration

Commit, branch, push, and manage repositories. OpenCode understands git workflows and can handle version control tasks.

MCP Servers

Connect to Model Context Protocol servers for database access, API integrations, and custom toolchains.

Agent System

Multiple Agent Types

Use the build agent for full development tasks, the plan agent for read-only analysis, and @general for complex multi-step searches.

Task Delegation

OpenCode can delegate work to sub-agents for parallel execution. Complex tasks are broken down and distributed automatically.

Todo Management

Built-in todo list tracking. OpenCode maintains a running checklist of tasks and marks them complete as work progresses.

Custom Skills & Agents

Create custom skills and specialized sub-agents for recurring tasks. Teach OpenCode your workflows once and reuse them.

Setup

Installation & Quick Start

One-Liner Install Recommended

Install OpenCode with a single command (macOS, Linux, Windows WSL):

curl -fsSL https://opencode.ai/install | bash

Package Managers

Install via your preferred package manager:

# npm (all platforms)
npm i -g opencode-ai@latest

# Homebrew (macOS & Linux — recommended)
brew install anomalyco/tap/opencode

# Scoop (Windows)
scoop install opencode

# Arch Linux
sudo pacman -S opencode

# Any OS via mise
mise use -g opencode

Basic Usage

Once installed, starting a session is as simple as:

# Open a session in the current directory
opencode

# Start with a specific task
opencode "Create a new React component"

# Use a different agent
opencode --agent plan

# List available models
opencode models list

Switch between agents with the Tab key during a session. Toggle dark mode with Ctrl+D.

Configuration

OpenCode is configured via opencode.json or opencode.jsonc in your project root, or globally at ~/.config/opencode/. You can configure:

AI Provider — Choose from 75+ LLM providers
Agent Settings — Custom agent behavior, permissions, and system prompts
Custom Skills — Define reusable skill workflows for OpenCode
MCP Servers — Connect external tools and APIs
Themes — Customize terminal colors and appearance
Permissions — Set file access and command execution rules
