Stremio + Torrentio — The Ultimate Streaming Setup

Transform Stremio into a Netflix-like experience with Torrentio and Real-Debrid. Watch any movie or show instantly with premium cached streams.

What is Stremio?

Stremio is a modern media center application that organizes movies, TV shows, series, live TV, and web channels into a beautiful, Netflix-like interface. It is cross-platform and uses addons to fetch content from various sources.

  • Addon-based: All content comes from addons. The core app is just a player and organizer.
  • Cross-platform: Windows, macOS, Linux, Android, Android TV, and web.
  • Library management: Add movies and shows to your library. Track what you’ve watched.

Installing Stremio

  1. Go to stremio.com and download for your platform.
  2. Run the installer. On Windows, standard installer. On macOS, drag to Applications.
  3. Create a free account or log in (required for library sync across devices).
  4. The app opens with pre-installed addons like Cinemeta for metadata.

Torrentio Addon

Torrentio is the essential addon that scrapes hundreds of torrent sources and provides streaming links for any movie or show:

  1. Open a browser and go to torrentio.strem.fun.
  2. Configure: quality filter, providers, and debrid service API key.
  3. Click "Install" and Stremio will open automatically.
  4. Browse any title and you’ll see available sources sorted by quality.

Real-Debrid Integration

Without Real-Debrid, Torrentio shows public torrent links. With Real-Debrid, you get instant premium cached streams:

  • [RD+] tags indicate Real-Debrid cached streams that play instantly.
  • How caching works: When someone requests a torrent via Real-Debrid, it downloads to their servers. Subsequent requests stream from cache.
  • No VPN needed: Your traffic goes through Real-Debrid’s servers.

Other Addons

  • CinemaSaur: Alternative scraper addon when Torrentio doesn’t find sources.
  • WatchHub: Shows where a title is available across different addons.
  • Trakt: Syncs your watch history and lists with Trakt.tv.
  • Subtitles: OpenSubtitles and Podnapisi addons for subtitles.
  • Pluto TV / Live TV: Addons for live TV channels.

Tips

  • Use Real-Debrid exclusively: It’s $3/month and transforms Stremio into a premium streaming service.
  • Quality filtering: Filter to 720p if you have a slow connection, 4K if fast.
  • Android TV / Fire Stick: Stremio works great. Configure addons on a PC and they sync.
  • Backup your config: Bookmark your Torrentio configuration URL.

Stremio Installation

Stremio is available on all major platforms. Here is how to install it on each:

  • Windows: Download the installer from stremio.com. Run it. The standard installer includes the player and all dependencies. Launch Stremio and create a free account.
  • macOS: Download the .dmg from stremio.com. Drag Stremio to your Applications folder. If macOS blocks the app, go to System Settings → Privacy & Security and click "Open Anyway."
  • Linux: Download the AppImage from stremio.com. Make it executable (chmod +x Stremio*.AppImage) and run it. Alternatively, use Flatpak: flatpak install stremio.
  • Android: Install from the Google Play Store or download the APK directly from stremio.com. Enable "Install from unknown sources" if sideloading the APK.
  • Android TV: Install from the Google Play Store on your TV device. If not available, sideload the mobile APK (use a file manager app to install it).
  • iOS/iPadOS: Stremio is available as a web app at app.strem.io on iOS. There is no native iOS app due to Apple restrictions.

Torrentio Addon Setup

Torrentio is the essential addon that powers Stremio for pirated content. Here is how to install and configure it:

  1. Open Torrentio config: Go to torrentio.strem.fun in a browser. Do this on any device with a browser — the configuration syncs to all your Stremio installations.
  2. Configure providers:
    • Debrid provider: Select Real-Debrid, AllDebrid, or Premiumize. Enter your API key from the debrid service’s account page.
    • Max results per resolution: Default is 5. Increase to 10-15 for more choices, decrease to 2-3 for faster loading.
    • Sort by: Choose between quality, seeders, or source size.
  3. Quality filters:
    • Check the resolutions you want: 4K, 1080p, 720p, 480p.
    • Uncheck lower resolutions to reduce clutter.
    • Set preferred language (English, Multi, etc.) to filter out foreign-language releases.
  4. Click Install: After configuring, click "Install." Stremio will open and prompt you to add the addon. If Stremio doesn’t open automatically, copy the install URL and paste it into Stremio’s addon installer.
# Torrentio configuration URL format:
# https://torrentio.strem.fun/configure
# Save this URL after configuring:
# https://torrentio.strem.fun/{YOUR_BASE64_CONFIG}/manifest.json
# Bookmark it for future backup and reinstallation

Real-Debrid Integration (In Detail)

Without Real-Debrid, Torrentio shows public torrent links that stream directly from P2P peers. This is slow, unreliable, and exposes your IP. With Real-Debrid, the experience transforms:

  • [RD+] tags indicate Real-Debrid cached streams that play instantly. These are the highest priority and should always be chosen if available.
  • [RD] tags (without +) indicate uncached torrents. When you select one, Real-Debrid begins downloading the torrent to its servers. Playback starts once enough data is cached (usually 1-5 minutes for a 2-hour movie).
  • How caching works: When any Real-Debrid user requests a torrent, it downloads to Real-Debrid’s servers. Once complete, the torrent is cached and available instantly to all other users. Popular content is almost always cached.
  • No VPN needed: Your traffic goes through Real-Debrid’s servers. Your IP never contacts the torrent swarm. You do not need a VPN when streaming through Real-Debrid.
  • Multiple devices: You can use the same Real-Debrid account on all your Stremio instances, as long as they share the same public IP (home network). For remote access, set up a VPN to share the same IP.

Configuring Quality Filters

Fine-tuning quality filters makes a huge difference in your streaming experience:

  • 4K (2160p): Requires ~50+ Mbps stable internet. Files are 30-80 GB. Best for 65"+ 4K TVs. Torrentio will prioritize HDR and Dolby Vision when available.
  • 1080p: The sweet spot. Requires ~15-25 Mbps. Files are 5-15 GB. Great for most users on most devices.
  • 720p: Good for slower connections (~5-10 Mbps). Files are 1-3 GB. Fine for phones and tablets.
  • 480p: For very slow connections or mobile data. Files are 500 MB - 1 GB.
  • Language filter: Set to "English" or "Multi" to exclude releases in foreign languages. If you watch content with many languages (anime, foreign films), set to "All."

Subtitle Addons

Good subtitles are essential for many users. Here are the best subtitle addons for Stremio:

  • OpenSubtitles: The largest subtitle database. Configure with your OpenSubtitles account for full access. Millions of subtitles in dozens of languages.
  • Podnapisi.NET: Backup subtitle source. Smaller database but sometimes has subtitles that OpenSubtitles doesn’t.
  • SubDL: Another subtitle provider with good coverage for TV shows. Free to use without account.
  • Integration: Install these addons just like Torrentio. In the Stremio addons page, search for the addon name and click install. Subtitles appear automatically in the player.

Catalog Addons

Catalog addons provide content browsing within Stremio. Here are the best ones:

  • Cinemeta (default): Pre-installed with Stremio. Provides metadata, posters, and descriptions. Not a streaming source — just a catalog display.
  • Trakt TV: Syncs your watchlist, history, and lists from Trakt.tv. Shows your personalized recommendations. Supports scrobbling (automatic tracking of what you watch).
  • IMDB Lists: Browse any public IMDB list (Top 250, Popular, Trending, Oscars, custom user lists). Items from the list appear in your Stremio catalog.
  • TMDB (The Movie Database): Alternative catalog addon with richer metadata than Cinemeta. Includes actor bios, production details, and community ratings.
  • YouTube: Browse YouTube channels and playlists within Stremio.

Mobile Setup

Setting up Stremio on mobile is slightly different from desktop:

  • Android: Install Stremio from the Play Store. Log in with your account. All addons sync automatically from your desktop configuration. Stream directly or download for offline viewing (choose "Download" from the source selection screen).
  • iOS/iPadOS: Use the web app at app.strem.io. Add to your home screen for a near-native experience. The web app supports streaming but not offline downloads. Casting to Chromecast/AirPlay works.
  • Fire Tablet: Install the Android APK from stremio.com (enable "Apps from Unknown Sources" in settings). Works the same as the Android version.

Android TV Setup

Stremio on a TV provides a full theater-like experience:

  1. Install: Open Google Play Store on your Android TV device and search for Stremio. Install it. If not available, sideload the Android APK.
  2. Log in: Sign in with your Stremio account. All your addons and library sync automatically.
  3. Navigation: Use the remote to browse. The TV interface is optimized for large screens. Long-press on a title to see more options (trailer, cast, similar titles).
  4. Playback: Stremio supports hardware acceleration on most Android TV devices. ExoPlayer is used for playback. If you experience buffering, try selecting a lower quality source.
  5. Chromecast / Fire Stick tips:
    • Fire Stick 4K Max: Best performance for 4K streaming.
    • Chromecast with Google TV: Good performance, supports Dolby Vision.
    • NVIDIA Shield TV: Best overall, supports all codecs and TrueHD/DTS-HD passthrough.

Multiple Devices Sync

Stremio syncs across all your devices using your free account:

  • What syncs: Library (movies/shows you’ve added), watch history, continue watching progress, and addon configurations.
  • How it works: Your account stores a manifest of all installed addons. When you log in on a new device, Stremio downloads and installs the same addons. The library and history are stored server-side.
  • What doesn’t sync: Player settings, download locations, and local cache. You may need to reconfigure your video player settings on each device.
  • Cross-platform example: Start watching a movie on your desktop. Pause it. Open Stremio on your Android TV. The movie appears in "Continue Watching" and resumes from where you left off.

Backup and Restore

Preserve your configuration in case you need to reinstall or switch devices:

  • Torrentio config URL: After configuring Torrentio, the URL contains your entire configuration (including your Real-Debrid API key). Bookmark this URL. If you need to reinstall, visit the URL and click Install again.
  • Account backup: Your Stremio account holds your library, history, and addon manifest. As long as you remember your email and password, your data is safe.
  • Manual addon list: Take a screenshot or note of your installed addons (go to Addons page in Stremio). If something goes wrong, you can reinstall them manually.
  • Restore process: Log into your Stremio account on the new device. Go to Addons and reinstall Torrentio (use your saved config URL). Other addons (subtitles, Trakt, catalogs) can be reinstalled from the addons catalog.
# Backup checklist:
# 1. Save Torrentio config URL (contains RD API key)
# 2. Note your Stremio account email
# 3. Screenshot installed addons list
# 4. Save any custom addon configs (Trakt auth, etc.)
# Store all of these in a password manager for easy recovery.

Alternative Addons

While Torrentio is the most popular scraping addon, others exist with different features:

Addon Type Strengths

CometScraper (P2P)Decentralized, uses BitTorrent protocol directly. No debrid needed but no caching.

Troubleshooting Buffering

Buffering is the most common issue. Here is how to fix it:

  • Check your internet speed: Run a speed test. You need at least 15 Mbps for 1080p, 50 Mbps for 4K. If your speed is lower, select a lower quality source.
  • Choose cached sources: Always select [RD+] sources over [RD] or non-debrid sources. Cached sources stream from Real-Debrid’s fast servers. Non-cached sources depend on the torrent’s seeders.
  • Hardware acceleration: In Stremio Settings → Player, enable hardware acceleration. This offloads video decoding to your GPU, reducing CPU usage and improving playback smoothness.
  • Try a different player: In Settings → Player, switch between ExoPlayer (default) and LibVLC. ExoPlayer is better for most modern devices. LibVLC handles older codecs better.
  • Disable IPv6: Some networks have broken IPv6. Go to Settings → Advanced and disable IPv6. This forces all traffic through IPv4, which may be more stable.
  • Clear cache: Go to Settings → Advanced → Clear Cache. This removes cached metadata but not your library data. A corrupted cache can cause buffering issues.
  • Network hardware: If streaming on Wi-Fi, try moving closer to the router or using a wired Ethernet connection. Powerline adapters can also help.
Stuttering or audio sync issues: Usually a codec problem. Try switching to a different source (different encode/group). If the problem persists across all sources, reinstall Stremio or try the VLC Player integration.

Custom Addon Configuration

For advanced users, custom addon configuration gives you more control:

  • Self-hosted addons: Run your own Torrentio instance locally or on a server. This gives you full control over providers, caching, and privacy. Use Docker: docker run -d -p 8080:8080 ghcr.io/torrentio/torrentio.
  • Custom addon development: Stremio addons are simple JavaScript/web apps. You can create custom addons for specific content sources or personal use. The Stremio addon SDK is available on GitHub.
  • Community addons: Browse the community addon list at stremio-addons.net for niche addons (documentaries, specific countries, retro games, etc.).
