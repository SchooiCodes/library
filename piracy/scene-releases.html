Scene Release Groups & Standards

The inner workings of the warez scene: release naming conventions, types, groups, P2P vs Scene, and how to find scene releases.

What is the Scene?

The “Scene” refers to a clandestine, organized community of groups that release pirated media (movies, TV, music, software, games) before anyone else. The Scene has strict rules about how releases are named, packaged, and distributed.

  • Not P2P: The Scene uses private FTP servers, not public torrents. Scene releases trickle down to the public via pre databases and torrent trackers.
  • Competition: Groups compete to be the first (“pre”) to release a given title.
  • Rules: Strict standards for encoding, NFO files, and packaging. Breaking rules gets a group banned.

Release Naming

Every Scene release follows a strict naming convention that encodes all metadata:

# Movie: Movie.Name.2024.1080p.BluRay.x264-GROUPNAME
# TV:    Show.Name.S01E05.1080p.WEB-DL.DDP5.1.H.264-GROUP
# Game:  Game.Name-REPACKNAME
# Music: Artist.Name-2024-Album-Name-FLAC-GROUP
#
# Key components:
# - Resolution: 720p, 1080p, 2160p (4K)
# - Source: BluRay, WEB-DL, WEBRip, HDTV, DVDRip
# - Codec: x264, x265, H.264, H.265, AV1
# - Audio: AAC, DTS, DDP5.1, FLAC, MP3
# - GROUP: The releasing group abbreviation

Groups

  • Movies: SPARTA, ViP3R, FGT, BOND, KOGi, ION10
  • TV: ELEANOR, SYS, KILLERS, CRONUS, BAMBOOZLE
  • Music: BABAS, DANDIS, FLAWLESS, SAUCE, COCAINE
  • Software: CORE, ASSiGN, DiDA, DEViANCE, ZWT
  • Games: PLAZA, CODEX, EMPRESS, HOODLUM, RELOADED, CPY

Finding Scene Releases

  • PreDB / predb.me — Track releases as they happen. Search release names.
  • r/CrackWatch — Tracks game cracks and scene releases.
  • Private trackers — Many distinguish Scene releases with special tags.
  • DDL sites — Many DDL aggregators specialize in Scene releases.
