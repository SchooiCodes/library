Media Servers for Piracy

Plex, Jellyfin, Sonarr/Radarr, Prowlarr, Overseerr, Usenet/Torrent integration for the ultimate media setup.

Plex Setup

Plex organizes your media library and streams it to any device. Install Plex Media Server on a dedicated machine or NAS. Organize media in proper folder structures (Movies/, TV Shows/). Access from phones, TVs, and browsers. Plexamp for music.

Jellyfin Setup

Jellyfin is the open-source alternative to Plex. No paid features, no telemetry, full customization. Supports all media types, hardware transcoding, live TV, and plugin system. Docker deployment is recommended.

Sonarr/Radarr

Sonarr (TV) and Radarr (Movies) automate media acquisition. Add content, set quality profiles, and they automatically search indexers, download, rename, and organize your media. Connect to Prowlarr for indexer management.

Prowlarr

Prowlarr is a centralized indexer manager. Add your torrent and Usenet indexers once, and sync them to Sonarr, Radarr, Lidarr, Readarr, and other Arr apps. Supports hundreds of indexers and automatic updates.

Overseerr

Overseerr provides a user-friendly web interface for requesting media. Users browse, search, and request movies and TV shows. Integrates with Sonarr/Radarr for automatic processing. Plex authentication for user management.

Usenet/Torrent Integration

Sonarr/Radarr support both Usenet and Torrent clients simultaneously. Configure SABnzbd for Usenet and qBittorrent for torrents. Content is automatically routed through the best available source.

# Docker Compose media stack (core):
version: "3"
services:
  sonarr:
    image: linuxserver/sonarr
    ports: ["8989:8989"]
  radarr:
    image: linuxserver/radarr
    ports: ["7878:7878"]
  prowlarr:
    image: linuxserver/prowlarr
    ports: ["9696:9696"]
  overseerr:
    image: linuxserver/overseerr
    ports: ["5055:5055"]
