Office Activation via Microsoft Activation Scripts

A comprehensive technical guide to downloading Microsoft Office and using Microsoft Activation Scripts (MAS) for educational activation research.

Important Legal Notice

Microsoft Activation Scripts (MAS) is an open-source project hosted on GitHub. It does not contain malware and its source code is publicly auditable. However, using MAS to activate Microsoft software without a valid license violates Microsoft's Terms of Service and may violate copyright law. This guide exists for archival, educational, and research purposes. The authors of this site do not condone software piracy.

What is MAS?

Microsoft Activation Scripts (MAS) is an open-source collection of PowerShell scripts for researching Microsoft software activation mechanisms. It is hosted on GitHub under the repository massgravel/Microsoft-Activation-Scripts and has been publicly available since 2020. The project is widely studied by security researchers, system administrators, and software engineers interested in understanding Microsoft's activation technology.

MAS supports three primary activation methods:

  • Ohook — A local hook-based method that permanently activates Office by intercepting activation checks at the library level. Works offline, no KMS server required. Supports Office 2016 through Office 2024.
  • HWID (Digital License) — A hardware-bound digital license method for Windows 10/11 that generates a genuine Ticket-based activation tied to the system's hardware ID.
  • KMS (Key Management Service) — Emulates a corporate KMS server locally to activate volume license editions of Windows and Office for 180-day intervals (renewable).

The scripts are written entirely in PowerShell and batch, making their operation transparent to anyone who reads the source. Because MAS is open-source, the community can inspect every line for malicious behavior — which has contributed to its reputation as a trustworthy tool in the activation research space.

Safety Note: Always download MAS from the official GitHub repository (github.com/massgravel/Microsoft-Activation-Scripts). Third-party mirrors or re-uploads may contain malware. Verify the SHA-256 hash against the official release page before running any script.

Downloading Microsoft Office

Microsoft Office can be downloaded using the Office Deployment Tool (ODT), which is the official tool Microsoft provides for enterprise volume license deployments. Using ODT is the safest and most reliable method — it downloads directly from Microsoft's CDN, guaranteeing authentic, unmodified files.

Step 1: Download the Office Deployment Tool

# Download ODT from Microsoft's official site
# https://www.microsoft.com/en-us/download/details.aspx?id=49117

# Or via direct download link (may change — check official page)
curl -o officedeploymenttool.exe "https://download.microsoft.com/download/..."

After downloading, run the ODT executable to extract the setup.exe and configuration.xml files to a folder (e.g., C:\ODT).

Step 2: Create a Configuration XML

The configuration.xml file tells ODT which Office products, languages, and settings to download and install. Below is an example for Office Professional Plus 2024 with core applications:

<Configuration>
  <Add OfficeClientEdition="64" Channel="PerpetualVL2024">
    <Product ID="ProPlus2024Volume">
      <Language ID="en-us" />
      <ExcludeApp ID="Access" />
      <ExcludeApp ID="Publisher" />
      <ExcludeApp ID="SkypeForBusiness" />
    </Product>
  </Add>
  <Display Level="None" AcceptEULA="TRUE" />
  <Property Name="SharedComputerLicensing" Value="0" />
</Configuration>

For Office 2021, use Channel="PerpetualVL2021" and Product ID="ProPlus2021Volume". For Office 2019, use PerpetualVL2019.

Step 3: Download Office

# Download Office files to C:\ODT\Office
.\setup.exe /download configuration.xml

This downloads all Office installation files to the local machine. The download is typically 2-4 GB depending on the products and languages selected. The command produces no output if it succeeds — check C:\ODT\Office to verify files were downloaded.

Alternative Download Methods: Some research communities provide pre-packaged Office ISOs via MEGA or Google Drive. These carry additional security risks (modified installers) and should be scanned with multiple antivirus engines and checksum-verified before use.

Installing Office via ODT

Once the Office files are downloaded, installation is a single command:

# Install Office using the same configuration.xml
.\setup.exe /configure configuration.xml

This launches the silent installation. The process takes 5-15 minutes depending on your system. You can monitor progress via Task Manager (look for Setup processes consuming CPU and disk I/O).

Post-Installation Checks

After installation completes, verify Office is installed but note the activation status:

# Open a terminal (cmd or PowerShell) and run:
cd "C:\Program Files\Microsoft Office\Office16"

# Check license status
cscript ospp.vbs /dstatus

The output will show the license status as Licensed or Not Licensed. At this point, a volume license edition will show as unactivated (grace period). This is where activation research tools come into play.

Antivirus Warning: Some antivirus software may flag MAS scripts or Office activation tools as potentially unwanted programs (PUPs) or threats. This is a false positive caused by the nature of activation tools. Temporarily disabling real-time protection during activation is a common practice among researchers — but only do this if you understand the risks. Always scan downloaded files with multiple engines via VirusTotal first, and re-enable protection immediately after activation.

Downloading Microsoft Activation Scripts

MAS is distributed exclusively through its official GitHub repository. Download the latest release:

# Option 1: Download the ZIP from GitHub
# Visit: https://github.com/massgravel/Microsoft-Activation-Scripts
# Click "Releases" and download the latest MAS_*_AIO_*.zip

# Option 2: Via PowerShell (direct from GitHub)
$url = "https://github.com/massgravel/Microsoft-Activation-Scripts/archive/refs/heads/master.zip"
Invoke-WebRequest -Uri $url -OutFile "$env:TEMP\MAS.zip"
Expand-Archive -Path "$env:TEMP\MAS.zip" -DestinationPath "$env:TEMP\MAS"
cd "$env:TEMP\MAS\Microsoft-Activation-Scripts-master"

Verifying File Integrity

Before running any scripts, verify the integrity of your download:

# Check the SHA-256 hash against the official release page
Get-FileHash -Path "MAS.zip" -Algorithm SHA256

The official GitHub Releases page lists the expected SHA-256 hash for each release archive. Compare the hash you compute against the published value before extracting or running any scripts. If they do not match, do not run the scripts — the archive may have been tampered with.

Security Best Practice: Run MAS in an isolated virtual machine or sandbox if you are unsure about its behavior. While the MAS source code is publicly audited, running any activation tool carries inherent risks. For maximum safety, review the PowerShell scripts yourself before execution — they are plain text and fully readable.

Activation Methods

MAS provides several activation methods. For Office, the Ohook method is the most practical because it provides permanent, offline activation without requiring KMS server emulation or periodic renewal.

Ohook Method (Office — Recommended)

Ohook works by hooking the Office activation library at the system level, causing Office to believe it is permanently activated. It does not modify Office binaries — it adds a hook library that intercepts activation queries and returns a "licensed" response.

# Run MAS (right-click PowerShell as Administrator)
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\MAS_1.5_AIO.cmd

# From the menu:
# Select [3] Ohook
# Select [3] Install Ohook
# Confirm when prompted

The script runs in a terminal menu interface. Navigate using number keys. Ohook installation takes approximately 2-3 seconds. Once installed, Office will report as permanently activated.

Technical Note: Ohook works with Office 2016 through Office 2024 on Windows 10 and Windows 11. It survives Office updates but may need to be re-applied after major version upgrades (e.g., Office 2021 → Office 2024). To uninstall Ohook later, run the same script and select the uninstall option — this restores the original activation library.

KMS Method (Office & Windows)

The KMS method emulates a corporate Key Management Server locally. It activates volume-licensed editions for 180 days, with automatic renewal via a scheduled task:

# From the MAS menu:
# Select [1] HWID (for Windows)
# Select [2] KMS (for Windows/Office)

# For Office KMS activation specifically:
# Select [4] Activation
# Select [2] KMS Activation

KMS activation requires an internet connection for the initial activation (to reach the emulated KMS server). Once activated, a scheduled task renews the activation every 7 days automatically. The activation is valid for 180 days before it must be renewed — the auto-renewal task handles this transparently.

KMS vs Ohook: KMS activation requires periodic renewal (every 180 days) and a lightweight background service. Ohook is permanent and fully offline. For Office research, Ohook is the preferred method. For Windows, HWID (digital license) is the preferred method as it is also permanent and hardware-bound.

Verifying Activation

After applying the activation method, verify the result using official Microsoft tools:

# Verify Office activation status
cd "C:\Program Files\Microsoft Office\Office16"
cscript ospp.vbs /dstatus

# Expected output includes:
# ---Processing--------------------------
# PRODUCT ACTIVATION: PERMANENT
# REMAINING GRACE: 180 days (KMS) or N/A (Ohook)

# Alternative: Check in an Office app
# Open Word or Excel > File > Account
# Product Activated should appear

The /dstatus flag shows the detailed license status. A successfully activated Office installation will display PRODUCT ACTIVATION: PERMANENT for Ohook or show the KMS activation expiry date for the KMS method.

Common Issues

  • "The product key you entered didn't work" — Ensure you installed a Volume License edition of Office (ProPlus2024Volume, not ProPlus2024Retail). The retail edition requires different activation methods.
  • Ohook not working after Office update — Major Office updates may replace the hooked library. Re-run the Ohook install from MAS to restore it.
  • Antivirus blocking MAS — Temporarily disable real-time protection, apply the activation, then re-enable. Add the MAS folder as an exclusion if needed.
  • "The Software Licensing Service reported that the product is not found" — The Office volume license key may not be installed. Use the MAS menu option to install the generic volume license key (GVLK) before activating.

Safety & Legality

Legal Considerations: Using MAS or any other activation tool to activate Microsoft Office or Windows without a valid license is software piracy and violates Microsoft's Terms of Service. In many jurisdictions, this constitutes copyright infringement and may carry legal penalties. This guide is provided for educational and research purposes only. The authors strongly recommend purchasing a legitimate Office license from Microsoft or an authorized reseller.

Risks of Unlicensed Office

  • Malware risk: Downloading activation tools from untrusted sources is a common vector for malware infections. Always use the official MAS GitHub repository and verify checksums.
  • No updates: Volume-licensed editions of Office do not receive the same feature updates as retail/subscription editions. Security updates are still delivered for volume editions.
  • No support: Microsoft will not provide technical support for unlicensed software installations.
  • Enterprise detection: Organizations can detect unlicensed Office installations via licensing telemetry. This is primarily a concern for business or educational environments.

Legitimate Alternatives

  • Microsoft 365 Free (web-only): Use Office Online for free in a browser at office.com. Requires a Microsoft account but is completely free and legal.
  • Microsoft 365 Personal/Family: Subscription-based access with 1TB OneDrive storage, starting at ~$70/year. Often available through workplace or educational discounts.
  • Office Home & Student 2024: One-time purchase for ~$150. Includes Word, Excel, and PowerPoint with no subscription required.
  • Free alternatives: LibreOffice, OnlyOffice, Google Workspace, Apple iWork — all free and fully compatible with Microsoft Office file formats.

Final Note

This guide documents the technical process of Office volume license activation research. The techniques described are widely studied in system administration and security research communities. If you find Microsoft Office valuable for your work or studies, please support the developers by purchasing a legitimate license. The cost of Office is small compared to the productivity value it provides.
