Music Piracy Guide — Streaming, Downloading & Automation

Complete guide to music piracy: Soulseek, Deemix, Lucida, BlockTheSpot, Freezer, Telegram bots for music, and more.

Soulseek

Soulseek is the oldest and most reliable peer-to-peer music sharing network, operating since 2000. Essential for finding rare, obscure, and lossless music. Browse other users’ entire libraries. FLAC is the preferred format. Use Nicotine+ (open-source client).

Deemix / Deemirror

Deemix downloads music from Deezer at up to FLAC 1411kbps. Requires a Deezer account. Deemirror is an alternative that uses publicly available ARLs. Quality: 128kbps MP3 (free), 320kbps MP3, or FLAC (HiFi).

Lucida

Lucida is a modern music downloader that aggregates multiple sources (Deezer, Spotify, Qobuz, Tidal, YouTube Music). Falls back if one source fails. Self-hostable as a web service.

BlockTheSpot / SpotX

These tools block ads in the Spotify desktop client and unlock premium-like features:

  • BlockTheSpot: PowerShell script that modifies Spotify files on Windows.
  • SpotX: Modern replacement supporting Win/Mac/Linux. Blocks ads, enables shuffle.
# SpotX Installation:
# Windows (PowerShell Admin):
# iex "& { $(iwr -useb 'https://raw.githubusercontent.com/SpotX-Official/SpotX/main/Install.ps1') }"
#
# Linux/Mac:
# bash <(curl -sSL https://raw.githubusercontent.com/SpotX-Official/SpotX/main/Install.sh)

Freezer

Freezer is a mobile Android app that downloads music from Deezer at original quality. Works like a streaming music player with download capability. FLAC downloads with a Deezer HiFi account.

Telegram Bots

  • @DeezermusicBot / @deemixbot — Send artist + track/album name. Returns MP3 320kbps.
  • @SpotifyDownBot — Send Spotify link. Downloads from Deezer/Spotify.
  • @MusicHarborBot — Search by artist, album, or track.

Soulseek Deep Dive

Soulseek remains the most reliable source for rare, obscure, and lossless music. Unlike streaming services, Soulseek gives you direct access to other users’ entire music collections.

  • Setup: Download Nicotine+ (nicotine-plus.org), the modern open-source Soulseek client. Available for Windows, Mac, and Linux.
  • Search tips: Use quotes for exact matches ("Pink Floyd"). Filter by bitrate (FLAC, 320, V0). Use the "Browse" feature to explore a user’s full collection.
  • User ranks: Soulseek assigns reliability ratings. Users with higher ranks respond faster and have more complete collections. Privileged users (donors) have unrestricted search results.
  • Download queue: You can queue multiple files from multiple users. Downloads proceed in parallel based on user availability.
# Nicotine+ recommended settings:
# Edit → Preferences → Searches
#   Default search mode: "Search All"
#   Results per page: 200
# Edit → Preferences → Transfers
#   Download directory: /music/downloads
#   Max simultaneous downloads: 5
# Edit → Preferences → Sharing
#   Share your own music to maintain good rank

Lucida Guide

Lucida is a modern, self-hostable music downloader that aggregates multiple streaming services as sources, falling back automatically if one fails.

  • Supported sources: Deezer, Spotify, Qobuz, Tidal, YouTube Music, SoundCloud, Apple Music.
  • Quality options: Up to FLAC (1411kbps) from Deezer/Qobuz/Tidal. 320kbps MP3 from Spotify/YouTube Music.
  • Self-hosting: Deploy via Docker: 
    docker run -d -p 8080:8080 --name lucida lucida/lucida
  • Web UI: Search for any track, album, or artist. Select desired quality. Download directly or queue multiple downloads.
  • Streaming fallback: If a track is not available on Deezer, Lucida automatically tries Qobuz, then Tidal, then YouTube Music.

Deemix / Freezer for FLAC

For true lossless audio, Deemix (desktop) and Freezer (Android) are the best tools for downloading FLAC from Deezer.

  • Deemix: Python-based CLI and GUI tool. Requires a Deezer ARL (authentication token). Downloads at up to FLAC 1411kbps.
  • Freezer: Android app that streams and downloads from Deezer. Premium feature unlocks FLAC downloads. Built-in player, search, and playlist management.
  • ARL extraction: Extract your Deezer ARL cookie from browser dev tools after logging into Deezer. Use the Deemix settings page to enter it.
  • Quality comparison: Deezer FLAC is true 16-bit/44.1kHz CD quality. Deemix also supports MP3 128kbps (free tier) and 320kbps.
Note: Deezer HiFi subscription (~$10/month) is required for FLAC downloads. However, shared ARLs are available on the internet that work temporarily.

Spotify Mods

Several tools modify the official Spotify desktop client to unlock premium features and block ads:

  • BlockTheSpot: Windows-only PowerShell script. Patches Spotify to block audio ads, visual ads, and enable unlimited skipping. Simple install: run a single command in PowerShell as Admin.
  • SpotX: Cross-platform (Win/Mac/Linux). Modern replacement for BlockTheSpot. Blocks all ads, enables shuffle by default, removes podcast/video restrictions. Actively maintained.
  • Spicetify: A theming and customization tool for Spotify. Not just for blocking ads — it lets you customize the entire UI, add extensions (lyrics, better playlists, mini player), and integrate with third-party services.
# Spicetify installation:
# PowerShell (Windows):
# iwr -useb https://raw.githubusercontent.com/spicetify/cli/main/install.ps1 | iex
#
# Linux/Mac:
# curl -fsSL https://raw.githubusercontent.com/spicetify/cli/main/install.sh | sh
#
# After install: spicetify apply

YouTube Music Downloaders

YouTube Music has a massive catalog, but downloads require a Premium subscription. These tools bypass that:

  • yt-dlp: The gold standard for downloading from YouTube/YouTube Music. Supports playlists, channels, and individual tracks. Can extract audio only (best quality: opus 160kbps).
  • Spotube: Open-source Spotify client that plays music from YouTube Music. No account needed. Downloads for offline play.
  • PipePipe: Android app that downloads from YouTube Music, NicoNico, and Bilibili. Ad-free interface.
  • Seal: Android app wrapper around yt-dlp. Simple UI for downloading audio/video from YouTube Music.
# Download album from YouTube Music with yt-dlp:
# yt-dlp -x --audio-format opus --audio-quality 0 \
#   --embed-thumbnail --embed-metadata \
#   "https://music.youtube.com/playlist?list=..."

Tidal & Qobuz Downloaders

For high-resolution audio enthusiasts, Tidal and Qobuz offer Master Quality (MQA) and Hi-Res audio respectively.

  • tidal-dl: Command-line tool for downloading from Tidal. Supports up to Master quality (MQA). Requires Tidal account.
  • Qobuz-dl: Python CLI tool for Qobuz downloads. Supports up to Hi-Res 24-bit/192kHz FLAC. Requires Qobuz Sublime subscription.
  • OrpheusDL: Modular downloader supporting Tidal, Qobuz, Deezer, and others. Advanced features like cover art embedding, lyrics download, and tagging.
  • Streamrip: Unified CLI tool that supports Tidal, Qobuz, Deezer, and SoundCloud. Automatic quality selection and tagging.

Organizing with Lidarr

Lidarr is the music-focused member of the Arr family of automation tools. It monitors artists you follow, searches for new releases, and downloads them automatically.

  • Setup: Run Lidarr (Docker or native). Add your music library folder. Configure indexers (Headphones VIP, Gazelle trackers, RSS feeds).
  • Quality profiles: Define profiles for different quality levels: FLAC preferred, 320kbps acceptable, minimum bitrate thresholds.
  • Artist management: Add your favorite artists. Lidarr monitors for new album releases. When a new album drops, it searches configured indexers and sends to your download client.
  • Integration: Works with SABnzbd, qBittorrent, Deluge, Transmission. Can also import and rename existing music collections.
# Lidarr Docker setup:
# docker run -d --name lidarr \
#   -p 8686:8686 \
#   -v /path/to/config:/config \
#   -v /path/to/music:/music \
#   -v /path/to/downloads:/downloads \
#   linuxserver/lidarr

Tagging with MusicBrainz Picard

Proper metadata tagging is essential for a well-organized music library. MusicBrainz Picard is the best tool for this.

  • How it works: Picard analyzes audio files and generates acoustic fingerprints. It matches these against the MusicBrainz database, automatically adding correct artist, album, track, year, genre, and cover art.
  • Batch processing: Drop entire folders into Picard. It scans all files, matches them, and applies tags in bulk.
  • Cluster mode: Groups unknown files into likely albums based on acoustic analysis. Useful for untagged downloads.
  • Plugins: Extend Picard with plugins for additional tag sources, cover art providers, and export formats.

Spectrum Analysis for Quality Verification

Some downloads claim to be FLAC or high-bitrate but are actually transcoded from lossy sources. Spectrum analysis reveals the truth.

  • Spek: Simple spectrum analyzer. Shows the frequency spectrum of an audio file. Genuine FLAC has content up to 22kHz. Lossy files cut off sharply at 16kHz (320kbps MP3) or 20kHz (V0).
  • Fakin’ The Funk: Windows/Mac tool that automatically checks if FLAC files are genuine. Color-coded results (green = genuine, red = transcoded).
  • Spectro: Online spectrum analyzer. Upload audio and view the spectrogram. Look for a clean, smooth top end with no brick-wall cutoff.
Warning: Many "FLAC" downloads on public trackers are actually 320kbps MP3 files transcoded to FLAC. They take up extra space with no quality benefit. Always verify with Spek or similar tools.

Private Music Trackers (RED / OPS)

For true audiophiles and collectors, private trackers offer curated, verified lossless music with excellent retention.

  • REDacted (RED): The gold standard music tracker. Everything is FLAC-only for uploads. Requires an interview process to join. Strict ratio rules.
  • Orpheus (OPS): Second-largest music tracker. Also FLAC-focused. More generous ratio system with bonus points. Easier interview process.
  • Gazelle Games: Video game soundtracks. Same FLAC standards.
  • What.CD (archived): The legendary tracker that set the standard. Its data lives on through RED and OPS.
  • Interview prep: Study the rules, codecs, bitrates, and transcoding detection. Questions cover audio technology, tracker rules, and encoding knowledge.
# Common interview topics:
# - Difference between lossy and lossless
# - Bitrates (320, V0, V2, Vorbis, Opus)
# - Transcoding: how to identify it
# - Tracker rules: H&R, ratio, seeding requirements
# - Audio formats: MP3, FLAC, ALAC, AAC, Ogg Vorbis
