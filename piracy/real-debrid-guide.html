What is Real-Debrid?

Real-Debrid is a premium link-hosting service that acts as a middleman between you and file hosters like RapidGator, Uploaded, Mega, and many others. It provides high-speed downloads, multi-host access, and torrent-to-direct-download conversion.

  • How it works: Instead of downloading from a slow file hoster as a free user, you paste the link into Real-Debrid. They download the file to their servers at max speed, and you download from them at max speed. Your IP never touches the file hoster.
  • Torrent support: Upload a .torrent file or magnet link to Real-Debrid, they download it to their servers, and you get a direct download link. No VPN needed.
  • Streaming: Real-Debrid generates direct streaming links compatible with Kodi, Stremio, and other apps.
  • Cost: ~$3-4/month for unlimited downloads. Dramatically cheaper than Usenet.

Setup

  1. Go to real-debrid.com and create an account.
  2. Choose a subscription plan. The 30-day plan is ~$4 for testing, multi-month plans are cheaper per month.
  3. Verify your email and log in.
  4. Generate an API token from the "API Token" section of your account settings.
# Testing Real-Debrid with curl:
curl -H "Authorization: Bearer YOUR_TOKEN" https://api.real-debrid.com/rest/1.0/user

# Unrestrict a link (convert to premium):
curl -X POST -H "Authorization: Bearer YOUR_TOKEN"   -d "link=https://rapidgator.net/file/xxx"   https://api.real-debrid.com/rest/1.0/unrestrict/link

# Add a torrent:
curl -X POST -H "Authorization: Bearer YOUR_TOKEN"   -d "magnet=YOUR_MAGNET_LINK"   https://api.real-debrid.com/rest/1.0/torrents/addMagnet

What is a Debrid Service?

A debrid service acts as an intermediary between you and file-hosting websites. The term originates from "debridement" (removing restrictions) — these services "unrestrict" premium-hosted files for you:

  • The problem: File hosters like RapidGator, Uploaded, and Mega cap free users at 50-100 KB/s, impose captchas, and limit daily downloads. Premium subscriptions for each hoster would cost hundreds per month.
  • The solution: A debrid service purchases premium accounts for dozens of hosters. You pay one subscription (~$3/month) and access all of them at full speed with no captchas.
  • Torrent support: Debrid services also download torrents on your behalf. You upload a magnet link, they download the entire torrent to their servers, then give you a direct HTTPS link. Your IP never touches the torrent swarm.
  • Streaming: Debrid services generate direct streaming links compatible with any video player. This is how Stremio + Torrentio + Real-Debrid works. The stream is fetched from the debrid server on the fly.

Real-Debrid Setup

Setting up Real-Debrid is straightforward. Follow these steps:

  1. Create an account: Go to real-debrid.com and sign up with your email and a strong password. Verify your email.
  2. Choose a subscription:
    • 15 days: ~$3.50 (good for testing)
    • 30 days: ~$4.00
    • 90 days: ~$10.00
    • 180 days: ~$18.00
    • 365 days: ~$30.00 (best value, ~$2.50/month)
  3. Payment options: Credit/debit card, PayPal, Amazon Pay, cryptocurrency (Bitcoin).
  4. Get your API token: After purchase, go to real-debrid.com/apitoken. Copy the long alphanumeric string. This token is used by Stremio, JDownloader, and other apps.
  5. Verify access: Use the API to check your account: 
    curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.real-debrid.com/rest/1.0/user
Tip: Buy the 365-day plan directly. You will save money, and if debrid services are blocked in your country, the small monthly inconvenience of finding an alternative is worth the savings.

Linking to Stremio + Torrentio

This is the most popular Real-Debrid use case. Here is the complete setup:

  1. Install Stremio from stremio.com on your device. Create a free account for syncing.
  2. Go to Torrentio config: In a browser, navigate to torrentio.strem.fun.
  3. Configure:
    • Under "Debrid Provider," select Real-Debrid.
    • Paste your Real-Debrid API token.
    • Set quality filters: 1080p+ (or 4K+ if you have the bandwidth).
    • Select providers: Torrentio defaults to all providers, which is fine.
  4. Click Install: Stremio will open and prompt you to install the addon. Confirm.
  5. Enjoy: Browse any movie or TV show. Sources tagged with [RD+] are cached on Real-Debrid and play instantly at your full connection speed.

Cached Torrents vs Remote Torrents

Understanding the difference between cached and remote torrents is crucial for using Real-Debrid effectively:

How caching works: When any Real-Debrid user requests a torrent, it downloads to Real-Debrid's servers. The torrent is now "cached" and available instantly to all other Real-Debrid users. Popular movies and shows are almost always cached. Obscure or very new content may need to be downloaded first (Remote).

Real-Debrid API Usage

Real-Debrid has a well-documented REST API that powers all third-party integrations. Base URL: https://api.real-debrid.com/rest/1.0/

# Get user info:
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.real-debrid.com/rest/1.0/user

# Unrestrict a file hoster link:
curl -X POST -H "Authorization: Bearer YOUR_TOKEN" \
  -d "link=https://rapidgator.net/file/xxx" \
  https://api.real-debrid.com/rest/1.0/unrestrict/link

# Add a magnet link:
curl -X POST -H "Authorization: Bearer YOUR_TOKEN" \
  -d "magnet=YOUR_MAGNET_LINK" \
  https://api.real-debrid.com/rest/1.0/torrents/addMagnet

# List active torrents:
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.real-debrid.com/rest/1.0/torrents

# Check if a hoster is supported:
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.real-debrid.com/rest/1.0/hosts

Rate limits: 5 requests per second per IP. If you exceed this, you will receive HTTP 429 responses. Most API endpoints return JSON. The API token must be kept secret — anyone with it can access your Real-Debrid account.

Download Limits & Fair Use

Real-Debrid has fair use policies to prevent abuse. Understanding these limits is essential for heavy users:

  • Traffic limit: Technically unlimited, but excessive use (multiple TB per day) may trigger throttling or warnings.
  • Concurrent streams: Multiple streams on the same IP are fine. Attempting to stream from different IPs simultaneously will be blocked (1 concurrent IP policy).
  • Torrent size limit: Max torrent size is approximately 200 GB per torrent.
  • Torrent active limit: You can have up to 25 active torrents (downloading or paused). Completed torrents remain in your cloud for 30 days after completion, then are deleted.
  • Streaming quota: No specific limits, but Real-Debrid may throttle if a single stream uses excessive bandwidth during peak hours.

Account Security

  • Never share your API token: Your API token grants full access to your account. Treat it like a password. Revoke and regenerate it immediately if you suspect compromise.
  • Use a strong password: Real-Debrid uses email/password authentication. Use a unique, complex password stored in a password manager.
  • Check active sessions: Visit real-debrid.com/account to see active sessions. Log out all sessions if you see unfamiliar activity.
  • Beware of phishing: There are fake Real-Debrid sites (e.g., realdebrid.com without the hyphen). Always verify the URL is real-debrid.com.
  • Two-factor authentication: Real-Debrid does not officially support 2FA. This makes your password protection doubly important.

Concurrent IPs Policy

Real-Debrid allows only one unique IP address to access your account at a time. If you attempt to use two different IPs simultaneously (e.g., home computer and phone on cellular), the second connection fails:

  • Multiple devices on the same home network: Fine — they all share the same public IP (NAT).
  • VPN usage: If you use a VPN, all Real-Debrid traffic must go through the VPN. Your VPN IP counts as your single allowed IP. Disabling the VPN while Real-Debrid is in use on another device will cause issues.
  • Family sharing: Everyone in your household can use your account simultaneously as long as they share the same public IP. For sharing with remote friends/family, use AllDebrid (allows 3 concurrent IPs) or Premiumize.
  • Workaround: If you need multiple IPs, route all traffic through a VPN at one location. Every device connects to the same VPN server, sharing a single IP.

Troubleshooting

Torrent stuck ("downloading" but no progress): This usually means Real-Debrid cannot find seeders. Cancel the torrent and try a different release of the same content. If it continues, check the torrent on a public tracker to verify it has active seeders.
Download failed / "Server unreachable": The file hoster may be temporarily down or your region is blocked. Try a different hoster. If the problem persists, flush your DNS and try again.
"Too many requests" / HTTP 429: You are hitting API rate limits. Slow down your requests (max 5 per second). This is common with aggressive scraping scripts.
Stream buffering: Check your internet speed. Real-Debrid streams are typically very fast. If buffering occurs, try disabling IPv6 or switching to a wired connection. Also ensure your streaming device has a good Wi-Fi signal.
"Your IP is not allowed" / "IP conflict": Another device using a different IP is accessing your account. Stop all other Real-Debrid activity or change your API token to force all sessions to re-authenticate.
General tip: Before reporting a problem, try: (1) Log out of all sessions and change your password, (2) Regenerate your API token, (3) Try a different file hoster, (4) Flush DNS and restart your router. Most issues resolve with one of these steps.
