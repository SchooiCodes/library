VPNs & Encryption

  • Use a VPN — Encrypts all traffic, hides IP from sites and services
  • HTTPS Everywhere — Ensure sites use HTTPS (most do by default now)
  • Messaging apps — Use end-to-end encrypted apps like Signal or WhatsApp
  • File encryption — Use VeraCrypt for full-disk encryption or 7-Zip for encrypted archives
  • Email — Consider ProtonMail or Tutanota for encrypted email

Password Security

  • Use a password manager — Bitwarden (open source), 1Password, or KeePassXC
  • Enable 2FA — Two-factor authentication everywhere it's offered
  • Strong passwords — At least 16 characters with mixed case, numbers, and symbols
  • Unique passwords — Never reuse passwords across different sites
  • Try our Password Generator — Download batch script

Windows Hardening

  • Keep Windows updated — Security patches are critical
  • Disable unnecessary services — Turn off Bluetooth, location, telemetry if not needed
  • Use Windows Defender — Built-in antivirus is sufficient with regular updates
  • SmartScreen — Keep Microsoft Defender SmartScreen enabled
  • Firewall — Ensure Windows Firewall is active and properly configured
  • User Account Control (UAC) — Keep UAC at default or higher
  • BitLocker — Enable full-disk encryption on Windows Pro/Enterprise
