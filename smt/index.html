Overview

What is SMT?

Schooi's Multitool (SMT) is a powerhouse collection of over 100 utilities packed into a single batch-driven interface. It includes IP geolocation, password crackers, app installers, system tweaking, Windows activation, and much more. Built mostly in Batch (91%) with some Python helpers (8.9%) — the latest released version is v2.2 (Jun 2024) and v2.3 is actively in progress on the dev branch. Check the GitHub repo or smtweb for updates.

Setup

Installation

Method 1 — One-Liner Recommended

Open PowerShell as Administrator and run one of these:

irm "https://getsmt.duckdns.org/" | iex
irm "http://smt.farted.net/" | iex
irm "http://getsmt.ftp.sh/" | iex
irm "http://getsmt.us.to" | iex

Method 2 — GitHub Release

Download the latest release ZIP or the NSIS installer from GitHub.

The .exe installer may be flagged by antivirus as a "dropper". This is a false positive — it just downloads the batch files. Use the PowerShell command instead if you're concerned.

Method 3 — Git Clone

If you have Git installed:

git clone https://github.com/SchooiCodes/smt
Guide

Usage

After installation, run SchooiMultitool.bat or use the desktop shortcut. The main menu lets you:

Tools (1) — password generator, URL shortener, folder organizer, ASCII art, school utilities, and more
Advanced Tools (2) — IP tools, danger zone, app installers, performance tweaks, crackers, and fixes
Edit Script (3) — open the batch file in Notepad for customization
Restart (4) — restart the script to apply changes
Info (5) — view version, tool count, and credits
Credits (6) — view third-party tool credits
Secret Commands — type sc to view all hidden commands: cl (color), shutdown, restart, bios (reboot to BIOS), git (open GitHub), 32 (add to System32), pf (add to Program Files), tcon/tcoff (terminal coloring), mdon/mdoff (auto resize), update (changelog), 99 (command history)

Navigate submenus by number. Type B to go back, E to exit, or / to search. Most tools are written in Batch with some PowerShell helpers.

FAQ

Frequently Asked Questions

Is SMT safe?

SMT is open source (MIT license). Some tools, like the info stealer generator and Windows destroyer, are designed for educational purposes. Use responsibly and only on systems you own.

Does it work on Windows 7/8?

Officially tested on Windows 10 and 11. Lower versions may work but are not guaranteed.

Can I contribute?

Yes! Open an issue or pull request on the GitHub repository. SMT uses the MIT license.

Are all tools made by SchooiCodes?

Most of them, yes! Notable third-party tools: Windows Activator by massgravel, Malwarebytes Premium Resetter by Scut1ny, Chris Titus Tool by Chris Titus, Network Speedtest by Ookla, SMB Bruteforcer by ebola man, Command Line Game by CHM Tech, Group Policy Editor Enabler by majorgeeks.com, Any Winget App Installer by @curlyhair., INI File Reader by rojo (StackOverflow), and Mystery animation by ascii.live.

AI-generated: Folder Organizer, Clipboard Manager, Screenshot Tool, and File Integrity Checker.

Why does the .exe get flagged?

The NSIS installer downloads batch files from the internet, which looks like "dropper" behavior to antivirus engines. This is a known false positive.
