📝 Font Resources

A curated collection of font foundries, developer-friendly coding fonts, icon fonts, and web font tools.

Font Foundries & Libraries

Google Fonts

Free, open-source font library with hundreds of typeface families. Easy to embed via CSS or download for self-hosting. Variable font support for performance.

Visit

Font Squirrel

Hand-picked free fonts with commercial licenses. Font generator and identifier tools. Each font is verified to be free for commercial use.

Visit

DaFont

Massive collection of free fonts categorized by style. From decorative and display fonts to handwritten and calligraphy. Check individual licenses for commercial use.

Visit

FontSpace

Free fonts for personal and commercial use. Community-driven with indie type designers. Browse by style, popularity, or newest additions.

Visit

Adobe Fonts

Thousands of high-quality typefaces included with Creative Cloud subscription. Seamless integration with Adobe apps and web embedding via Typekit.

Visit

Developer & Coding Fonts

Nerd Fonts

Popular programming fonts patched with a huge set of icon glyphs (Font Awesome, Material, Octicons, etc.). Includes Fira Code, Hack, JetBrains Mono, and more.

Visit

JetBrains Mono

Free, open-source monospace font designed for developers. Features ligatures, clear distinction between similar characters, and a tall x-height for readability.

Visit

Fira Code

Fira Mono extended with programming ligatures. Arrows, comparison operators, and multi-character glyphs are rendered as ligatures for cleaner code.

Visit

Icon Fonts

Font Awesome

Industry-standard icon font with thousands of scalable vector icons. Use as a web font or individual SVGs. Free tier includes solid, regular, and brand styles.

Visit

Material Icons

Google's official icon library with over 2,500 icons. Available as icon font, SVG, or React/Vue components. Covers common UI actions and objects.

Visit

Web Font Tools

Fontsource

Self-host open-source fonts via npm. Drop-in replacement for Google Fonts with better privacy and performance. Supports all Google Fonts as packages.

Visit

Transfonter

Online font converter and subsetter. Convert TTF/OTF to WOFF, WOFF2, EOT. Optimize and subset fonts to reduce file size. Generate CSS @font-face rules.

Visit

fonts.css

A CSS library providing consistent font stacks for system fonts. Includes macOS, Windows, Linux, Android, and iOS font stacks with sensible fallbacks.

Visit
