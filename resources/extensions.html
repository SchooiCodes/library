Extensions

🧰 Browser Extensions

Essential browser extensions for privacy, productivity, development, and fun — curated for Firefox and Chromium browsers.

Privacy & Security

uBlock Origin

The gold standard of ad blockers. Lightweight, efficient, blocks ads, trackers, and malicious domains. Essential for every browser.

Visit

Privacy Badger

By the EFF. Automatically learns to block invisible trackers. Gets smarter as you browse.

Visit

HTTPS Everywhere

Forces HTTPS connections on sites that support it. Now largely built into browsers, but still useful as a safety net.

Visit

NoScript

Blocks JavaScript, Java, Flash, and other plugins by default. Only allow scripts you trust for maximum security.

Visit

DuckDuckGo Privacy Essentials

Privacy grades for sites, blocks trackers, forces HTTPS, and lets you search privately.

Visit

Bitwarden

Open-source password manager. Generate, store, and autofill strong passwords across devices.

Visit

Ad Blockers

  • uBlock Origin — ublockorigin.com — Lightweight, efficient, the best
  • AdGuard — adguard.com — Standalone app + extension, system-wide blocking
  • Ghostery — ghostery.com — Tracker blocking with detailed analytics

Productivity

Convert all your tabs into a list and save up to 95% of memory.

Visit

Dark Reader

Dark mode for every website. Reduces eye strain for late-night browsing.

Visit

Grammarly

AI-powered writing assistant. Checks grammar, spelling, and tone across the web.

Visit

Raindrop.io

All-in-one bookmark manager. Save, organize, and access bookmarks across devices.

Visit

Tabby Cat

A new friend in every new tab. Cute cats that collect items.

Developer Extensions

  • React Developer Tools — Inspect React component trees in Chrome/Firefox
  • Vue.js Devtools — Debug Vue.js applications
  • Redux DevTools — Time-travel debugging for Redux state
  • JSON Viewer — Pretty-print JSON in the browser with syntax highlighting
  • Wappalyzer — Identify technologies used on any website
  • WhatFont — Hover over text to identify the font
  • ColorZilla — Eyedropper and color picker for the browser
  • Page Ruler Redux — Measure elements on any web page

Fun & Customization

  • Momentum — Beautiful new tab page with focus, to-do, and inspiration
  • Picture-in-Picture — Watch videos in a floating PiP window (built into most browsers now)
  • Stylus — Custom CSS themes for any website
  • Video Speed Controller — Speed up, slow down, and control HTML5 video playback
